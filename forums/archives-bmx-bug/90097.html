GfK(Posted 2010) [#2]
OK, if anybody else is having this problem (and probably anyone using OpenAL is), edit C:\BlitzMax\Mod\brl.mod\openalaudio.mod\openalaudio.bmx

Change line 34 from:
Const CLOG=True 'False

to
Const CLOG=False


...then rebuild modules.
