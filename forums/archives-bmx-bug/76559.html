Discovered that if a remark is not ended with a "return" character the entire code which follows will be highlighted as if it was a remark. Lightblue color.
It does not effect the operation of the code.

This happens when I cut code from code archives then paste that code into the MaxIDE.

Correct highlighting comes back if I insert a return character at the end of each remark line.

Using iMac system 10.3.9 BlitzMax v1.2.8
Also using iMac system 10.4.11 BlitzMax v1.2.8
