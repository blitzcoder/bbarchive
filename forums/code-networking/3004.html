Get PC's local IP address by BlitzSupport2012
Just a quickie that returns a string containing your PC's local (LAN) IP address, eg. mine returns "192.168.0.2". 
Function LocalIP:String ()
	Return DottedIP (HostIp ("", 0))
End Function

Print LocalIP ()

Function LocalIP$()
	n=CountHostIPs("")
	sIP$=HostIP(1)
	Return DottedIP(sIP)
End Function

Print LocalIP$()
