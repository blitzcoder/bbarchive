Winni(Posted 2010) [#2]
In OS X, an .APP is a directory, not just a single file. The executable binary of the application is in the directory Contents/MacOS of that .APP folder. To make this visible in Finder, right-click on the application and select "Show Package Contents".

Since Mac OS X uses this concept, there shouldn't be a need to append data to the executable itself. Just put your stuff somewhere in that .app folder.
