ima747(Posted 2012) [#4]
if possible make a clean install of OS X (use a flash drive or external hard drive) and see if booting from that resolves it. If it does then it's probably a driver error and you may be able to re-install OS X on top of itself and update from there, or bit the bullet and re-install clean (always better with any OS when you can afford the time and hassle). If it still doesn't work then you might have a firmware issue, or another hardware issue (possibly controller board in the drive) in which case a repair or replacement might be in order (I assume you're not on a mac pro, so you might want to consider an external enclosure or adapter and a cheap burner.
