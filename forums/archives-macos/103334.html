AdamStrange(Posted 2014) [#2]
use a timer event
with a delay in the loop:
SuperStrict

Graphics 640,480


EnablePolledInput()
Local AppQuit:Int = False
Local AppQueue:Int 

Local Timer:TTimer
Local nHertz:Int = 10000 ' 1 second
Timer = CreateTimer(nHertz)
		
'stoptimer
'If Timer StopTimer Timer
		
While Not AppQuit
	AppQueue = PollEvent()
	If AppQueue = 0 Then
		Delay(2)
	Else
		Select EventID()
			Case EVENT_TIMERTICK
				Print "hello t="+MilliSecs()
			Case EVENT_APPTERMINATE, EVENT_WINDOWCLOSE 
				AppQuit = True
		End Select
	End If
wend


this is how all programs should be setup in any modern OS so that multitasking operates correctly. There are other ways, but this is the simplest
