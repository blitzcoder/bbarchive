MrTAToad(Posted 2008) [#5]
I'm running 10.5.4 and BlitzMax IDE is fine here.

However, contratry to popular opinion, I did find that software stability could only be achieved by re-installing 10.5 fresh, and not doing an upgrade from 10.4, so if you went by that route, its probably wise to reformat and install 10.5

However, if its a new installation of 10.5 (which is most likely the case), I would recommend doing the two things that Mac people like to keep quoting : Repair permissions and do a verify of the hard drive.

Failing that, if you have AppleCare, use the TechTools to do a full scan of everything (memory, hardware etc etc).

If not, or after that, you need to find out whether its dependent on the user profile - so create a new user and see if BlitzMax works in that.

Personally, I do find program stability is lacking in most Mac software, unfortunately - when I initially got my Mac, and I tried one of the standard pieces of software, it crashed straight away...
