A new version of BlitzBuddy has been released. It has the following changes:

* Added Add Project and Edit Project toolbar buttons.
* Add new buildandrun AppleScript command.
* Added Build and Run toolbar button and menu item.
* Fixed bugs in AppleScript commands.
* Added Build and Run example scripts.
* Added Run script, which enabled running apps in Terminal for console output.
* Added checkboxes to enable or disable scripts.

** Update: Version 1.3.1 has been released, which fixes a minor bug.
** Update: Version 1.3.2 has been released, which enhances Help searching.
