xlsior(Posted 2009) [#2]
...Because Dim doesn't exist in Blitzmax.

You declare an array like any other variable, for example, these are all valid:

Global A:Int[100]
Local B:String[20]
Local C:Int[10,10]
Global D:Float[10,10,2,2]
