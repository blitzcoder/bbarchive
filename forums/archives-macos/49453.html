Hi everyone,
Who would be interested in better integration of BlitzMax into BBEdit/Text Wrangler?

I am the author of BlitzBuddy, which was a simple attempt to integrate BlitzMax into BBEdit and TextWrangler so I didn't have to use the feature-poor standard editor. Until recently it was only possible to write plugins for these editors using CodeWarrior, which I did not have. Beacuse of this BlitzBuddy uses a combination of a codeless language module and applescript to get things done.

Recently Barebones released a new SDK which allows plugins to be written in XCode. This should allow better integration and allow things like the function popup menu to work properly.

If enough people express interest I might look at implementing a plugin, so let me know your views. I might also want to have a minimal shareware payment to make it worth while :)
