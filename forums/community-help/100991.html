Imperium(Posted 2013) [#6]
What kind of machine are you on? If you're using an older setup with Master/Slave settings it should boot to the Master first.

Also check your boot.ini file in XP and post what it says. It should be something like this:



[boot loader]
timeout=5
default=multi(0)disk(0)rdisk(1)partition(1)\WINDOWS
[operating systems]
multi(0)disk(0)rdisk(1)partition(1)\WINDOWS="Microsoft Windows XP Home Edition" /fastdetect
