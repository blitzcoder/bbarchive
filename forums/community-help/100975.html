Where do I find that option on my monitor is in hardware or software?


Usually, it is to the left of the button you use to turn on your monitor. Read the manual that came with your monitor. You also need to make sure you are using a resolution supported by your monitor and graphics card.


It should be noted that B3D does not work properly on Windows 8 when run full screen. This is not a problem with B3D, it is because of DX7 which is not properly supported on Windows 8. You will only be able to get a max of around 30 frames per second when running full screen.
