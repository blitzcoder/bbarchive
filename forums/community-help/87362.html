xlsior(Posted 2009) [#2]
If you happen to have Windows working on the machine, then run CPU-Z. It will give you every info imaginable about the memory modules present.

Also, to tell apart USB 1 from USB 2 withing windows:
In the device manager, USB root device, see if it makes any mention of "advanced USB controller". If so, it's USB2. If not, it's 1.0 or 1.1.
