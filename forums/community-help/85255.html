_PJ_(Posted 2009) [#3]
I don't know how much is possible, because I understand Vista has a lot of 'fixed security' implementations, but generally, speeding up windows can be achieved by:

Increase Caching
Decrease Visual Options pertaining to the Desktop/Themes
Kill Unnecessary Processes
Disable Unnecessary Services
Disable AntiMalware On-Open/Running Process Checks
Disable Automatic Disk Defragmentaiton (Not sure if Vista does this, I presume it does)
Disable Firewall (Not recommended)
Disable any Virtual Drives
Depending on the Speed of your HD, you may find Swap File usage is more of a slowdown.
If you have a multicore processor, and the game does not support this, ensure it runs on a core separate form more intensive CPU processes.
Ensure drivers are up to date.
If running audio through a chip-based onboard codec suite rather than a dedicated card, better results may be obtained by disabling the sound device.
Keep system cool.

xlsior(Posted 2009) [#9]
All things being equal, Vista is slower in part because it runs a lot more crap in the background. Of course, everything that runs will be fighting for part of the available CPU power.

Some of the slowdowns will also be due to the additional need for memory, which can lead to delays due to swapping stuff from RAM to the harddrive and vice-versa.

The #1 speed boost, is to make sure that you have a ton of memory. Vista isn't really happy unless it has 2GB or more to work with.

You can further optimize a bit by going through the windows services and disabling some of the unnecessary parts, which would cut down a little on CPU and memory overhead. You can also disable a bunch of the graphical eyecandy and the likes.

but yeah -- all things being equal, XP will outperform Vista on identical hardware. All the new additions and tweaks do come with some extra overhead, after all.
