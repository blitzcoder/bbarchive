Some of the newer ATI video adapters have on-board HDMI, which can install a secondary sound device on your computer. (HDMI = DVI + sound)

D4NM4N(Posted 2009) [#4]
goto terminal and type "lspci"
look up the list for anything that looks like your sound card.
If it is there then it is probably just muted in alsa. There are two sound controls. The one in gnome and the basic alsa mixer. The latter seems more reliable for debugging.

Go to terminal and type "alsamixer" make sure the main channel and PCM are not muted. Its got an odd controls i think its like arrow keys and space to change things (check the man/help on alsamixer or figure it out by fiddling)

This usually fixes the problems if the sound system seems to be there but has no sound. Failing that, goto ubuntu forums you usually find a quick fix.
