IF you are sure your internet settings are set to allow downloads from the site, and that you are allowing the activeX controls for that authentication stuff to work:

Ensure you have at least updated windows with Windows Update.

I would check your Services:

%SystemRoot%\system32\services.msc /s

look for one called Windows Management Instrumentation - Make sure it is enabled (Manual or Automatic) and STARTed.

Also do the same for:
BITS
Cryptographic Servics
DCom

I dont own a Zune or have any experience of one, so I wouldn;t know about the firmware upgrade, but at least, this should help with your windows authentication issue :)
