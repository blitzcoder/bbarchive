Hi,

I have a HD in my laptop which was split in 2. (OS & DATA).

I've put in an ssd as the main drive and put the HD in the second drive bay.

Is there anyway I can 'format' the HD into the 'one' drive?

Jim


dawlane(Posted 2014) [#2]
If your using Windows Vista onwards on the ssd. You can use the the virtual disk manager. Right click the My/Computer icon and select manage. Go to storage->Disk Management, select the drive that you want to format, select each partition and delete it. When done you can create a partition to use the whole drive and format it. How to from here http://www.7tutorials.com/how-manage-your-disks-using-disk-management-utility
