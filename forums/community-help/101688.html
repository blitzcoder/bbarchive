I'm using Sound Forge Pro 11 - when I'm using the Event Tool to split a track into multiple parts, is there a way of rejoining two parts if, for example, I split them in the wrong place?


Gabriel(Posted 2013) [#2]
If you drag select an entire clip and then drag it onto another clip, you'll see an overlay of the audio where it will be pasted. If you drag it past the end of the target clip (the one which comes first, chronologically) it'll "stick" to the end of the track. Let go and they'll be rejoined.

GfK(Posted 2013) [#4]
I've worked it out. You delete either one of the events, then drag the end of the other event to fill the gap (the audio data is still there).

Oh, this Event Tool is apparently a new feature introduced in Sound Forge 10. Up til this morning I was using an old version which is why it's thrown me a bit. Now I'm used to it, it MASSIVELY helps with editing audio, i.e. if you've got a bit of dialog and you want to edit out a word, you can just cut the piece in two, and drag them so they overlap slightly and it automatically cross-fades between the two. And it's not permanent either. You can drag lumps of audio about all day. Certainly come a long way since the old Sonic Foundry days.
