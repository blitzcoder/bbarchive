Yasha(Posted 2012) [#4]
I have been trying to get MiniB3D to load up 3DS files


miniB3D doesn't support 3DS.

What it does do is silently convert the string to the path to a B3D file and then try to load that, which of course doesn't exist in your case. I assume this is a placeholder for a 3DS loader that is intended to be added at some future point, but it's a strange way of going about it.

The Blender plugin is probably not a Mac problem but a version problem, as far as I know: due to engine changes in Blender the B3D exporter hasn't worked correctly in years. You could download an old version of Blender that supports the plugin (e.g. 2.48, I think) and install that alongside your "real" Blender for the sole purpose of converting to B3D.

If you have Wine (if you want code archive entries, you'd better have a means of running Blitz3D itself), you can use PaceMaker to convert 3DS to B3D. Other B3D conversions would best be done by not looking in the archives for a converter, but two bits: load-whatever and export-B3D.
