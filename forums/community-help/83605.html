It's actually a file type setting in Windows

Go to the Control Panel and click on Folder Options.
Click the File Types tab and find the entry for zip files.
Click the Advanced button
Check the box that says Confirm open after download

That's how its done in XP at least. I'm pretty sure it's basically the same in Vista
