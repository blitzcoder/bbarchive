xlsior(Posted 2011) [#3]
About running 32bit applications: does it take more memory to run on a 64bit OS than on a 32bit OS?


If you want to split hairs: Very marginally so, because it also needs to load a 32->64 bit thunking library, (WOW64), in order to run 32-bit executables. The extra overhead is insignificant though.

The memory allocations made by the program itself don't change, IIRC.
