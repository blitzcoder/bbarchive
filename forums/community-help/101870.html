I was just exporting a BMP from GIMP and noticed there's a Compatibility Options drop-down in the save dialog.

If you open it up and hover over the "Do not write color space information" box, it says some applications may not be able to read unless you check the box, which is unchecked by default (ie. potentially incompatible) -- maybe try that?
