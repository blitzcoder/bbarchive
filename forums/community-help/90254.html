Sauer(Posted 2010) [#6]
Along with what SLotman said, use CSS Divs. For the largest container, containing the background, set the margin-left and margin-right parameters to "auto" which will center it on the screen.

As for the rollover buttons and text, use more Divs within the main container, using absolute, relative, or fixed positioning. More information on that can be found here.
