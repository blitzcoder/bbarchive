If u use Thunderbird, add all the Russian non Latin looking vowels to a filter. Works perfectly for.me ;)
I also did it for Thai and the most common Chinese letters.
Takes a while but worth it. You can backup the filters too so you only need do it once.
Wish someone would write a plugin though.:/

Ps if you tell the filter to move them to deleted they will be expunged from the IMAP server. I dump my trash every session (an option in settings somewhere).
What I do is keep a copy of Thunderbird running in its own session on my IMAP server tuned in to localhost, that way it cleans my inbox automatically for my client pcs every time the IMAP server collects all my mail from various places.
