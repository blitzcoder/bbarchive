Kryzon(Posted 2011) [#2]
The 'editing with effects' will be more difficult to find in freeware products, so you might want to consider using more than one solution if you really have to go the free way.

For US$100 you can get TMPGEnc Authoring Works 4, which is a very reliable package, with all that you mentioned. Check its features, it's awesome.

I also route you to the Authoring Tools section of VideoHelp.com which has all you need: http://www.videohelp.com/tools/sections/authoring-dvd?orderby=Type
If you do find something you like, please report back so people who read this thread can get a quick-start :)
