D4NM4N(Posted 2012) [#10]
Not seen this particular problem before. It sounds to me as if you have somehow messed up the profile.

Try this:
-Back up your Amon Home folder and call it amonbackup
-Create a new ADMIN user called tempmofo.
-restart- the computer (this is important)
-log in as tempmofo
-run explorer as admin so you can delete your amon folder
-log in as amon again.
-copy whatever you need back from amonbackup to amon. All non hidden stuff should be ok, docs, favs etc... (and whatever hidden appdata/localsettings you need (but be selective here, don't copy any MS windows settings back. In fact i would avoid copying all MS stuff back altogether)
-leave tempmofo in place until you are sure profile is working properly. You may need to set up some stuff again because it will all be back to defaults (unless you copied the relevant app/data stuff)
-reboot and make sure it all works.

You should be back to normal.
