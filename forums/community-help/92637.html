xlsior(Posted 2010) [#3]
I've successfully used PING to clone HD's myself: http://ping.windowsdream.com/

Alternatively, you can use Junctions or symbolic links to move just part of your directory structure to a different drive. See the command-line tool mklink.exe that's part of Vista and Windows 7 for that.
It can link a folder to another folder on a different drive, while it looks like they're still in the same location... So, for example, c:\games really redirects to d:\games instead, but as far as windows is concerned it's still where it used to be.
