ardee(Posted 2011) [#2]
Import for DirectX models was removed for Blender 2.5-2.6 so...

Blender 2.49b
File-->Import DirectX (.X)
Find the file and import it.
File--> Save
Name your file and save it as .blend

ardee(Posted 2011) [#4]
Just reattach the image to the model in the UV Image editor, the UV co-ords should be fine.

As for animation make sure the models are in DirectX 8.0 format. Worked fine for me with old models I had from FPS Creator. You may have to run them through Ultimate Unwrap to get them into DirectX 8.0 first.
