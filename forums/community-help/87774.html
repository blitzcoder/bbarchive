Get him to run DXDiag and make sure hardware acceleration shows as enabled on the Display tab, and that all the tests run.

Also, if the game uses a fixed screen size, try compiling with a simple Graphics3D 640, 480, 0, 1 and also windowed (640, 480, 0, 2).

Make sure his drivers are updated too!
