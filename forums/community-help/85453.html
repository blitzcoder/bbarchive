xlsior(Posted 2009) [#5]
Possible reasons:
1) there may have been a glitch with the blitzbasic webserver or DNS server (I haven't noticed any problems, though)
2) there could have been an outage somewhere between your ISP and BRL's hosting provider
3) you could have some spyware on your PC that's screwing around with your DNS queries (new.net and the likes can cause problems for example)

#2 can be worked around using a proxy, like you did.

Just out of curiosity: were you able to resolve the IP address of the website?
e.g. open a command prompt and type NSLOOKUP blitzbasic.com --> see any IP returned?
and if you do, try PING blitzbasic.com --> See any response packets?
