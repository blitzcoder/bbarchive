Yan(Posted 2013) [#6]
You do have to play around a lot with the codec and video settings to get reasonable results from CS. Personally, I find it a bit flaky.

I've taken to using Open Broadcaster Software for my, very occasional, screen recording. It's really intended for live streaming so it's a little bit fiddly to set up and the audio/visual codec support is very limited, but I've found it to be rock solid in use.

Perhaps I'm just old and jaded, maybe even paranoid, but the ezvid site set alarm bells ringing. I won't be downloading it.
