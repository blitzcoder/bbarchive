Well Linux Mint 15 “Olivia” is here at last and I've just installed it.
Normally I would back-up all my files and do a clean install as if you've ever used Linux just doing an upgrade can actually break your system.

Well this time I couldn't be bother to a clean install and just updated the sources and did a full system upgrade.

After rebooting and figuring out that I had to reset the desktop session back to use my default one of Mate as it just crashed. I then jumped straight into testing out if Monkey would still compile in glfw......
It didn't as it complained of undefined references. Easily fixed by modifying the make file in the targets directory to explicitly link those libraries.

So if you should get link error either by doing an upgrade or a clean install then just update your makefile with this and all should be well.
