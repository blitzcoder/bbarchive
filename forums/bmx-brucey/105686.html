The latest code (MaxIDE, MaxGUI, Scintilla-textarea) now support configuration of the caret - you can set the width (1,2,3) and color. It does make a big difference on the Pi ;-)
This will be available in the next release, once everything is built, tested and packaged up.

There are also some recent improvements in bcc which improve compilation times - which should also help on the Pi. FYI, a release build of the BRL modules only takes 42 seconds on the Pi 3. Which isn't too bad :-)
