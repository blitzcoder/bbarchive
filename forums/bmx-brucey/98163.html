Hi,

There appears to be a bug in WxWidgets for Ubuntu/Gnome which prevents a menubar from being displayed. You can see this in the wxMax samples minimal.bmx and menu.bmx. The shortcuts work, you just cannot see the menu!

I have done some research and the problem seems to be the default Auto-Hide Menu feature of Ubuntu. There is a wxWidgets bug report on this problem at:
https://bugs.launchpad.net/ubuntu/+source/wxwidgets2.8/+bug/662077

I have tried uninstalling appmenu-gtk and the menu appears, but I get an error message so this is not a good work-around.
