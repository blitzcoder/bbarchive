Bass top tip!

BlitzMax Forums/Brucey's Modules/Bass top tip!

jkrankie(Posted 2011) [#1]
After battling with the membit not set and stack underflow errors that result from streaming music from a file, i was very pleased to discover that streaming from memory (i.e. and incbinned file) seems to stop these errors and also makes running things in debug mode much quicker. It also stopped my game crashing on Win 7/Vista 64bit with ATI cards, which has nice!

Anyway, i'd recommend anyone else using BASS for streaming music does it this way, rather than the other.
