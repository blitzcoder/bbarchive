I've just added Rar (read) and BZip (Read/Write) support to wxMax through its wxStream interface.
There's also a small sample each for them, just to give an idea about how they work.

I admit that the Rar one is a bit on the crappy side - as far as functionality goes - so I might have a go at making it a little better if I get some time. (the problem is the original implementation was a barely functional wrap, with most of the useful stuff like file info and listing rar contents, missed out). But it extracts files okay.

I've updated both to the latest versions of the libs involved too.

Normal zip support will come too at some point :-)
