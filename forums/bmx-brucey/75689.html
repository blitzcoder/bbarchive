The weekend is upon us again, so I thought we must be due a new module.

Today's release is BaH.MAPM (Available from HERE).

MAPM is an Arbitrary Precision Math library, which lets you manipulate numbers far bigger than Ints and Doubles.

Why would you need something so precise?

For a game, perhaps Float/Double is adequate, but for some apps, they need accuracy much greater than 16 digits. MAPM can handle those kinds of figures with ease!

Okay... a bit of a niche Module this one... ;-)
