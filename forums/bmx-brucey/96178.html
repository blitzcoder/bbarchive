Winni(Posted 2011) [#1]
Hi, it seems that Bah.Growl no longer works with Growl 1.3 on Lion - apparently the required growl delegate cannot be installed.

Also, probably to be App Store compliant, Growl 1.3 no longer installs anything in the System Preferences and accordingly IsGrowlInstalled() now always returns false even though Growl is running (IsGrowlRunning() correctly returns true). That probably is not a bug, just something to keep in mind when talking to Growl 1.3.

The Growl 1.3 SDK still is not available for download on http://growl.info, maybe the problem will dissolve once the framework files can be updated. (Or maybe that's just wishful thinking on my part.)
