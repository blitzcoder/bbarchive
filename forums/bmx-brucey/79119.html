BlitzMax 1.30 supports more Audio Drivers

BlitzMax Forums/Brucey's Modules/BlitzMax 1.30 supports more Audio Drivers

Brucey(Posted 2008) [#1]
With the release of BlitzMax 1.30, audio drivers for irrKlang and FMOD now work "out of the box".

An Audio Driver is an implementation of BRL.Audio, which allows the use of standardised commands to load and play audio.

The BaH range of audio drivers use libraries such as irrKlang and FMOD as backends, and therefore add to BlitzMax technologies such as audio streaming, without requiring the user to learn any new APIs.


A BASS audio driver is also in the pipeline.

The Audio Driver module which sits on top of a particular Audio module abstracts away the underlying APIs (of FMOD etc) to give you use of the standard commands.

You can of course directly use the Audio module if you want/need to, which gives access to all that audio-library's functionality, like better control over effects, 3D, etc.


As for a comparison :
http://www.fmod.org/index.php/products/fmodex
http://www.un4seen.com/bass.html
http://www.ambiera.com/irrklang/features.html

They are all cross-platform, except that BASS is not available on Linux.

Essentially, they all accomplish the same thing, in slightly different ways. Some have more features when it comes to adding effects.

It depends what you need out of your speakers :-)
