Yet another new widget for the weekend.

This time, it's wxGradientButton :



The colours for normal and pressed states are customizable, of course, and you can set the font as you would normally for any wxWindow-based widget.


For those that are interested, both the wxGradientButton and wxMozillaLikeAboutBoxDialog widgets come from http://wxwidgets.info/ . A Russian site with lots of interesting articles (many in Russian) and tutorials.

:o)


Note that this is only available via the 2.9.0 SVN - so if you want to try it out be sure that you want to use the SVN version with its current problems (wxPropGrid etc)
