Brucey(Posted 2016) [#10]
Most likely something to do with this in MaxIDE :
' gcc error
		err=err.Replace(EOL+"   "," ")
		While err
			p=err.find(EOL)
			If p=-1 p:+err.length	'equiv. to p=err.length-1 ;-)
			mess=err[..p]
			err=err[p+1..]
			p=0
			Repeat
				p=mess.Find(":",p)+1
				If p=0 Exit
				q=mess.Find(":",p)
				If q<>-1
					file=mess[..p-1]
					line=Int(mess[p..q])
					If line
						mess=mess[q+1..]
						SelectError file,column,line
						Notify LocalizeString("{{output_error_compileerror}}").Replace("%1",mess)
						Return
					EndIf
					p=q+1
				EndIf
			Forever
		Wend
