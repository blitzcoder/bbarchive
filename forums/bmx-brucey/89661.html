Pete Rigz(Posted 2010) [#4]
Try

local url:string = "zipe::data.zip::xml/data.xml::helloworld" 
Local doc:TxmlDoc = TxmlDoc.parseFile(loadtext(url))
If doc = Null then Throw "Error"
