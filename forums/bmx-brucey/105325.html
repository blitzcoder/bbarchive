Brucey(Posted 2015) [#4]
@MinGW

It looks in this order :
* BlitzMax/MinGW32 - local folder with complete MinGW distribution
* %MINGW% - environment variable
* fallback to using BlitzMax/bin and BlitzMax/lib

The first case makes it easier to have multiple BlitzMax builds using different versions of MinGW. (otherwise, you'd have to change MINGW more often).
