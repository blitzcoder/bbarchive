Yeah puzzle difficulty is never easy, in the end I created a level editor that largely auto generates the puzzles, I couldn't really find a way to gauge a levels difficulty so I went for quantity rather then quality thinking it more a time waster than a mind-twister.

There are currently no levels in the game that feature the polar bears, cept the tutorial so I need to correct that.

It's built in such a way I can easily add level packs with updates or even IAP so I plan to hand craft a level pack or two if it proves popular (downloads).

The game was more a test of my framework than anything else really and to just get a game finished.
