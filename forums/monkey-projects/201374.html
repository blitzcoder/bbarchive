More fun with Monkey...
Strict

Import bah.bank

Function Main:Int()
	Local b:Bank = New Bank(32)
	
	For Local i:Int = 0 Until 4
		b.Poke(i * 4, i * i)
	Next
	
	For Local i:Int = 0 until 4
		Print b.PeekInt(i * 4)
	Next
End


output :
0
1
4
9

Bank uses a class called Pointer, which is a pseudonym for a real pointer (malloc/free).

Fun fun... :-)
