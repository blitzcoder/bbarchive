@therevills I am not using Diddy in my apps just because I am uncertain about the license. I'm not sure how to interpret the MIT license because it requires a copyright credit. I don't have a TOS popup on my new games, and didn't really want to do that, but I'd have to in order to display the entire MIT license disclaimer.


therevills(Posted 2011) [#29]
Basically MIT means you can use it free... but if you decide to use the library in another library (for example if you make your framework using Diddy) you must include the license and credit Diddy.

For a game, you can use it without any credit (although it would be nice ;))
