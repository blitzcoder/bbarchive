I am currently deep in development of a rather nifty tool to let you manage your 2d texture packing, animation, character classes, etc. The tool is based on a maxgui framework that I have written which will allow new features to quite rapidly be added in the future. That's if it gets enough interest/purchases that is! Features like a map/level editor would not be impossible!

For now objecty will be focused on managing texture packing, animation, game assets and game data. I am hoping to have two licenses available one for individuals and another for teams. I'm not expecting anyone to pay an insane amount so hopefully it will be attractive to some monkey devs!
