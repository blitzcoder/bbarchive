Jesse(Posted 2012) [#5]
the whole process to submit to Mochi is real simple:
1) register with Mochi.
2) create a a submission page for your game.
3) add the registration number to your produced *.AS file as stated in the tutorial that's here some where( a single line of code nothing else).
4) recompile your *.monkey game.
5) submit the *.swf file to your Mochi game page..
6) download the newly created flash game from Mochi and add it to your website.
7) test it. if you seed a notification for adds then in your Mochi game page set the flags for adds and distribution and you are set.

if you do not see adds or the adds notification, you did something wrong and you are going to have to edit the *.as file and repeat step 4 to 7.
you don't need to add the module if you just want to display adds.
