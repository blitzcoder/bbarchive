Several months back before I took a hiatus from monkey, i began working with a friend on a little old skool beat-em-up type game which i called "Bad Bros".

Dunno if i will continue on this project again or not.

Anyways, screens :





Download for Mac
Download for PC

Controls (quite buggy) :
move left - left arrow key
move right - right arrow key
run - arrow key twice in either left or right direction
jump - up arrow key
drop down a platform - down arrow key
attack - z key
"upper cut" - a key, I think this was suppose to be an uppercut...

Apparently, if you have a 360 controller, it also works too (PC version).

My attempt at a "combo system" :
rapidly attacking - throws an attack combination
super kick - run, then attack will perform a super kick on contact
super jump kick - run, then jump, then attack will perform a "super jump kick"

Note : occasionally, the player may freeze up for some odd reason. This is a bug.
