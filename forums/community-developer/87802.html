Drak(Posted 2010) [#3]
I use anim8or for modeling because it is soooo easy to use once you get the hang of it. Maybe all modeling programs are like that. Anyway, the creator of Anim8or never implemented exporting .3ds models WITH animation. There is an option for it in the menu system, but it does not work. The model will export, but will have no animation data exported with it. You'll have to use another animation program. I think Anim8or's animation system is pretty lacking anyway, although I do like the modeling part of it.

3DS modles don't support any animation other than object transform changes (entityscale, entityposition, entityrotation)

There is no skeletal animation or anything alike that on 3DS :)
