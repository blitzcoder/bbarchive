I have 3 3d modeling programs. None of them have the b3d file format but they all have the 3ds file format and 1 of them has the "x" format. Can these formats be used for saving animations for use in Blitz 3d?


nadia(Posted 2005) [#2]
3ds files can be loaded directly into Blitz3D but have limited support for animation.
'.x' has full support for boned animations and can easily be converted into Blitz3D's native file format 'B3D' with a tool like Ulimate Unwrap or Milkshape.
