Dear gameSpace Light owner,

We hope you are enjoying gameSpace Light and finding
it a useful and productive tool!

It came to our attention that some have experienced
problems with gameSpace Light, and to resolve those,
we are releasing a new patch. All gameSpace Light owners
should download this patch using the address below:
http://www.caligari.com/download/gs/gslpatch.exe

Please copy this file to the directory to which you
installed gameSpace Light (by default, this would be
c:\gameSpacelight) and then run the patch, and your
copy of gameSpace Light will be updated!

Thanks,

Team Caligari
