dawlane(Posted 2016) [#3]
Use LMMS. LMMS is a free cross platform Digital Audio Workstation (DAW) a lot like FL Studio. You will need to acquire some soundfont2 files if you wish to use General Midi sounds, but you can play around with the built in instruments. Check out http://zdoom.org/wiki/sound_fonts for links.

You should be able to import the midi data directly into D.A.W. editing tools and export them into a number of formats. LMMS suports wav and compressed ogg.

Kryzon(Posted 2016) [#8]
In my experience, don't use ModPlug, use OpenModPlug. The latter is a community edition with several improvements.

Open your MIDI file and export it to WAV. OpenMPT will render the audio using the soundbank from Windows (those infamous MIDI sounds that are not very realistic).

With a WAV in your hands you can use any other tool to convert it to the desired format.
Audacity makes it easy to export something to OGG.
