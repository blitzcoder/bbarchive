Basically yes, you're right, Milkshape won't import embedded animations.

jhocking(Posted 2004) [#3]
There is a roundabout way to import animations into Milkshape. You will need Ultimate Unwrap. Import the animated model into Ultimate Unwrap (it supports several animated file formats, including b3d.) Now save in Milkshape file format.
