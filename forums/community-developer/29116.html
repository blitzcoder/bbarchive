no, o2c is a 3d format which includes texture data. it is pretty one-way, there are several whatever-to-o2c converters, but nothing o2c-to-whatever. It is a format designed for web presentation of Meshes. I think it is under some patent protection and allows to publish Models which are useless for anything but watching.

Programs like Archicad are using it. I have a little Architecture App which exports o2c. However. I just found it surprising that I couldn't find a converter.
