It should be noted that this is really just a nice configuration for the user interface. Indeed, beats the default by a long shot!
The vertical layout there really helps with the panels collapsed by default. Strange I never thought of that... somehow I ended up convinced that horizontal was the way to go.

I have this working just fine with simply the .B.blend file from that archive. I imagine the rest of it changes provides Mayaish translations for different locales, but to get just the nice looking UI, you don't need much :)

As for replacing the default cube scene, you can press Ctrl+U to replace the default settings with current ones. (I take it they ran out of letters in the alphabet...)
