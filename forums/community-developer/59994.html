Gabriel(Posted 2006) [#4]
Microsoft setup files have the extension stf, and try as I might I can't think of any texture or image format which uses that extension, and certainly if there is one, Blitz doesn't support it.

Could it be an archive of some kind? Like a zip or rar or something but with that extension? Or perhaps it's some obscure format that a particular game engine uses, but it's no use for Blitz.

Stick to JPG and PNG wherever possible.
