napole0n(Posted 2004) [#5]
I wouldn't use Maya for leveldesign, but rather for creating art elements for use in your level. I prefer to write my own leveleditors (and there are plenty good shareware leveldesign tools available from members of this forum) and use my Maya-crafted elements as 'tiles' if you will. This saves a lot of re-exporting and other quirks.
