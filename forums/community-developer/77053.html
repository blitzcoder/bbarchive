Moraldi(Posted 2008) [#6]
Hi,
Unlimited Drop Pro first demo released!
Here is the download link:
http://www.moraldigames.com/UDProInstall.exe

The application is fully functional (except saves) and runs under WinXP environment

Install:
--------
Unzip the file in any folder you want (I am sorry about that but the installation program is not ready yet).
There is a sample.udp map file in the 'Samples' sub folder
There is a udp.dll and the corresponding .decls file inside the 'Blitz3D' sub folder.
There are also sample Blitz3D programs to access all information provided by the sample.udp file

Uninstall:
----------
Delete the folder containing the application and subfolders
There is also another file created in 'Documents And Settings\Curent User' subfolder named UDropPro.ini. You can delete it also

Quick Guide:
------------
Because help files are not complete yet please follow this quick guide to start playing with the application:
1. The basic idea behind UDropPro is Actor Types. First you create Actor Types and then create Actors (based on Actor Types) to place into your scene.
2. You can browse your hard disk to find 3D files (.x, .b3d, .3ds). You can switch to Preview Model mode to see the meshes
3. Create an Actor Type by pressing the 'Create' button on Actor Type tab
4. Create and place Actors by double click on an Actor Type

Futures:
--------
- Organize your actors to layers
- Create lights and alter their properties with a simple mouse click
- Create brushes and textures to paint Actors
- Define animation sequences for all animated Actor Types
- Define custom properties for all Actor Types
- Preview your level
- Customize the application user interface and look
- Undo list similar to Photoshop
- Define custom properties for your scene
- Define Ambient light

and many more...

And the most important:
Unlimited Drop Pro files are sqlite3 files. Advanced users will appreciate this.
In any case with the udp.dll that comes with the application you can access all the information stored. See samples!

Feel free to contact me if you have questions about the application. I will answer to you individually
