I really like 'DeleD' - I downloaded it today. I never normally use modellers - I have stuff like 'CINEMA 4D CE6' on my drive, but I never bothered using them. As 'DeleD' is so easy to use it has kind of woken up an interest in 3D modelling in me that no other package on my drive has done before - purely because 'DeleD' is simple and easy. It may not be a cutting edge modeller with all the snazzy features, but it is a very good starting modeller for a novice. Good stuff.

If anyone out there has thought about modelling, or tried it and found it too hard/complicated, then try 'DelD' (it's in the Toolbox section on this site).

PS - It is free, so if you don't have/use a modeller, you may as well go and download the sucker.
