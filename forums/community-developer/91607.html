jfk EO-11110(Posted 2011) [#6]
Make sure to set a key for every bone on the very first frame. Also make sure all vertices are bound to the correct bones. Make sure not to use too much weighting, don't bind vertices to more than 2 bones (IMHO), this would slow things down and if you by accident use too much bones, who knows what's gonna happen, bugous behaviour wouldn't be surprising.
