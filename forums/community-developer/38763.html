morduun(Posted 2004) [#2]
You need to select the faces you want to map and select mapping methods for those faces. Select the faces however you like, then open the 2D Tools menu to the UV Mapping submenu and go from there.

You may also need to create a new material, if one hasn't already been created for your model. Do this by selecting 'Materials" in the top right list box, right-clicking and selecting "Add..."

Having a material is irrelevant. The issue is whether or not any UV coordinates are set. If not you won't see anything in the UV window. But don't fret. Just set a simple UV projection (I think the UV Maps are under the 2D editing window) like Planar for the entire model.
