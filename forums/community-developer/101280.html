Just discovered this incredibly useful thing:

http://www.greentoken.de/onlineconv/

It looks like it's pretty much just a web interface for AssImp's import and export features, but hey, that's often all you need.

Importantly (from my perspective, anyway), it can import B3D files! An easy way to get any old B3D models you need to bring forward to a non-Blitz project into a more widely recognised format like e.g. Collada.
