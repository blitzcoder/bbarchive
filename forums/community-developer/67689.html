While searching on the net for a decent 3D program that includes modelling, UV mapping, texture painting, animation and rendering I came across this one: Cheetah 3D

It can do polygon and spline modelling as well as scripted mesh creation. It has an UV editor and UV unwrapping is really really easy with it. Texture painting is not only limited to just textures but you can also draw high or displacement maps. Animation is done via Keys and F-Curve and the renderer supports radiosity, caustics, HDRI backgrounds, ambient occlusion, anisotropic texture filtering, bump mapping, environment mapping, reflections, refractions, soft shadows, and more AND it's multithreadened!
It currently costs $99 and the upgrade to version 4.0 (which will bring joint based skeletal animation, morph targets, Inverse Kinematics and much more) is free!

Website: http://www.cheetah3d.com
