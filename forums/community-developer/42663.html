Rob Farley(Posted 2005) [#2]
Personally I set the UV map from the unwrap tool as a base layer on a paint program then just add a layer and start drawing over the top. This way you can keep the various elements of your UV map seperate. This is handy if at any point you change the positions of some of your UVs you can just move the layers around and line up your image again.
