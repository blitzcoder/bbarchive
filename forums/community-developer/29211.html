Hi fellas,

Just a quick note to tell you that version 1.1 of gile[s] is out.

Here is a list of changes and additions:
- Render Engine Optimisations (2-10 speed increase on average, but up to and beyond 50 times on some scenes)
- Registration System Changed
- OBJ importer. Courtesy of Lee Page
- Clone Model Fix
- Remember Window Position
- Remember File Paths
- Toggle Forced VRam for lightmaps
- Clear Vertex Colors
- Fix of LWO loader
- Sort model list by name or type
- Cube map support + cube map renderer
- Merge lights with gls files fixed
- Preferences moved to seperate window
- AC3D importer fixed
- Model Editing fixes
- User definable shortcuts
- Extended lightmap parameters!
- Vertex Lighting no longer forces Vert. Color flag

So head on over to the gile[s] website and get it, and drop by the forum to see what's going on :)

Have fun!
