Pete Carter(Posted 2011) [#4]
You can just use any of the animation tools in Cinema4d i used the joint tool and paint tool to paint the vertex weights on. you can then just save as a c4d file then load in unwrap with the c4d plugin. you may need to normalize the model as there many be some bones that effect to many vertices as blitz only supports 3 i think. but then you can save as a b3d file. there are limits it doesnt like it if you have more than one surface in the model. and the scale is really large. but it works and doesnt take ages going back and forwards between programs. oh i forgot a step bake the keyframes before export :O)
