Many have asked for a cheaper version of molebox, as well as for the ability to additionally pack data to a file. The people from Molebox did it, although not both things in the same product. Of course.

Molebox has now two versions: The Pro Version with a new feature that allows to add data packages and a new interface for 99$, and a basic version without the additional data packaging and dll embedding support for 49$. It is a cheaper solution for those who don't plan to use external dlls (or at least don't need to pack and protect them).

The Pro Version is a free upgrade for all customers of the original Molebox.
