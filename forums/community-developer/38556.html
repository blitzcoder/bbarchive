jfk EO-11110(Posted 2004) [#4]
CFX is nice, but there is also the free Pacemaker, just search the forum. It's written in Blitz and can do nice animations.

UU is the state of the art UV Unwrapper, a thing you need these days, and AFAIK the only tool that is capable of importing an animated B3D file and edit the UVs, or save it as something else.

UU is the sequel of Lithunwrap 1.3, that was freeware too, but it lacks of several features found in UU.

Particle Candy is nice, but maybe not THAT essential. Maybe you should also think about B3D Tweak Utility, Gile[s], Milkshape.
