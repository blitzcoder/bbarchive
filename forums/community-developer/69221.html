bradford6(Posted 2007) [#2]
1. open unwrap
2. apply a composite material to your model in the DIFFUSE slot. (pick composite in the dropdown)
3. click properties, there you can assign a bitmap to use UVSET 1 or UVSET 2 (for example, have your detail textures use UVSET 1 and have your lightmap use UVSET 2

there is a [1][2] on the toolbar that will tell you which UVSET you are currently working on within UUnrap.

hope this helps

A Composite Material is just multiple textures applied to the same object. In this case it uses different UV Coordinates (UV Sets)

generally, you want to use this for models (like game levels) that have detail textures ( for instance for the walls, floor, windows, etc) and a single, large, lightmap

the detail textures would use UVSET1 and the lightmap would use UVSET 2
