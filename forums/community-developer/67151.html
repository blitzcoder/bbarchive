jfk EO-11110(Posted 2007) [#4]
Maplet is really a oneway street. ALWAYS make sure to finfish things in Maplet. You may have to learn this the hard way :o/ and so did I.

Currently, when I really have to edit something, then I import the B3D in in Giles, save as 3ds (without lightmap), import in 3dsMAX, edit (use eg: EditMesh Modifier), then save as 3ds, reimport in giles and finish lightmapping etc. in giles.

Instead of 3dsMAX you can also use other Mesh editors that are capable of importing 3ds files properly.

For this 3ds path make sure to use texture names that are no longer than 12 chars, in 8.3 format.

I'm not sure if things like VertexAlpha etc. are handled correctly. That's however no problem with plain Maplet meshes.
