Kryzon(Posted 2010) [#2]
Yes it does.

Unwrap3D does export B3D animation with bone hierarchies.
However, it exports it as a single skinned mesh. Mesh hierarchies
are not exported. Visually, there is no difference.

From UU3D Support mail.


jhocking(Posted 2010) [#3]
Also note that Ultimate Unwrap doesn't support vertex weights in playback (or rather it didn't in the old version I have, I don't know if the latest version does.) So if your model has blended vertex weights then it'll look messed up while animating in Ultimate Unwrap, but the weights will be exported correctly.
