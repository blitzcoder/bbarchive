I checked out the demo. Looks good. How do you import your own textures in the full version.?


JoshK(Posted 2005) [#2]
Place your own texture anywhere in the ENGINEPATH directory. The default ENGINEPATH directory is the same as the program directory. You can change the ENGINEPATH directory to your own project's directory. Go to Tools>Options to do this. The directory will be scanned recusively, and you textures can be stored anywhere within.
