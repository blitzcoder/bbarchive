Wow, I've just discovered this, maybe you folks have already.

http://robersonaudio.tk/prophanity/

What these are basically, are VST plugins that run with a software called saviHost.

So you can get many great plugins for free or lowprice, and use SaviHost to run them. Then record using Audacity. This is a great way to make your own sound effects and music. GREAT STUFF!
