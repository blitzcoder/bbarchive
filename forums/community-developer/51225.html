There is now a new Fontext library available for BlitzMax. The library was written by Tom (otherwise known as Scouse):
http://playerfactory.proboards25.com/index.cgi?board=fontext&action=display&thread=1127095317

Fontext has now been used successfully in Blitz3D, Blitz+, and BlitzMax. As well as others: DBP, C++ (on a PDA) etc.

You can use Fontext fonts directly in gile[s] (using a plugin), which enables you to add great looking text to your scenes (no more blurry images of signs etc).


If you haven't heard of Fontext before:

Fontext is an application and suite of libraries, for the creation and display of bitmap fonts. As well as text, the libraries handle such things as HUDs, user interaction, and 2D animation.

It has been used successfully in well over 20 games, many of which are commercial.

It has even been used in one game to handle the whole display including game graphics.
