Ricky Smith(Posted 2006) [#9]
If you're planning on using bones then you'll have to convert from .x to .b3d. Blitz3d only supports the DirectX7 hierarchial mesh animation when importing an animated .x. For skeletal animation with bones it only supports its own native format .b3d.
You can use Fragmotion to convert DirectX8/9 skeletal animation to .b3d. It only cost 15$ and is a great little tool.
