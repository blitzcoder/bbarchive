If your using a modelling tool that allows you the use multiple UV's and texture baking then you can set up one set of UV's for the whole model then the other UV's can point to the textures that you want to use for each part of the model. Then when you bake the model all the individual textures will be applied to a texture that you can use for the whole model.

I do believe that blender can do this just google "texture baking uv set to uv set" and see what pops up.
