jfk EO-11110(Posted 2004) [#7]
Check out my forest sample in the code archives.It's a free alternative for fast outdoor vegetation. The maketree function uses only one texture in the demo, but the function takes a texturehandle as a parameter, so you can use a variation of plants as well. It also includes a single surface grass system.

Sometimes I find it a bit disappointing that everyone is trying to sell YET ANOTHER BLITZTOOL YOU DEFINITIVELY NEED. But hey - in the code archives are lots of candies that will allow you to make your own perfect tools (eg. lightmapping: "YAL"), or for CSG check Markus Rauchs CSG Editor - etc etc etc.

One Freeware tool I hardy recommend is LithUnwrap 1.3. It's the last free version, and of course its commercial sequel "UltimateUnwrap3D" is better, but LithUnwrap is still very useful.

In short we will have an Animation tool named "Pacemaker" for Characters and so on that comes with a free light version. Check this discussion:
http://www.blitzbasic.com/Community/posts.php?topic=33087
This will be the first freeware Animation tool ever! (byside one very old md2 editor) Try the demo of Pacemaker, it's great.

Then there's wings3d and many other 3D modellers. Visit www.3dlinks.com , but take some time, lots of links.

Check out the Project Plasma Thread in the 3D Forum Region, I have released a simplyfied open source version of the placement editor i use for my Demos (see sig) that will allow to load and position stuff, such as buildings, furniture, items. Or have a look at Droplet, which is doing about the same and maybe has a better user interface (but isn't open source).
