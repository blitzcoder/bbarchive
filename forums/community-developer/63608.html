Hiho all,

here is my B3D-Export plug-in for Cinema4D:



Works with all versions of Cinema4D R6 or later.

Download(2.5 MB)!(including documentation and examples)

Features supported right now:
-Animation
-Autokeyframing
-Full range of material options
-Weighted/non-weighted bones
-Second UVW-set to support lightmaps
-GUI & Log
-Additional useful plug-ins like "Disconnect All"
-Multitexturing
-Several surfaces per object

Features to come(in a far future):
-Support of smoothing tags/groups

This plug-in is still in development, so don’t hesitate to contact me, if you found any bugs or miss useful features. Of course, general feedback is welcome too.

BIG BUG(Posted 2007) [#8]
Ok, here is a new update for my exporter.

There are several new features since V1.1.
Here are the most important:

-Any texture, which is restricted to polygons will be exported as separate surface now. This makes level design and especially creating of lightmaps much easier.

-Multitexturing is now supported also.

-Via Vertex Maps it is now possible to set vertex alpha / color

-All polygons are disconnected automatically by default now, to ensure correct texture mapping all the time.
