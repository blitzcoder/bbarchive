Ricky Smith(Posted 2007) [#3]
If you're using biped with both models it should be just a case of exporting the animation from one model as a .bip and then importing it back into the other. You can choose a starting frame on import so you can add the imported frames to the current animation. Can all be done from the motion panel IIRC.
