jhocking(Posted 2004) [#2]
I'm not sure where to get it from because the company stopped developing it a while ago, but one really good free renderer is BMRT (Blue Moon Rendering Tools.) While I'm not sure if this is still the case, at one point Wings integrated BMRT for rendering. I mean, it was a separate download, but once you downloaded BMRT you could just hit "Render" in Wings.

Note that you can bring texture mapped models from Ultimate Unwrap back into Wings for rendering. Wings can import obj file format. So even if BMRT is no longer available, you can use YafRay to render in Wings.

POV-Ray is another free renderer to look into. Ultimate Unwrap can export to POV-Ray files.
