Unless Brad has made an update I'm not aware of (which is certainly possible) Ultimate Unwrap cannot do multiple material layers. It CAN do some multitexturing, and I'm pretty sure reflection maps are included in this. You set multiple texture layers within a single material.

For the most part however multitexturing is why I purchased B3D Tweak. I can export b3d files from Ultimate Unwrap with a single texture layer and animation intact, and then use B3D Tweak to, well, tweak the b3d file and add additional effects like multitexturing.
