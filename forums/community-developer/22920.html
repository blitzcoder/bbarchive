I have found this GPL tool, which generates a BSP file from a Quake3 MAP file.

http://map3bspc.sourceforge.net/

This would remove any id Software licence issues, as long as the BSP format is not licenced itself, is the compiling tools which have restricted use. It is intended to use with 3rd-party engines which uses the BSP format for levels.
