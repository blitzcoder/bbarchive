fredborg(Posted 2004) [#1]
Hi,

Version 1.30 is now up for grabs. Here's a list of changes and new stuff:

- Modify Plugin System
- Plugin System changes
- Import Plugin System
- General internal tweaks
- Select Light Method Drop Box
- Minor bug fixes
- Auto Export Lightmaps
- Export Selected
- Exporters moved to plugins
- Selection on Merge/Create
- Textbox/spinner errors fixed
- Smarter Statistics Gathering
- Smarter Lightmap Preparing
- Faster Lightmap Packing
- Post Rendering Hanging Minimized

But the biggest change is that rendering is now much much faster. On small scenes it's twice as fast, and on complex scenes it's up to and beyond 10 times faster.

The plugin system allows you to write your own plugins, to modify, import, export or create models.

Get it here: http://www.frecle.net/giles
Toolbox entry: http://www.blitzbasic.com/toolbox/toolbox.php?tool=41
