http://kvazars.com/littera/

it seems that this site is unknow to the BlitzMax userbase (while I found a link in MonkeyX site).

In any case, it's an online bitmap font generator, quite complete.
You can choose your own font, colour, bevel, shadows, character set, etc and then export in various format. (It generates a .zip file whit an image and a file for the coordinates/informations required).

I did some test in BMX and MX1 and it works well.
