I am currently using PaceMaker ( http://www.goddysey.com/ ) for Character Animation etc.

How do I delete the keys of a certain Sequence in the Animation Mode?


Ricky Smith(Posted 2005) [#2]
Select the sequence from the list box and click the DELETE button. Click YES when asked if you want to delete the underlying animation frames. The sequence and the underlying animation keys will be deleted. If you only want to delete the sequence and leave the underlying keys intact then click NO - only the sequence will be deleted.
