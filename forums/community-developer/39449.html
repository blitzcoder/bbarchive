I've created my first ever model in Wings (yay!) but it's in some random orientation - is there any automatic way of aligning it square to the axes?


Pongo(Posted 2004) [#2]
easy way,...

1. create a cube to align your object to
2. select a face that you want aligned
3. right click, and select put on.
4. now select the face on the cube to align it to.
4. right click, and the object snaps to the new alignment.

edit: you may need to have advanced menus turned on,... look in the prefs to turn them on.
