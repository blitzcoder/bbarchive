Sorry worked it out for myself. Simply set the camera field of view to 90 horizontal and render each frame facing at 90 degree angle intervals around the y axis then one facing directly up and one facing directly down.

Only problem - the only format XSI easily put into blitz is the .X format which is not quite right anyway as it flips the texture coordinates so that all the images need to be flipped horizontally. But this is a known problem with XSIs X exporter.
