jhocking(Posted 2008) [#5]
shame shame Beaker, you were not nearly emphatic enough. gile[s] is a VERY good lightmapper. It is a VERY VERY good lightmapper.

Ultimate Unwrap and gile[s] are the two tools that I can't imagine a Blitz developer not having.

As for how gile[s] compares against 3D World Studio, as Beaker points out they are simply not the same thing. 3D World Studio is used to build your level, gile[s] is used to lightmap a level you've built somewhere else. I'm guessing that gile[s] is a better lightmapper than whatever is built into 3D World Studio, so a potential workflow would be to built the level in 3D World Studio, export the b3d, and then import that into gile[s] to setup lighting.
