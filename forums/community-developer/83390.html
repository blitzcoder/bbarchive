I've been playing around with FrameToon lately, which is a really cool bitmap based 2D animation program. I've only been using the trial so far, but have been very impressed with. It's priced at only 6000 Yen (about the same price as copy of GTA), which is not too bad at all. The official site is here...

http://www.portalgraphics.net/ja/products/frt/

As you may notice it's all in Japanese, but don't panic. Some nice guy made an English patch for it which can be found in this thread.

http://www.animationforum.net/forum/showthread.php?p=37036

So far it's been working nicely. It does multiple levels of onion skinning, with control over how they behave, plus one reference frame. You can rotate the work surface, for easier drawing. And it supports multiple layers per frame. The drawing tools are not too many, a basic round brush, plus a blending brush. It's made by the same people that make openCanvas, and shares much of the same tools, except it's made for animation. Like openCanvas it boots fast, and seems to be really low on resource usage. I played around with animating in full HD (1920x1080) on my machine with 1gig of RAM without any problems.

This could be a very handy tool for doing sprite animation, just need to test out the different export options a bit more.

Now, if I could figure out how to purchase it from their site, I would be good to go. :)
