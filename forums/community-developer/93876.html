Hi Everyone,

I'm pleased to announce that Medieval4D ( v0.7) is now available for purchase.

I'm currently offering an early adopter licence for an introductory price of US$30. This licence allows you use any models created with Medieval4D to be used in any of your game products, and also receive free tool/media upgrades when they're ready. Because Medieval4D is still a work in progress please pay particular attention to the EULA before deciding to purchase.

What is Medieval4D
Medieval4D is a 3D tool and media set which allows you to build a variety of medieval structures such as houses, inn's, cottages etc by using a simple drag and drop interface. You can create 'shell' structures for an RTS style game or create fully 3D structures that the player can explore inside. You can build your own LODs for each structure by adding or removing the smaller detailed parts of the model, and there is even the ability to export a separate collision model. Once your structure is finished you can export it to the .X format for use in your games.

Finished models can also be shared with other Medieval4D users, so even if you don't have time to build your own structures you can just copy/paste models from the online Structures Database and modify them to your own needs.
