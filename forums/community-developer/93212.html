Hello

Anybody using DIGIMI package by DAZ Studio ?
It includes 3 modules : Decimator - Atlas for textures - FBX exporter

You can import a DAZ model, reduce the number of polys , create a texture suitable for game engines and export an FBX file

To me it seems to be a good opportunity for those who dont have skill and / or time to create complex characters since you can rely on the huge DAZ library

There are many " poly reducer " on the market but the beauty of the decimator is that it is optimized for the DAZ UNI Mesh
As you know all the DAZ models share the same mesh
