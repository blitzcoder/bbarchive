Pongo(Posted 2007) [#7]
This is not what you asked for, but Wings3d is a very good modeler. (And Free!)

You cannot create vertices and faces like what you are asking for, however, because all models in wings need to be closed. That is to say that there cannot be any holes in the mesh, and you cannot create a single face on it's own. Technically speaking, you can create holes in objects by using a "hole material" that simply does not save out when you export the file.

Anyways, it is an exceptional modeler, and well worth checking out.
