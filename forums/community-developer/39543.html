jhocking(Posted 2004) [#2]
I don't know exactly how the exporting is done (I don't use 3D Studio Max,) but I do know for a fact that the b3d file format supports alpha transparency. Note that alpha transparency is only supported in Blitz via TGA or PNG file format, and the alpha channel must be part of the same image file as the color image (as opposed to a separate greyscale image for the alpha.)

Hopefully someone knowledgable with B3D Pipeline will pipe up.
