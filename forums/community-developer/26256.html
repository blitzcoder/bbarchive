Hello,
I couldn't go anywhere in the blitz3D game development without this 3 simple executable tools.

They are an extended MediaViewer, able to load .b3d, .x, .3ds, and all the image formats supported in blitz (very similiar to the one included in blitz)

A Mesh Vertex Finder, able to find what's the vertex number in a given mesh.

* An Animated Models Viewers, able to show bones within the mesh, and to retrieve useful information.

What's their peculiarity ?
Why I shuold use these ones instead of the one included in blitz ?

Well, the first simple but important reason, is that you can associate in windows all your blitz media files with these application, so that a double click on a .b3d file will result in the viewer opening in front of you with the desired model.

Or you can use it simply by dragging on its shortcut the media you want.

The real good thing, is that it supports any path, so it will never return a Unkown file type because of quotes " or because of spaces in the paths.

Plus all the viewerz will give additional informations, which the original mediaviewer won't give.
