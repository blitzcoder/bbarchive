I created this little tool to use it for my own project and thought some people in the forums might have a use for it too.
It is very easy to use: Just copy a mesh into the tool's folder and start the program. Type in the name of the mesh and the outputimage size and the tool will do the rest for you.

http://www.file-upload.net/download-1339877/m2h.zip.html

Supports .b3d .x and .3ds
