ZombieWoof(Posted 2004) [#1]
Select any installed Windows Font and TTF2PNG will allow you to set color/width/height for the character set.

http://www.camelot-software.com/files/ttf2png.zip

Files saved are PNG with transparent background suitable for use with GUI libs that take bitmap fonts. Some libs may require specific character cell sizes or colors. Consult your GUI Lib documentation.

Requires the .NET runtime
http://www.camelot-software.com/files/dotnetfx.exe
