fredborg(Posted 2006) [#2]
Hi,

Can you give this a try:
http://www.frecle.net/giles/download/ImportX.zip
Unzip it to /gile[s]/plugins/importers and overwrite the existing plugin.

It should allow you to import vertex colors and alpha from ascii x files. You need to set the materials to use vertex color and alpha, as it does not autodetect it when importing. If you have the option in xsi to save as either binary or ascii, select ascii.

The obj fileformat does not support vertex colors at all.

Let me know if it works!
