I Just wanted to give everyone a heads up on a new lightmapping application called Luminescence 3D. http://www.insight-concepts.com/products/luminescence3d/
It is not officially released yet, but they are offering a public beta. No help file though. It supports B3D and a few other file formats as well. The performance is incredible and the quality is just as impressive. Ill list a few features below:

Radiosity
Individual radisosity settings for each object
Individual lightmap settings for each object
Reflections
Multiple Object Import
Directory Import
You can save the lightmaps as 7 different file formats

There is too much to list… They have a full list on their site: http://www.insight-concepts.com/products/luminescence3d/features.php
