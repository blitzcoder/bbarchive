Lens Flare effect by markrosten2001
Example Blitz 3d program using lens flares.

4 graphical types of lens flares are used over 7 seperate lens flare effects.

The example includes a sun, a planet, and a rotating cube to show what happens when the sun is hidden from view.

The camera can be moved around to show the distances between lens effects.

The code is fully commented so hopefully you'll be able to understand what is going on ;)
