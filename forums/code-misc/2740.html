Floyd2010
You can tell blitzcc.exe to dump a list of keywords. The versions.txt in the main Blitz3D folder tells about the various versions. See V1.78 for instructions about dumping keywords.

I think it includes keywords added by userlibs. If you don't want these you can "disable" userlibs temporarily by renaming it to, say, userlibs.bak.

I get the following list for the 1.105 version of Blitz3D. This is a fresh install so userlibs are not an issue.

NOTE: This was done by running blitzcc.exe from the command line and "piping" the results to a file with the command:

blitzcc -k >keywords.txt
