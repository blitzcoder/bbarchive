Text 'Shadow' by UnderwoodNullium2008
Super simple text shadow. 
Function DrawText2(s:String,x%,y%,shadowoffset%,r%,g%,b%)

SetColor(Max(0,r - 100),(Max(0,g - 100),Max(0,b - 100))
	DrawText(s,x+shadowoffset,y+shadowoffset)
SetColor(r,g,b)
	DrawText(s,x,y)

End Function
