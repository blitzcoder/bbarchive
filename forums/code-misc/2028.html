Display ASCII chart by _332007
It's a self running program that displays an ASCII chart with the character and code from 0 to 255. 
Graphics 800,600,32,2
j = 0
For i = 0 To 255
 Write Str$(i) + " = " + Chr$(i) + "   "
j = j + 1: If j = 10 Then Print "": j = 0
Next
WaitKey()
