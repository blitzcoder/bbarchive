I have a series of PNG files named xxx000.png, xxx001.png, etc. and I would like to have a single xxx.png file that contains all the frames so I can use the LoadAnimImage command. Does anyone have a utility or know of one that will go through the seperate files and create a single one?


Bremer(Posted 2003) [#2]
You could program this in blitz. Just use createImage to make an image big enough to hold all the images. Then load all the images and copy them to the new image shifting the X position for each image, and when done save the new image.

If you have 20 image that are 20x20 pixels, then create a 400x20 pixel image and copy each into x*20,0

Hope that made sense :-)
