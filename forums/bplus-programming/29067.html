Jumping scorpions takes practice. Get the timing right. Make sure you are running when you jump. That's all I can say about that.

You can't jump over a frog that's jumping over a ladder, but you climb down the ladder while the frog is on the other side (and then climb back up once the frog has jumped over you). Make sure you press down before you get to the ladder or else you will fall through.

Read the readme file for more info.
