Yasha(Posted 2009) [#6]
Type ReturnValue
   Field a,b,c
End Type

Function Foo.ReturnValue(a,b,c)
   ret.ReturnValue=New ReturnValue
   ret\a=a+1
   ret\b=b+1
   ret\c=c+1
   Return ret
End Function


That's how I would do it (in Blitz3D).
