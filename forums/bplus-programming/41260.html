The utility UpdateResource (http://www.miharemec.com/tools.html) can be used to add icons via the command line. It can also do other nifty things, like add version info. I haven't used it, but it's the follow up to something called "ReplaceIcon" that I use.

At least it's command line so you can just write a .bat file to do the work for you.
