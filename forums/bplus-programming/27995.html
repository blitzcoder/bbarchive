GfK(Posted 2003) [#2]
Its Tahoma 8pt, but you should be aware that its possible to change the fonts used by Windows. So if you set your treeview gadget to use Tahoma, and the end user uses something else, its still not going to look right.

That said, most people probably don't bother 'customizing' Windows to that extent and just use the default font.
