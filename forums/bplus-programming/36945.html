soja(Posted 2004) [#2]
Make sure EventSource() = toolbar.

Your event loop would look something like this:
Select Waitevent()
Case $401 ; Gadget action
Select EventSource()
Case toolbar
Select EventData
Case 0 ; Button 0
Case 1 ; Button 1
Case 2 ; Button 2
End Select
End Select
End Select
