Sarge(Posted 2005) [#7]
I use this in blitzmax so i dont know if this will work for blitzplus but give it a go.

Const CB_SETITEMHEIGHT=339

Function ComboBoxHeight( gadget, height )
SendMessage_(gadget,CB_SETITEMHEIGHT,-1,height)
End Function
