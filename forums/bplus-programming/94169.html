Sipko1(Posted 2011) [#2]
Since nobody is helping me with this... this is what I figured so far:

I made a ping.bat file with the following line:
ping.exe www.yahoo.com>ping.txt

Then I open the text file from my Blitz program and I read the results. The only problem with this one is that the *.bat file takes several seconds to generate the result, so I have to stop my process in my program until the ping.txt is generated.

It is not perfect, but it works :-)
