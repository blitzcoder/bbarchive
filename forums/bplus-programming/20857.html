If you want to create a standard "underline" beneath your window's main menu, use:

CreateProgBar(0, 0, <window width>, 2, <parent window handle>).

Give it a whirl...


cyberseth(Posted 2003) [#2]
Alternatively, if you're the sort of person that thinks this would be a waste of a progress bar, use a panel like this: (Assuming your window is called 'window')

pnl = CreatePanel(0,0,ClientWidth(window),4,window,1)
SetGadgetLayout pnl,1,1,1,0

NB: The last variable in CreatePanel ("style") is not documented, but is shown in the statusbar when you press F1 in the usual way. It creates a panel with a "sunken" border.
