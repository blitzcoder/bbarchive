Here ya go, this will work for you. Taken from an old thread. Grab Resource Hacker (its free)


1. Open your EXE from Reshacker
2. Action > Add New Resource...
3. Locate the icon file you want to add.
4. In the dialog box, call the resource "ICON".
5. Click OK.

well first things first: make an icon (say 32x32 in true colour) in a decent paint package and save it as ICO. If you can't do that, save it as .bmp and use something like Imagicon to convert it to an icon with a transparent background.
