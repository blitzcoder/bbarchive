Floyd(Posted 2003) [#2]
The first 8 bits are alpha. This is used in textures.
When ReadPixel is used on an image the alpha value is always 255.
This corresponds to fully opaque.

ReadPixel(x,y) And $FFFFFF

This zeroes out the first 8 bits and gives the RGB value you expect.


Beaker(Posted 2003) [#3]
col = ReadPixel (x,y, ImageBuffer(Mask) And $FFFFFF
should do it.

Strips the alpha values which can make it appear like a negative.
