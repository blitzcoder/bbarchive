Banshee(Posted 2005) [#3]
Example of global array usage,

dim myArray(100)

myArray(1)=1

myFunction()

function myFunction()
   print myArray(1)
end function


Declare the array in the main program before the function call and your array will be accessible in all functions called thereafter.
