Sauer(Posted 2009) [#2]
Off the top of my head, I think this should work:

If KeyDown(205)
		For a.APPLE=Each APPLE  ;for every apple
                            a\x=a\x-3   ;move left 3
	            Next

		p\x = p\x - 3
		
		
		b\x = b\x - 3
		h\frame = h\frame + 1
		
		If h\frame > 2 Then 
		h\frame = 0
		
		EndIf 

EndIf
