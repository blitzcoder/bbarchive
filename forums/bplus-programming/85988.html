Matty(Posted 2009) [#5]
Functions don't need brackets if they are not going to return a value into a variable in blitz.

For example:

readpixel x,y

will compile

as will

readpixel (x,y)

but

col = readpixel x,y

will not compile

while

col = readpixel(x,y)

will compile.


blackgecko(Posted 2009) [#6]
Right, functions need brackets, if you want them to return something.
