Andres(Posted 2008) [#3]
Function SimulateMouseClick(x%, y%)
	cur_x% = MouseX():cur_y% = MouseY()
	api_SetCursorPos(x%, y%)
	api_mouse_event($0201, 0, 0, 0, 0)
	api_mouse_event($0202, 0, 0, 0, 0)
	api_SetCursorPos(cur_x%, cur_y%)
End Function
