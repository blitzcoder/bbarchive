em22(Posted 2008) [#2]
If you use :
command$="cmd.exe /c echo %errorlevel%"
a=CreateProcess (command$)
 
While Not Eof(a) ; repeat loop until process has ended
d$=ReadLine(a)

DebugLog (d$)

Wend 
notify "done"


Should do the trick.
