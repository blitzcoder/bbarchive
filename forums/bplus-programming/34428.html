Remember Blitz3D isn't really an upgrade, it's a different product. If you buy Blitz3D, you'll lose all the GUI commands, but gain 3D ones. Also, the 2d parts of each are different. The blitzPlus 2d stuff was written from the ground up.

Anything you write in blitzplus, minus the GUI stuff, should still work (with a few modifications) in blitz3D.
