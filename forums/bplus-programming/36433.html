Hi Pyro

check this out:


Graphics 640, 480

Repeat

xs = MouseXSpeed()
ys = MouseYSpeed()

MoveMouse 320, 240
MouseXSpeed()
MouseYSpeed()


If(MouseDown(1))

xs = xs * 2
ys = ys * 2

End If

x = x + xs
y = y + ys

Text x, y, "*"

Flip

Cls

Until KeyHit(1)

regards

Mr Brine
