Grey Alien(Posted 2005) [#2]
In this very thread someone posted some code to change the icon on the title bar recently, it worked well.

Also when I asked about .exe. icon they said download ResHack (resource hacker), open your .exe in Reshack, Click Action/Add new resouce then open file with resouce (and open your .ico file made with standard paint software recommend 32x32 pixels, windows will scale if need be), type icon in the two blank edit boxes and click Add resouce. Save and voila and icon. It is really quick when you get used to it and you only need to do it with final release anyway.

Good luck
