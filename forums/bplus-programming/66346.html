I am working on my next game. I was interested in making sprites transparency, scaling, etc for making modern 2D.

I know that we can't do that with Blitz2D or BlitzPlus.

So I checked for "Extended Blitz 2D v. 1.60", some great functions created by Patrick Lester alias "aka turtle1776".

I was a newbie Blitz coder when he released the first versions, and I didn't understand how to use the functions.

Now, with this last version, I am completely amazed by his work ! All is very simple to use, powerfull, fast and very well documented !

If you are interested with "Extended Blitz 2D v. 1.60", if you didn't heard about this, try this link :

http://www.blitzcoder.com/cgi-bin/showcase/showcase_showentry.pl?id=turtle177605192003023744&comments=no

Good coding !
