WolRon(Posted 2004) [#2]
You can't use variables in For-Next loops but you can do this:
For loop = start to quit Step incr
	;do stuff here
Next
becomes...
loop = start + incr
Repeat
	;do stuff here
	loop = loop + incr
Until loop > quit

Note that you may need a seperate loop if your increment is a negative value.
