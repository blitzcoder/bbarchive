After being around the world and trying out other game development tools like Unity, Game Guru, Torque 3D, Godot, Playcanvas, Cyberix 3D, Jmonkey, and Babylon J's, I finally run the full circle and have come back to Blitz. Though I might try to learn other development tools just to continue to broaden my scope and what not, I'm pretty much here to stay now I It seems like throughout my journey, I discovered that I don't really like Game Engines that much. It seems like with code based game development tools and Programming languages, I have more control over how my game is ran. And it's like playing the guitar with a capo, instead of building to play guitar chords without a capi. Thus, that's just something I discovered about me. I may keep in touch with Babylon js (as it is code based too and I like making webgames), but out of all of the tools I tried, I like Blitz/Monkey X and BASIC dilelect Programming languages the best. They do everything that I need and then some.
