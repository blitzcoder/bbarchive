I must say I prefer fresh install for the following two reasons :

1. Who says your backup isn't still infected, fresh install is a way to be sure.

2. Installing the latest drivers for graphics card etc. directly saves more room on your harddrive (No old files) and you are sure you are using the latest drivers.

It takes a bit longer to install, but you are sure you won't have to restore the pc anytime soon.
