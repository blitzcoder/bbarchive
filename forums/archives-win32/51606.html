...Stupid question, but is there any particular reason you -need- version 7?

DX8 and 9 are backwards compatible with version 7, and I don't believe that the DX installer will let you install version 7 on a machine that already has a newer version...

If you really do need an installer, you could check out some game CD's for games from that era if you happen to have some on hand, many came with the DX installer on the game CD itself.

It appears that Microsoft no longer has DirectX 7 up for download on their own site, but a quick google showed the DirectX 7 download at the following URL: http://www.free-game.com/Directx.htm
Although I have no idea who these people are, so download/use at your own risk. :-?
