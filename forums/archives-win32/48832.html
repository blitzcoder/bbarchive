Do you want something like this ?
Function WaitEvent()
Local msg:MSG = New MSG
	While apiGetMessage( msg,0,0,0 )
		apiDispatchMessage(msg)
		apiTranslateMessage(msg)
	Return msg.wparam
	Wend
FlushMem()
End Function
