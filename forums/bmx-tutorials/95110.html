Digital Wizard’s Lab is a framework for making 2D games in BlitzMax. It has a lot of features including sprites, tilemaps, layer hierarchy, loading / saving system, collision mechanism and simple physics. These features will be shown in following tutorial where we create almost exact copy of 1985 Nintendo’s Super Mario Bros first level. You will learn how to accomplish different tasks of game development using the framework.
