A word of advice on this one.

If you are using Leadwerks Engine 2.0, then you need BMax 1.28 - for Leadwerks Engine 2.1, you need BMax 1.30


In the video, "halo" has shaders.pak in the main Leadwerks directory - I spotted this straight away and copied it into mine. Leadwerks 2.0 does not seem to need it there, that is why it isn't there when you install 2.0. His video tutorial is for LE 2.1 though so bare that in mind.

Just thought I would clear this up as having shaders.pak in the main Leadwerks 2.0 directory can kill the engine features, but the program/script will run fine with no errors - you just lose lighting, shadows, shaders, etc.
