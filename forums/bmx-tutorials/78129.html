* This forum is for people to post programming tutorials.

* The tutorial must be written by yourself.

* If possible, the tutorial must be included in your post.

* If the tutorial is too big to include in a post, and/or uses media files, please send any files that are part of the tutorial to webmaster@... and we will host them ourselves.

* Please don't post any links to external sites - even your own - as over the years these have a tendency to die.

* Please do not post simple code postings - these go in the code archives.

* Please do not post tutorial requests or help requests - use the other forums for programing help.

* Please only reply to a topic if it is related to the tutorial or the subject the tutorial covers
