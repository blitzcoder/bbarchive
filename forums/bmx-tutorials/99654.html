Blitzbasic.com Search for Chrome

Scaremonger(Posted 2012) [#1]
This tutorial is for Chrome, but the same feature can be created for Firefox etc.

In Chrome Browser: Right-Click on URL entry box, select "Edit Search Engines"

A New tab will open. Scroll down to the bottom and add a new Entry:
Column 1 is the Name, Type "Blitzmax"
Column 2 is a Keyword, Type "bmx"
Column 3 is Search URL - paste the following into it:
"http://blitzbasic.com/Community/_search.php?bbs=&forum=&terms=%s&method=simple&case=no&results=topic&user=&submit=Search"

Press OK.

Now open a new tab and type "bmx" followed by a space and your search details; when you press return the search is redirected to Blitzbasic.com. (If you prefer blitzmax.com then use that in the url instead).
