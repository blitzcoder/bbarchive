nawi(Posted 2008) [#3]
You can enable access to shaders and other stuff by using GlewInit(), and then use for example GLSL. You can search the forum for more information. I found this code in MiniB3D which tells the version, and for me I get GL 2.0 after glewInit.

If GL_VERSION_1_1
	If GL_VERSION_1_2
		If GL_VERSION_1_3
			If GL_VERSION_1_4
				If GL_VERSION_1_5
					If GL_VERSION_2_0
						Print "GL 2.0"
					Else
						Print "GL 1.5"
					EndIf
				Else
					Print "GL 1.4"
				EndIf
			Else
				Print "GL 1.3"
			EndIf
		Else
			Print "GL 1.2"
		EndIf
	Else
		Print "GL 1.1"
	EndIf
Else
	Print "GL ?.?"
EndIf
