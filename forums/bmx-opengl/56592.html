Dreamora(Posted 2006) [#4]
You would need to do it this way:

Type Tight

	function ActionWrap(entity:object)
	        Tight(entity).Action
        end function

        Method Action()
		Print "Action packed!"
	
	End Method

End Type 



Local WhatIsIt(entity:object) 

ItIs:Tight = New Tight

WhatIsIt = Tight.ActionWrap
WhatIsIt(ItIs)


WaitKey()

End
