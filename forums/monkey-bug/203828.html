Ted crash if Help tab left on PDF

Monkey Forums/Monkey Bug Reports/Ted crash if Help tab left on PDF

DruggedBunny(Posted 2012) [#1]
If I view one of the PDF tutorials (eg. mine) in the Help tab, then close Ted, the next time I start Ted the Help tab will load up the PDF file.

However, if I then hit the Home icon to get back to the main Help page, Ted crashes -- happens with both mine and Robbert Prins's tutorials -- and then when you load up again you get the PDF again!

You can get around it for now by closing the Help tab containing the PDF, then clicking Home, which takes you to the main Help page. I reckon it should probably just default to the main page on startup as the Blitzes have always done (easy fix!), but maybe some people do want it to load up whatever they were last on... ?
