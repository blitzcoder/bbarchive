The DrawPoly textured override should be like this:

Function DrawPoly( verts#[],image:Image,frame:Int=0 )
#If CONFIG="debug"
	DebugRenderDevice
	If frame<0 Or frame>=image.frames.Length Error "Invalid image frame"
#End
	Local f:= image.frames[frame]
	context.Validate 'this line has been added
	renderDevice.DrawPoly2 verts,image.surface,f.x,f.y
End


Currently context.Validate is missing so it doesn't take into account the current matrix.
