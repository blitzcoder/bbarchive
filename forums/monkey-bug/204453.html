Samah(Posted 2013) [#2]
This has nothing to do with type conversion.

str[0..1] creates a String containing the first character of str. str is an empty string, so it fails, as expected.

If you want to supply a default value, you should do something like this:
[monkeycode]Local str$=""
Local J%=3
If str Then J = Int(str[0..1])[/monkeycode]
