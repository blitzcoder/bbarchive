Changing font size in IDE

Monkey Forums/Monkey Bug Reports/Changing font size in IDE

EdzUp(Posted 2013) [#1]
In the default IDE if you change the font size (say you change it from 12 to 14) it will cut off the bottom of the source files so you cannot scroll to the bottom in the IDE. You have to close the IDE and reopen it for the font to be resorted and the scrollbars to be the right size and position.
