Yasha(Posted 2012) [#2]
Breakpoints (for debugging?) are built-in to B3D: just use the Stop command and make sure to run your program in Debug mode (enable this in the Program menu, or by pressing F9 if using IDEal).


Les points d'arrêt (pour le débogage?) sont intégrés pour B3D: il suffit d'utiliser la commande <<Stop>> et assurez-vous d'exécuter votre programme en mode débogage (désactiver cette fonction dans le menu Program, ou en appuyant sur F9 si vous utilisez <<IDEal>>).
