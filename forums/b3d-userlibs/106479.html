Um, they're two completely different products.

Particle Candy was/is a 3D particle engine library.

Sprite Candy was/is a 2D single surface hardware-accelerated 2D graphics library (and more).

What you say you've supplied is Sprite Candy.

What you've *actually* supplied is Particle Candy, no matter how much the folder within the zip file is called "Sprite Candy" (and the folder within *that* folder says "Particle Candy").

(to be clear, they're both great products)
