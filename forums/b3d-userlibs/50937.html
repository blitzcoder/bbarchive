The library works with any "Universal" color code controled LCD glasses. It also supports lensticular 3D monitors, white cde operated glasses, blue code glasses (Thats high end VR stuff mostly) Micro-Pol 3D displays, Crystal Eyes origional, SynthGram monitors, and the Snaryo hologram generator / transmisson system.
It will support any digital projector that can run at 60 Hz or higher frame rate (requires a "universal" blanker dongle for this)
Alltogether, over 137 glasses, monitors, hologram systems and projectors are currently supported.
