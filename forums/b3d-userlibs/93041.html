I have found a way that works, though a bit awkward.

You will need Giles Lightmapper and AD3C, both (I think) are free.

Export from Blender as 3DS

Then import into AC3D. The texture will not show but load in seperately. UV works too.

Then SAVE as AC3D

Load into Giles, set up lights and render. Save as B3D.

It's taken a while for me to work this out, so it will save you time.
