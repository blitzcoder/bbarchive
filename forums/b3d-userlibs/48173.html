We've been having issues with our on-line score system in GEOM, we have been using GNET which works fine but hangs the system when it's trying to connect to the server. Several of our customers complained about it so something had to be done.

Wheel in the indieNet.dll

This .dll will allow you to post highscores to a .php file using the GNET format. It runs in a low priotity thread so it won't hang the game, it will keep trying to post the information and will die when it succeeds.

This is not a freebie, contact me if you would like further details.
