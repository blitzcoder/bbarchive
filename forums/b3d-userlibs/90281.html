Xors Team(Posted 2010) [#1]
As you already know, PhysX Wrapper for Blitz3d (and Xors3d) was abandoned after Render left XorsTeam.
To support users of PXW somehow, we persuaded Render to publish the source code.

Note. XorsTeam didn't change anything in the source code except adding a few comments and removing the code which checked the license key.
http://mycop.xors3d.com/public/PXW/PhysXWrapper.zip

Now XorsTeam is working actively at the implementation of physics simulation for Xors3d which should be connected with the core of the engine more tightly. More information is here http://area.xors3d.com/devblog/built-in-physics-first-test.html
