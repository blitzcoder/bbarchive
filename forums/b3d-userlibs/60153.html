VIP3R(Posted 2006) [#2]
The 'userlibs' directory is part of the Blitz3D install - C:\Program Files\Blitz3D\userlibs

Copy the EasyTok 'decls' file into the userlibs directory and you're good to go, it basically tells Blitz where to find the specified DLL and defines the functions used to access it.
