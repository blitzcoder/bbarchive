Shifty Geezer(Posted 2005) [#2]
I don't have any media but it wouldn't be too hard to compile a demo. Though without that I vouch for it whole-heartedly. My project needs to run on the lowliest of machines, and I've had it running comfortably on a 233 MHz K6. With the optimizations for local spaces for objects, elliminating uneccesary collision checks, it's performance has gone a good deal to. It's easy to work with (when you understand it ;) ), easy to add jointed structures to, stable and reliable (to date!), and with some nice features including specific JV-ODE functions.

The only thing it's lacking as an easy editor. Much of my coding has been to create a simple editor good for my own needs (basically adding cubes and spheres, and now ragdolls).

I say go for it. Well worth the money, and i haven't looked back.
