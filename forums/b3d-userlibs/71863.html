@ Beaker - Plugin to run Flash animations in Blitz. Originally mentioned here -

http://www.blitzbasic.com/Community/posts.php?topic=51053

Very effective and straightforward, although there's no documentation and zero support from the developer. He's never replied to any of my emails. IIRC the demo programs don't function correctly. Thankfully it's straight forward to figure out. The .decls explains all the functions pretty obviously.

I'm using it to embed animations in my app. I can use any Flash authoring tools (SwishMAX is good and cheap!) to create animations of any complexity, and embed them. The only alternative is creating your own animation system and authoring tools!

I'm a bit surprised it hasn't got more interest TBH.
