VIP3R(Posted 2009) [#2]
You can create infinite scenery using multiple mesh tiles.

It's also possible to create TriMeshes from code generated 3D meshes.

However, you must create the ODE TriMeshes before the simulation is started, it would be too slow to create them in real-time.
