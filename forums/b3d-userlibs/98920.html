With this library, you can load files .tmx created by the free tool Tiled Map Editor, in BlitzPlus and Blitz3D!!!
Forget create files .txt using characters as entities..., now you can make 2d games smoothly and with a minimum of simple commands.

Features
- Fast loading and easy to use
- Full control with a few simple commands
- Layer Control, tiles, objects and pieces
- Full documentation and easy to understand
- Unlimited layer, tiles, object and pieces
