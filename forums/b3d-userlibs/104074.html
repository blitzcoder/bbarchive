xlsior(Posted 2015) [#5]
Would anyone advocate getting a laptop with Linux instead of Windows?


Depends on what you want to do, really -- for most semi-professional use Linux is just fine, but it's definitely a 2nd tier platform when it comes to games. If you're an avid gamer, you really need windows. For everything else, you can make due with Linux.

It's better than windows in some areas, and sucks in others. If you haven't used Linux much, be prepared to have a learning curve, and you'll miss some software suites that you're probably familiar with and may miss when you have to give them up. (e.g. no photoshop, but there are alternatives like Gimp that give you most of the missing functionality)

Given that there rarely is a cost benefit to buying a system without windows, you may want to consider getting a windows box anyway, and install Linux as a dual boot system. That way you get the best of both worlds -- if it turns out that you need windows, it's an option. If it turns out you can get by with just Linux, you can always delete windows down the road.
