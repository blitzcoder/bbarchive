skidracer(Posted 2016) [#2]
The free version of mojo (graphics) that comes with Monkey2 gives you 2D with some pretty fancy lighting.

Monkey2 is showing a lot of promise for developing tools as it comes with a free gui module called mojox.

The monkey2 editor is called Ted2Go and it is a native monkey2 application. I think it is a good indication of how solid monkey2 applications are going to be.

I am not sure if anyone is using monkey2 to code finished games yet.

There are a bunch of demos that come in the bananas folder of monkey2 when you install that follow the Blitz Research tradition of a mix of Mark and community demos to get you going.
