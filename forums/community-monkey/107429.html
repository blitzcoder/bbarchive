degac(Posted 2016) [#3]
BlitzMax, MonkeyX, Monkey2 are all FOSS

Of MonkeyX there's a free version and different packagse (PRO, Ignition, Studio etc - see the Store page)

MonkeyX is multitarget and has complete mobile support (iOS, Android) and HTML5

BlitzMax (vanilla) has only target support (Max, Win, Linux) - BlitzMax NG (another open-source branch) has many targets (ARM, Android etc)

Monkey2 is still in development, main target is desktop+HTML5 (emscripten) - mobile support is still a WIP (not fully tested)
