Hi folks,

I thought I'd let you know that AGBIC 2012 is now in progress! It runs 17 June – 18 August 2012 so still penty of time to "get involved"

http://www.agbic.com

It is a competition that challenges game developers to create new games based on fictional game artwork.
We have an archive of 430 fictional game cartridges, so there is no shortage of inspiration for the indie developers out there.

The 2010 competition was run on TIGsource forums, but they have a policy to not repeat competition themes so gave me permission to run the competition elsewhere. It now has its own domain and friendly, fledgling community.

Prizes are still being finalised, but the winning games will be displayed at GameCity7 in Nottingham, UK, this October.

Any questions, let me know.

Cheers,
matt
