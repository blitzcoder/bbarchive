GfK(Posted 2013) [#7]
To overcome the window size issue, you should use virtual resolutions, and use the largest window you can within the desktop resolution, i.e. if the user has a 1280x1024 desktop, use a 1024x768 window. if they have a 1024x768 desktop, use an 800x600 window, and so on.

There are some weird-ass resolutions out there, especially with all these super-widescreen monitors that seem to be appearing, and of course, netbooks.
