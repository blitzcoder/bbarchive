So Double Fine is going to be putting out this pretty nerdy game in Summer called "Hack n' Slash," pun intended. The idea is you run around like Zelda but solve puzzles by hacking the actual game variables of the objects. This game really doesn't seem geared towards your everyday gamer and definitely towards programmers and those who like the idea of tweaking and modifying games.

Great Video: http://kotaku.com/its-like-zelda-but-your-sword-can-reprogram-the-entir-1548928671
Official Blog: http://www.hacknslashthegame.com/
Secret zip file with 3 files under the guise of a jpg using an old trick: http://hacknslashthegame.com/

I personally think this is a pretty clever idea, although I'm sure it's been done before but may have not been executed in quite the same fashion.
