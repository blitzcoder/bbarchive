Samah(Posted 2012) [#2]
You can fudge it by passing in an array and modifying the contents.
Function Main:Int()
	Local arr:Int[1]
	Test(arr)
	Print arr[0]
	Return 0
End

Function Test:Void(someParam:Int[])
	someParam[0] = 5
End
