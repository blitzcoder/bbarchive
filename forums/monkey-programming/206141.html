dawlane(Posted 2013) [#3]
First a Method should End with End or End Method not a 'Return' and there is a syntax error within the print statement +keys+ should read +key+

And if you ran the code once fixed in Strict mode then it wouldn't work unless you defined the return type in Compare e.g Compare:Int

Edit: You should aways use strict mode as this can trap any coding errors.
