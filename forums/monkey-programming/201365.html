GfK(Posted 2011) [#7]
Found a better solution. Download KeyRemap4MacBook (its not just for Macbooks).

http://pqrs.org/macosx/keyremap4macbook/index.html#t0p0

1. Install it, and reboot.
2. Go into system prefs>>KeyRemap4MacBook
3. Select the 'Change Key' tab.
4. Open up "For PC Users" (its near the bottom)
5. Check the first and third options ("Use PC style home/end", "Use PC style page up/down").

Done! My new keyboard is no longer a £40 door stop!
