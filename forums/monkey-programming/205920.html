SLotman(Posted 2013) [#7]
I'm still using Paint Shop Pro 7 here ^_^

I tend to mix PSP and Illustrator when drawing things - but for pixel art PSP is still the best there is for me.

Just tick the "show brush outline" in the tools option and then you suddenly have a very precise pixel 'pointer' on screen. Adding the grid you can set to any size you want (usually 8x8 or 32x32), I've yet to see a better tool for the job.

(I used it to do the graphics on my 'Escape from Alcatraz' game, only title screen and other stuff were made on Illustrator)
