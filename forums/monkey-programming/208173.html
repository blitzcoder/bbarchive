therevills(Posted 2014) [#4]
Nope not yet, but thanks for reminding me :)

As for a "cleaner" solution to generate a random seed you could do the following:
Strict

Import mojo

Function Main:Int()
	New MyApp()
	Return True
End

Class MyApp Extends App
	Field rndNo:Int
	
	Method OnCreate:Int()
		Local date:Int[] = GetDate()
		Print date[5] + "." + date[6]
		Seed = date[5] + date[6]
		rndNo = Rnd(1000)
		SetUpdateRate(60)
		Return True
	End
	
	Method OnRender:Int()
		Cls
		DrawText Seed, 10, 10
		DrawText rndNo, 10, 20
		Return True
	End
	
	Method OnUpdate:Int()
		Return True
	End
End
