ziggy(Posted 2014) [#3]
You can just create a substring
Function Main()
    Local myString:="Hello world!"
    Print myString[..1]
    Print myString[..2]
    Print myString[..3]
    Print myString[..4]
End

This outputs:
H
He
Hel
Hell
