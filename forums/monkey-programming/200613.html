Hi,

I've added a little module registry to the site, in attempt to keep track of all the modules people are working on, and to provide a way to allocate 'top level' module names.

See the 'module registry' link above.

If you want to register a module name, please email me with the necessary info - name, description, author, license, links.

And only modules currently in development please, not just stuff you may want to do one day, ie: no module squatting!

Try and keep module names unique and not too generic, preferably containing something like a proper noun or product name. Your user name is probably OK in most cases if all else fails.
