Just a quick warning: some of the functions in your assert.monkey conflict with Diddy's, so be careful what you import.

Example:
Import diddy
Import assert

Function Main:Int()
	AssertError("test")
	Return 0
End


Error : Duplicate identifier 'AssertError' found in module 'assert' and module 'assert'.


Very nice though. :)
