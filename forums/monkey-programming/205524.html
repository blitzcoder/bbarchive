Midimaster(Posted 2013) [#2]
does your text file has the Extension "*.TXT"? Thats important

here is a sample code:

Strict
Import mojo

Class Game Extends App

	Global T$,Lines$[]

	Method OnCreate%()
		SetUpdateRate 60
			T=LoadString("MyText.txt")
			Lines=T.Split("~n")
		Return 0
	End	

	Method OnUpdate%()
		If KeyHit(KEY_ESCAPE) Then Error ""
		Return 0
	End	

	Method OnRender%()
		Cls 0,0,0
		SetColor 255,255,255
		DrawText T,100,100
		SetColor 255,255,0
		For Local i%=0 To Lines.Length()-1
			DrawText Lines[i],100,150+i*20
		Next
		Return 0
	End	
	
End

Function Main%()
	New Game
	Return 0
End
