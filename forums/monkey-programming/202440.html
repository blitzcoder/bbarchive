Thanks, I had not set the Mass Data at all, after adding this, Force/Impulse works :)

Local massdata:b2MassData = New b2MassData()
massdata.center = New b2Vec2(0,0)
massdata.mass = 1.0
massdata.I = 0.0
playerEntity.body.SetMassData(massdata)
