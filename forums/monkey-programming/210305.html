SLotman(Posted 2015) [#6]
Even newer keyboards have this problem. It's called "key ghosting" - some keys won't register when pressing multiple keys at the same time.
That is the reason for some games using WASD as movement keys, because there is no "ghosting" effect with those.

More on the subject: https://www.microsoft.com/appliedsciences/antighostingexplained.mspx
