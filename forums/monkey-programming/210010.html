Check out the Cls command:

http://www.monkey-x.com/docs/html/Modules_mojo.graphics.html#Cls

Function Cls : Int ( r:Float=0, g:Float=0, b:Float=0 )


For example to change the background to red:
Cls(255, 0, 0)
