Yes, buy the frameworks. They can dramatically cut down on your dev time if you want to make serious projects.You should read as much of the framework's source code, and monkey's source code / the source in the examples folders.

Mojo2 enables some extra graphics functionalities than Mojo1, but it also still has some problems. You can transition to Mojo2 from 1 easily it just takes changing some rendering methods.

mx2 is still in development, is not prime time ready, but is also probably the future of anything monkey. If you are not planning on making commercial projects then I'd say go for it with using mx2 now.
