Hi,

Use an empty array, eg:

Import reflection

Class C
	Method Update()
		Print "C.Update!"
	End
End

Function Main()
	Local c:=GetClass( "C" )
	Local i:=c.NewInstance()
	Local m:=c.GetMethod( "Update",[] )
	m.Invoke( i,[] )
End
