ziggy(Posted 2015) [#3]
Qube, it's Lifetime.

Let me explain a little more:

This bundle updates are handle from the Monkey-X site, and it work the same way as in regular Jungle Ide, the IDE will detect new versions automatically, and allow you to download them and install them automatically too. The difference is that the version of Jungle Ide included in this bundle will also include the compiler, ignition modules, fontmachine modules and editor etc and will update them when required too. So it's also a time saver and makes setting up the whole Monkey X Studio bundle a breeze.
So sumarize: There's a new fix on Ignition? Monkey X Studio will install the fix. There's a new stable Monkey X compiler version? An update will be available when you start the IDE, etc.
