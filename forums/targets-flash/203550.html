dawlane(Posted 2012) [#4]
It can be stored in a number of locations. If you used a web browser it will be in where it stores it's data (on Vista/Win7 C:\Users\the_user\AppData\Local). It's usually in a folder called shockwave Flash.
If on the other hand you used Flash Player it's in ( Vista/Win7 C:\Users\the_user\AppData\Roaming\Macromedia\Flash Player\#SharedObjects ).
The easiest way to find it is to go to your user folder and use the search engine to look for gxtkapp.sol.
