Floyd(Posted 2011) [#3]
SuperStrict insists that variables and functions have an explicit type, even for integers. You can use % or :Int.
Function SystemParametersInfo2%(uiAction%, uiParam%, pvParam$z, fWinIni%) = "SystemParametersInfoA@16"
