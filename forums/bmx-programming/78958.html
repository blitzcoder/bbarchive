I haven't actually tried this, so use at your own risk (make a backup of your blitzmax folder(s) just in case) but... there is a text file /cfg/modservers.cfg which points to the latest syncmods folder.

Change the contents of that modservers.cfg file to say this instead:
http://blitzbasic.com/modserver126/

and try the following from the command line from inside your /blitzmax/bin folder:

syncmods -u username -p password axe

That ought to make it see the axe modules and download them... Remember to change modservers.cfg back to the correct 128 modeserver afterwards.
