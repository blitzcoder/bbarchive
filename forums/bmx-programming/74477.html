I've had this sitting on my harddrive for a while. So I thought I would rar it up and release it as a mod.

For those that remember the Amiga days, there used to be a solitaire/patience game called Klondike. This use REKO card sets so that people could have their own pictures on their card packs.

This module will let you load in those card packs of yesteryear as an AnimImage or single image.
