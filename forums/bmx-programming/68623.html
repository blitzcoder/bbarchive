Mark Tiffany(Posted 2007) [#5]
There have been some minor tweaks to the CVS version of the CE IDE since the last "official" release, but nothing significant. We have been slowly working on a revamped version (maxide2 in the CVS) which will hopefully "soon" gain an improved tokeniser and custom code area (yes, including folding)...you're always welcome to download the latest source, but bear in mind it's a work in progress.

The problem with the CE IDE is that there aren't many people contributing, and those that are don't have that much time to spare (I know I don't), but I would like to thank dmaz, Byteemoz, and grable for their contributions.
