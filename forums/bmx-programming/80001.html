skidracer(Posted 2008) [#32]
Here is a patch for makedocs, try adding the following to blitzmax/src/makedocs/fredborgstyle.bmx:

Global wikiURL$="http://blitzmax.org/index.php?title="

and after the block that begins with
			If t.example

add
			If wikiURL
				Emit "<tr>"
				Emit "<td class=docleft width=1%><a href="+wikiURL+t.id+">Wiki Page</a></td>"
				Emit "<td class=docright>iframe here???</td>"
				Emit "</tr>"
			EndIf


you then need to rebuild makedocs and run it, i used the following script saved as blitzmax/src/maketools.bat:

bmk makeapp -a -r -o ..\bin\makedocs makedocs\makedocs
