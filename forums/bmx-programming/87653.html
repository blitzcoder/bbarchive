You can get it here: http://www.blide.org/BLIde_manual.pdf

This manual covers most areas of current BLIde Plus version. It does not really contains a very 'deep' look to BLIde internals, it's just a reference manual with a brief explanation on BLIde functionalities. It has about 120 pages, but can be useful for people using BLIde and wishing to have a printed manual.

this PDF will be updated every once in a while to reflect latest changes in the IDE.

Also, next BLIde version will have a new help system, html based, that will be available as an online service (instead of being a local html setup) so BLIde updates will be smaller and help will be always updated.

This manual has been written by me. That means there will be some spelling issues and surelly some language mistakes. I'm doing my best but I'm not English!... So I would appreciate a lot any corrections language wise.

Hope you find it useful.
