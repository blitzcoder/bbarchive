plash(Posted 2008) [#4]
To change the current directory use:
ChangeDir "C:\"
ChangeDir "D:\"


EDIT: Or if you want the same path on a different drive you could look for "C:\" and replace it with "D:\".
Print "C:\WINDOWS\".Replace("C:\", "D:\")
