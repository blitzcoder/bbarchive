ziggy(Posted 2005) [#1]
there's a new free BLIde update for all BLIde users.
Some details on this update:



0.2.8 version
-------------

Conceptual change: Undependant files are now known as 'Additional files'

Fixed a bug that was not letting an application remember the 'additional files' added by hand

Fixed some tipos

Fixed resolution changes after a program is run in windows

Better debug treatment

Fixed Bug that was not showing the drive selection box in the configuration wizard

New graphics in the Configuration Wizard

Better Start-up window management

NEW FEATURE: Add-Ons manager. This little application will help you to organize additional applications you use while you're coding. It's very usefull to take advantatge of applications such as external debuggers or framework assistants


BlitzMax Help examples are now opened in the same BLIde session, avoiding reopning a new instance of BLIde

Lots of little bugfixing here and there




It's a highly recomended update.

The BLIde official website has moved its server. Now it's alocated at: http://www.blide.org

Let me know any bug at: supermanel*at*menta*dot*net

Thanks to everybody!
