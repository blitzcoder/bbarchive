xlsior(Posted 2009) [#4]
If you're using Brucey's BMK NG, it's easy to recompile in just debug mode from the command line:

-a = all modules
-d = debug mode
-h = threaded mode

bmk makemods -a -d
bmk makemods -a -d -h
