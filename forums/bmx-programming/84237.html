The base idea is to design a framework with help by the community for the community.
The framework will later when its more complete, contain all the stuff you need to write a game.
As you could do all the stuff yourself the framework should help people stop wasting time and allow them to create games.
Many people out there wrote some modules they never shared and everyone writes the same again which is definitly wasted time!
The Framework should, when its complete, contain nearly everthing you will ever need in any of your projects, and for sure it will be all good documentated and completely free!
