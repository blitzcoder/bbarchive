Byteemoz(Posted 2006) [#2]
The menu-entry raises the EVENT_APPTERMINATE-event:
Local window:TGadget=CreateWindow("My Window",40,40,320,240)
Repeat
	WaitEvent 
	Select EventID()
		Case EVENT_WINDOWCLOSE, EVENT_APPTERMINATE
			End
	End Select
Forever
