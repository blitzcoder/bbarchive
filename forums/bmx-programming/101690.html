ziggy(Posted 2013) [#8]
But what for do you want ASIO? Asio was designed to allow sample-perfect syncronization between the sequencer and the A/D D/A conversor. It's a driver model for realtime audio recording and processing software, but I can't see how it would be useful to be used with BlitzMax games. It would require users to define buffer sizes, preferred bit depth, choose the desired output channels. adjust latency, etc.
