Select Operator
			Case "="

				If Value1 < Value2 Then Return True Else Return False

				If Value1 >= Value2 Then Return True Else Return False

			Case "!"

		End Select



Yasha(Posted 2012) [#2]
What's wrong with:

Case "and"
    Return Value1 And Value2


...?


For that matter, doing the same for the others:

Case "="
    Return Value1 = Value2
Case ">"
    Return Value1 > Value2


...would be far more readable, and possibly more efficient too.
