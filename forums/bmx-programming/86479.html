N(Posted 2009) [#7]
Could do something like this:

SectionName.bmx
SuperStrict

Include "SectionName+A.bmx"
Include "SectionName+B.bmx"

SectionName+A.bmx
Type A
  Field zB:B
End Type

SectionName+B.bmx
Type B
  Field zA:A
End Type

Then just import whatever SectionName.bmx is.
