Hi All,

I've just release version 1.0 of my (free) LogFileHandler module for BlitzMax. It's actually a refactored version of my Logger module that was released a couple of years ago which is no longer available.

The module allows you to create and write date/timestamped messages to one or more log files. You can specify the level of logging you wish (or change it whilst running) and only messages that fit within that level of logging will be written. Therefore you can have enhanced logging whilst in Debug mode for example.

Log rotation is supported by disabling overwrite mode and setting a maximum log file size.
