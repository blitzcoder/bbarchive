Oh snap. Does this mean that x86 programs wont work on new OSs?


It does not. x64 is fully backward-compatible with x86 if the right libraries are installed - which they are guaranteed to be on all current versions of Windows. x86 isn't going anywhere any time soon.

Increasingly Linux distributions are leaving these libraries out by default to lighten the installation load, but they're only a quick `apt-get install`away on there too.
