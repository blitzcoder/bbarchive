Ok i came up with this, it's not pretty but works great.

For i = 0 To 23
If ChangeDir(Chr(67+i)+":\") = True
Print "CurrentDir()="+CurrentDir()
EndIf
Next

Improvement on the above code:
For i = 0 To 23
drive$ = Chr(67+i)+":\"
If FileType(drive) = 2
Print "Drive="+drive
EndIf
Next
