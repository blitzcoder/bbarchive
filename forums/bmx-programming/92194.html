Gabriel(Posted 2010) [#2]
I used Bass on the basis that it's more solid and dependable than OpenAL and Audiere, but a hell of a lot cheaper than fMod. I'm using Unity now, so fMod is free, but Bass is just as solid as fMod, it just doesn't have all the platforms and advanced tech. Bass is available for Windows and Mac, though, so if Linux is not an issue for you, and you don't mind spending a bit of money, I think it's still a good choice.

If you're releasing freeware, mind you, you might as well use fMod, which is free for freeware.
