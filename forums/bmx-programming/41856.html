AntonyWells(Posted 2004) [#2]
All types are derived from the object class/type.

So you can do this,

Type MyType
Field Test
End Type

A:MyType = New MyType

Obj:Object=Object( A )

I.e you can make any type an object and pass it around. And transform it into any other class below it in the class tree.
