Dabz(Posted 2007) [#2]
Extern "Win32"
		Function AllocConsole:Int()
		Function FreeConsole:Int()
		Function GetConsoleWindow:Int()
End Extern

AllocConsole()
Local hWnd% = GetConsoleWindow()
Print hWnd%
Delay 1000
FreeConsole()


Not cross platform (and complete), but I did write this five seconds ago while thinking the same! :)
