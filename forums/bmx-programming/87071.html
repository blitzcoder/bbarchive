Brucey(Posted 2009) [#6]
"Build Modules" is enabled by the presence of the environment variable MINGW.

This is what the IDE source says :
?Win32		
		Local mingw$=getenv_("MINGW")
		If Not mingw
			DisableMenu buildmods
			DisableMenu buildallmods
		EndIf
?
