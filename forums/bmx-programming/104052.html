ziggy(Posted 2015) [#4]
First, you need to sign the application as DPI aware. You can do this from the BLIde publisher. Mark the "GDI aware mark". Additionally, you'll have to ensure everything scales properly in your forms design. Be aware that windows allows both font scaling and DPI scaling, and having proper scaling there IS A PAIN. One of the things that is very badly supported in windows is native DPI scaling, it's usually a lot more complex than it should be, if you've used GTK, Swing or Mac Os. So...
