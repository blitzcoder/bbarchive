print LoadText("http::www.blide.org/version.txt")


fredborg(Posted 2006) [#2]
You need to specify the protocol correctly. BlitzMax does not use the same syntax as a browser. You should use "protocolname::" like this:
Print LoadText("http::www.blide.org/version.txt")
