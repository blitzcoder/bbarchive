ITSpeedway modules are as follows:

MaxGUI Stylesheets
A CSS stylesheet module for MaxGUI providing features that allow you to quickly change the look and feel of your application.

Behaviour Tree
This module provides the tools for writing in-game AI using a modular object oriented Behaviour tree.

Printing library
Library to provide options for Printing documents from BlitzMax.

Vector Fonts
A set of Vector Fonts to use in your games.

Windows Service/Daemon
Turn your game engine into a Windows Service/Daemon giving you the ability to write background Server applications in Blitzmax.
