Hello,

This is just to let you know that a new version of BLIde has just beed released which addresses several Windows 10 compatibility issues.

This new version requires Internet Explorer to be updated to (at least) version 11. That's the default version on Windows 10.

If you run BLIde on a very old system that does not support IE11, BLIde will default to an older version of IE, but webpages and help documents are not granted to be rendered properly if they use modern html standards.

Changelog:
BLIde version 15.09.2001 SP1
New: Added windows 10 compatibiltiy changes, including a new web browser
New feature: Gutter area can now highlight code blocks on mouse over
New: Text rendering speed has been improved
NOTE: Internet Explorer 11 or greater is required for this version to run. (it's the default minimum IE version on windows 10)
