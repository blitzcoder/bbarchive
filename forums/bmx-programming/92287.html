ziggy(Posted 2010) [#1]
A new version of the free FontMachine module is available for download in the BLIde website (here: http://www.blide.org/?p=145 )
Also, a new version of the commercial FontMachine editor that supports unicode is available in the BLIde Plus customer's downloads section.

This update allows the FontMachine module to load and draw unicode fonts, that support all available unicode alphabets, such as Cyrillic, Arabian, Japanese, Chinese, etc.

Hope you find it useful.
