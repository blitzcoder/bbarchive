I wrote this code a few years ago for a friend and forgot about it. Anyway: I thought it about time that I published it, so I've tidied up the code, improved the examples and turned it into a module.

The download comes with a module, 24 English Hershey Vector fonts and four examples (which also include the code to compile the fonts). Japanese Hershey font definitions are included but not compiled.

Based on information from Paul Bourke

I do hope someone else finds a use for it.
