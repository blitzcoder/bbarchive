REDi(Posted 2009) [#17]
Error = Attempt to index array element beyond array length (@line 30)

So just change this...
For Local sd:Int = 0 To 9
	Print "Sound driver " + sd + ": " + SoDrivers[sd]
Next

To this...
For Local sd:Int = 0 Until SoDrivers.length
	Print "Sound driver " + sd + ": " + SoDrivers[sd]
Next
