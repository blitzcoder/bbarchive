Kryzon(Posted 2014) [#1]
You use this module to support digitizer tablets such as Wacom etc. in your Win32 programs.
This module was written by Mikkel Fredborg. Special thanks to Timur "Taron" Baysal, the author of Verve Painter, for sharing his copy of this module.

You can download the module here: https://www.dropbox.com/s/2dmnak65fc0p27s/frecle.tablet.zip

The original thread for the module is this:
http://www.blitzbasic.com/Community/posts.php?topic=43618

Nicolas Porter has written a basic version for OS X:
https://github.com/surjikal/surj.mod
