EOF(Posted 2006) [#5]
3 Julian Date functions I posted in the code archives for BlitzMax:
Print JulianDay%("26 MAR 2006")       ' = 2453821
Print JulianDate(2453821)             ' = "26 MAR 2006"
Print DayFromDate$("26 MAR 2006")     ' = "Sunday"
