I can answer some of those:

Function pointers:
Local blah()

blah = func1
blah

blah = func2
blah

Function func1()
Print "1"
End Function

Function func2()
Print "2"
End Function

Static Data Members:
Type blah
Global thing#
End Type


Integers/Floats in BlitzMax are simple variables. Strings are objects.

A GUI module is in the works. I think I'm right in saying that it won't be free.
