Mark would not only need licenses for the next generation consoles, but he would not be able to make legal distributions of the port unless you were a licensed developer for the consoles. That doesn't mean that a community member can't hack one together using free libraries and cracked data. But an official port, usable by the 99.9 percentile, probably not. Unless tjhe second level of XBox arcade and the once-touted incubator program takes over Microsoft's mindset again or we see a PS3 Linux kit.

Console licensing is far from easy. It would be nice to see it, though, as it would definitely generate buzz. Unfortunately, I have a feeling that when a number of indie developers saw the cost of the console license, publishing agreements, and start thinking about their console shackles, it could turn rather ugly.

What would be fun is if something like the PS2 Linux kit is released and BMX ported to that, legally. Then you could prototype, prove, and port if you snatched up a license.
