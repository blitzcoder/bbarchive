I've discovered and fixed a few bugs in the past week. None have been reported so I didn't rush for a new release, but the bug fixes are on fairly important stuff, so now that I've tested them a bit, and ensured that everything works and nothing got broken in the process ( well, hopefully! ) I'm releasing this updated version. I didn't make a note of the fixes, just fixed them as I found them, but this release certainly includes fixes to the Shader, Camera, Scene and Mesh classes, and adds a couple of commands I missed wrapping in the Shader class as well.

EDIT: Sorry, I assumed everyone knew by now. For those who don't know, the module names have changed. Both Pro and Lite have been renamed to Glimmer.TV3D65 instead of Glimmer.TV3D65Pro and Glimmer.TV3D65Lite. Nothing else has changed. You still get source and support with Pro, and you still don't with Lite. None of the Functions, Methods or Types have changed. And it still only works with the current version of BMax unless you have Pro, in which case you can recompile for whatever version you want. ( Not that I can think of a good reason for not using 1.20 offhand. )
