Also to be able to squish it so the aspect ratio is different. A bit like that split screen on Sonic 2 VS Battle where the screen is rendered twice. So I can call the draw commands twice and draw to separate windows but in a different aspect ratio.

1) Set the projection matrix to the aspect and postition of the top screen
2) Render the scene normally
3) Set the projection matrix to the aspect and position of the bottom screen
4) Render the scene normally
