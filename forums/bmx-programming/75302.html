ImaginaryHuman(Posted 2008) [#1]
Okay, so whenever I try to come up with names for projects I do a google search and I get some really quite obscure companies that are already using the name or a whole list of weird products that are using it. Sometimes it escapes me exactly why certain products or companies or domains have decided to use particular terms, but I guess it's all a personal choice. Just when you think you've figured out a really cool name for something, you find it's been used already.

The trouble is that coming up with a unique name for anything seems next to impossible. It is ridiculous to think that given there are a finite number of words out there, anyone could have a monopoly on a word, because ultimately that must mean we run out of words eventually. (Same old issue with patents and the like, ownership always leads to a stalemate).

My real question is, I have noticed that several companies have the same name and several products do also. This suggests there is some kind of `context` within which it is acceptable to use the same name. Does anyone have any idea of what this context is or how it is defined, ie when is it safe to finalize a name that you want to use for a game or a games company? Does the name have to be absolutely unique or can it be the same as something else so long as the product isn't in the same market, or something?
