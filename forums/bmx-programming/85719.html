After opening an image file in the MaxIDE and it actually opening like bmx source code, I whipped this up as I thought it would be a nice addition to the official IDE. I made it with Blitz3D's media viewer in mind, so, if the idea is liked and accepted, the .exe will go in BlitzMax's bin folder and will be called from the IDE when an image is opened (BMP, JPG, PNG, TGA) like -> OpenURL("bin/imgviewer.exe ~qpathtoimage.png~q").

Let me know what you think!
