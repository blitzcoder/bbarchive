BMax 1.30 is bound to be your problem. Upgrade to 1.36.

	If KeyHit(KEY_F1) Then System_("./systemtest.sh")

in the main loop and this for the systemtest.sh...
#!/bin/bash

amixer -q -c 0 set PCM 255
amixer -q -c 0 set Master 130
exit $?;
