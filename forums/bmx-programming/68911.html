tonyg(Posted 2007) [#4]
Does this help?
Graphics 800,600,0
Local mousepos:String="IN"
While Not KeyHit(KEY_ESCAPE)
 Cls
 PollEvent()
 Select EventID()
    Case EVENT_MOUSEENTER
      mousepos="IN"
    Case EVENT_MOUSELEAVE
      MOUSEPOS="OUT"
  End Select
  DrawText MOUSEPOS,0,0
  Flip
Wend
