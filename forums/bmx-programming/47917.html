fredborg(Posted 2005) [#5]
Apparently you need to reestablish the OnEnd when it's been triggered once.
Graphics 640,480,0,0
	
OnEnd(quit)
main()

Function main()

	Repeat
		Cls
		DrawText "Press Escape!",10,10
		Flip
		Delay 100
	Until KeyHit(KEY_ESCAPE)
	
EndFunction

Function quit()

	ret = Proceed("Really quit?")
	If ret<>1
		OnEnd(quit)
		main()
	EndIf

EndFunction
