ziggy(Posted 2005) [#1]
there's a new BLIde update available.
for newcomers, BLIde is a free IDE for blitzMax, for windows.

This is the change-log of this version:


NEW FEATURE: Local shortcuts are now opened at the top-line of the editor window, to omptimize reading

IMPROVED: Learning module's keywords is now a little faster

IMPROVED: Auto-list auto desapearing when pressing [, (, and many other usual keys

NEW FEATURE: Find and Replace integrated in the docking environment

NEW FEATURE: Call Stack window with 'automatic' shortcut

NEW FEATURE: Refresh dependencies tree

Fixed error detecting the line number when there was a run-time error in the debuging application

Fixed BLIde wrong Z-order when there was an Update Available and application lost focus

Fixed BLIde crash when deleting selected text with backspace, when virtual spaces was on

Fixed a bug that was not handling correctly the folders, when using a specific notation

NEW FEATURE: Now BLIde generates an 'ERROR REPORT' on the console window, when a running application crashes. It provides call stack releated information.




Hope you'll find it useful.
