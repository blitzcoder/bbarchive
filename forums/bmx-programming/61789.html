Otherwise, setcolor, setalpha and drawrect the size of the screen?
Graphics 640 , 480
image:TImage = LoadImage("max.png")
DrawImage image , 0 , 0
SetColor 255 , 0 , 0
SetBlend alphablend
SetAlpha 0.3
DrawRect 0 , 0 , 256 , 256
Flip
WaitKey()
