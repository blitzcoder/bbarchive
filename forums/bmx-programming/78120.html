I think Matt is correct in what he said although it doesn't answer the original query.
Import "hello_world_private.bmx"
hello()
world()

with hello_world_private.bmx = 
Private
Function hello()
	Print "hello"
End Function
Public 
Function world()
	Print "World!"
End Function
