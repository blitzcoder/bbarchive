N(Posted 2010) [#11]
You can get that event information by implementing -scrollWheel: in an NSResponder subclass. BlitzMax already handles this in the brl.System module through the bbSystemEmitOSEvent routine, but only gets the Y delta from the scrollwheel event. I don't know why it ignores the X scroll, but it's pretty clearly not handling the event correctly.

Edit: See here: http://www.blitzbasic.com/Community/posts.php?topic=88534
