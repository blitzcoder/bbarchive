Yep, this is working, thanks...

Local Bank:TBank
Local Result:Int = 0
Bank = LoadBank(RemoteURL)
If Bank = Null Then
	Result = 0
Else
	SaveBank(Bank, LocalURL)
	Result = 1
End If
