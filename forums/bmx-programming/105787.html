The error says: you give a "Float array" to a variable defined as "Double array"

-> [1.0, 2.0, 3.0] is an array of floats

SuperStrict
Framework Brl.StandardIO

Global a![]=[1.0!,2.0!,3.0!]

Without those "!" you are actually passing floats.
