Russell(Posted 2011) [#18]
That error appears when you compile any of the example code that came with maxmod2, even after the module itself compiles without errors.

It seems that the maxmod2.modplayer module was replaced by the maxmod2.dumb module, but the example files were not updated.

Change 'Import maxmod2.modplayer' to 'Import maxmod2.dumb' (and move the above mentioned libdsound file) and you should be alright.

Russell
