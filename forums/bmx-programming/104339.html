xlsior(Posted 2015) [#2]
Depending on your MinGW version, you'll also need to replace a number of the MinGW libs into the Blitzmax/lib folder -- Any file that exists in blitzmax/lib as well as your mingw/lib folder, copy the mingw/lib version and overwrite the blitzmax/lib one with it.

There are some threads on this forum that contrain a command line script that can automatically do this for you.
