(Like if you click Save in MSPaint it will place "Untitled" in the File Name space).

  filename$=RequestFile( "Save ANSI file as",filter$,True )

Yep...
filename$ = RequestFile("Save ANSI file as", filter$, True, "Untitled.ext")
