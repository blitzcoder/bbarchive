I have recently wondered whether or not there is a TextMate bundle for BlitzMax, but after much searching I have found there isn't. Anyway I set about developing one myself, and although it is still very much in development, it is quite usable. It has fully folding code, syntax colouring and has basic compiling capabilities. (So far the code document has to saved, your copy of BlitzMax must be installed to <b>/Applications/BlitzMax/</b>, and it does not omit unused frameworks and modules from the the final executable, nor will it debug the program.)

I am providing it for download at the moment to make sure that everything so far is working and to get nsome feedback so far, and see exactly what is wanted in this bundle.
