Retimer(Posted 2007) [#5]
All I needed. Unfortunately it's only for windows =(

By Mark:
Strict

Extern "win32"
Function GetCursorPos( point:Byte Ptr )
End Extern

Type TPoint
	Field x,y
End Type

Local point:TPoint=New TPoint

Repeat
	GetCursorPos point
	Print point.x+","+point.y
	Delay 100
Forever
