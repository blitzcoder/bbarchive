The example source :
SuperStrict

Framework brl.glmax2d

Graphics 800, 600, 32
HideMouse

While Not KeyDown(key_escape)

	Cls
	
	DrawText "Escape to quit", 400 - TextWidth("Escape to quit") / 2, 300
	
	Flip


Wend


End
