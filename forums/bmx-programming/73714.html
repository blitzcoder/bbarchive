Rem
bbdoc: Numeric 'clamp' builtin function
returns: Clamps the number within two numerical extents.
keyword: "Clamp"
End Rem
Function Clamp:Float(value:Float,minvalue:Float=0.0,maxvalue:Float=1.0)
	value=Max(value,minvalue)
	value=Min(value,maxvalue)
	Return value
EndFunction
