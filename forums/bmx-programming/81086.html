Hi. Very Nice :-)

Just a couple of things you might think about fixing :

* If I run it from the dmg, it shouldn't crash because it can't save the high scores file. (perhaps, if it can't save the file, forget about saving altogether? You could always show the user a message to tell them their highscore won't save. Better than crashing ;-)
* Don't save the "q.dat" file in the same folder as the app. It's not the Mac standard. Either save it in the user's Library/Application Support, or Library/Preferences folder. Those are two standard folders for storing save data.
I have a module (bah.volumes) which can help you with locating the proper folders, if required.


The Apple website has a games/download area which might suit you. I believe Grey's game has been featured there.
