Custom mesh tiled terrain engine by Beeps2003
A Blitz lib that creates endless terrains using dynamic meshes tiled together. There's some single surface grass code in the demo as well - it's all a bit rough and ready but if you're at the point where you're trying this then it'll probably help. Enjoy!! 
grab it at www.binary-people.com/downloads/freemeshterrainlib.zip  (2.5mb including textures and a huge funky map.
