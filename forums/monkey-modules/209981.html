I created the following module since I couldn't find any cli parsing library. It's kinda posix compliant, but not too much :)

https://github.com/bitJericho/bit.arguments

To test it out, you'll have to compile the example, then navigate to it using the command line. Using the command:

main_winnt --help

Should get you started.

Let me know if there's any corner cases I missed!
