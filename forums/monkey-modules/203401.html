Description:
PureMVC is a lightweight framework for creating applications based upon the classic Model, View and Controller concept.

Based upon proven design patterns, this free, open source framework which was originally implemented in the ActionScript 3 language for use with Adobe Flex, Flash and AIR, is now being ported to all major development platforms.

Repository on GitHub:
https://github.com/KaaPex/monkey-puremvc

Original puremvc website:
http://puremvc.org/

Puremvc documentation:
http://puremvc.org/content/view/98/189/

On any errors report by email or in the tracker: https://github.com/KaaPex/monkey-puremvc/issues
