Hi, this is small module with two functions for stdcpp (also GLFW).
Built on a high resolution timer by Song Ho Ahn (http://www.songho.ca/misc/timer/timer.html)

Millisecs:Float()
Delay:Void(millisecs:int)

Example of use:
[monkeycode]Strict
Import cpptimer

Function Main:Int()
Local time:Float

time = Millisecs()
Delay(1000)
time = Millisecs()
Print(time)

Delay(1500)
time = Millisecs()
Print(time)

Delay(1671)
time = Millisecs()
Print(time)

Return 0
End[/monkeycode]
