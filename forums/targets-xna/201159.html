dopeyrulz(Posted 2011) [#7]
No - no stuttering at all. It's a Pitfall II remake so screen paging left/right and up/down scrolling which is super-smooth.

I have 4 layers - background, objects, front and score. There are around 50 objects all up which I loop through each (didn't need to do any sort of minimising here just visible/not visible). The thing I don't do is create them as I go - all done up front.

The tile blocks are 32x16, but the collision tiles are 16x16. I was going to scale up from 320x240 to 640x480 but the scaling bleeding caused some probs so just upscaled everything. Thought that might also help with the speed. There are some animated tiles for the water also.

I've nearly finished but work has halted my progress in the past month. Really happy with it all though. Will post up some builds (xbox and glfw/mingw) once I get it finished.

Oh - have the newest S model.
