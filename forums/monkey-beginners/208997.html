dawlane(Posted 2014) [#2]
For the Desktop it's easy enough to do. But the other targets are another matter.
Import mojo
Import os

#TEXT_FILES="*.txt|*.xml|*.json|*.html"
Function Main:Int()
	OpenUrl "file:///" + CurrentDir().Replace("\\", "/") + "/data/instructions.html"
	Return 0
End Function
