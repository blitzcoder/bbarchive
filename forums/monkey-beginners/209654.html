Yup after MUCH head scratching and cursing the day that computers were invented I finally found out what the problem was with my monkey X install.

File locations were all perfect, as was the paths in the config files.

The error that stopped it building on android was ... Multiple entries for JDK_PATH etc, I always thought that monkey checks them and chooses the one that works. It seems it doesnt so after deleting them all but the ones I set it now compiles perfectly.

Incidentally the error I got was it couldnt find the javac compiler.
