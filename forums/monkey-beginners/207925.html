Class GenerateFloor

Method qwer(blocks:List<Block>)

For Local x:= 0 Until 5
blocks.AddLast(New Block(x))
End
End

Class Block
Field position:Int

Danilo(Posted 2014) [#2]
To write
GenerateFloor.qwer(blocks)
directly, change this Method to a Function.


With Method, you first need to create an instance of the GenerateFloor class:
Local gf:GenerateFloor = New GenerateFloor
gf.qwer(blocks)
