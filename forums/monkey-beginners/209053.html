Samah(Posted 2014) [#3]
If you're using Diddy you could use the Arrays utility class.
Local bla:Int[] = [1,2,3,4,5,6,7,8]
' overwriting values
Arrays<Int>.Copy([9,10,11,12], 0, blah, 8, 4)
' joining arrays
bla = Arrays<Int>.Join(bla, [9,10,11,12])
