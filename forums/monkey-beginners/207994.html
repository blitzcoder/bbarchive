Goodlookinguy(Posted 2014) [#2]
# = Shorthand for Float
% = Shorthand for Int
$ = Shorthand for String
? = Shorthand for Bool

So
Field points#[]
Field inc#

is the same as writing
Field points:Float[]
FIeld inc:Float

I strongly recommend against writing the former, as the shorthand makes it less readable in my opinion.
