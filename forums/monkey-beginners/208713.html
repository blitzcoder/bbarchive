ziggy(Posted 2014) [#4]
On older Monkey X versions, you need to set the update rate on OnCreate.

Import mojo
Class Mygame Extends App
    Method OnCreate()
        SetUpdateRate(60);
    End
    Method OnRender()
        SetColor 111,1,44
        DrawCircle 1,1,55
    End
End
Function Main()
    New Mygame()
End
