Samah(Posted 2015) [#3]
If you look in the header of threading.monkey you can see:
C++ targets (GLFW, stdcpp, iOS, WinRT) require the TinyThread++ header files to be copied to the build directory.

And in thread.monkey:
' Don't forget to copy the header files into your build directory after the first build


So yes, that was the right way to do it. :)
