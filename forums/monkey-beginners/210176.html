ziggy(Posted 2015) [#2]
Once Studio is installed, you can launch FontMachine Editor to create you own bitmap fonts. It's an executable and should be available in the windows "start menu" or list of available apps.
Then, in any game or application you write, you can just import fontmachine module to use it on mojo applications.
There's also a mojo2 version of the fontmahcine module that is not yet being included in the default Studio package, but it can be downloaded here https://github.com/ziggybcn/FontMachine-for-Mojo-2
