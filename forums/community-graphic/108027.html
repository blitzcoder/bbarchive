RemiD(Posted May) [#10]
the graphics style is consistent (same modeling style, same coloring style) so this is a good thing, however i suggest you learn about "smoothing groups" so that the lighting/shading makes sense/looks good. (especially for the tree leaves and for the dinosaur)

Now you need to animate the scene, else it is not a demo/game but rather a sculpture !

You can use Fragmotion to "rigg" the dinosaur (add joints/bones), and then to "skin" it (set the influences/weights of some joints/bones over some vertices), and then to "animate" it (create animation sequences composed of poseframes and emptyframes), and then you can use Blitz3d to play these animations (AnimSeq(), AnimTime(), Animating(), Animate())

Try it !
