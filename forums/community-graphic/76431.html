Model Pack: Suburbs Volume 1 -

Need a set of models to quickly build a 3d suburb community in your application/presentation?
Need some housing models to fill in those barren spots on your 3d maps? This pack is for you.
Suburbs Volume 1 is a collection of low-poly, highly detailed 3d mesh geometry useable within any enviroment.
The models are highly optimized and trimmed - making use of every polygon. Nothing has gone to waste.
Low poly for use in real-time enviroments, games, presentations etc. ( model average polycount: 1000 )
The textures are high quality, photo-realistic normal maps.
Each model is a single mesh, external lighting details are saved in a text file for easy importing.
Each model contains the formats: Obj, X, GLS, B3D.
All textures and lightmaps are included.

This pack consists of -
5 - low poly, highly detailed house models ( including textures and lightmaps )
