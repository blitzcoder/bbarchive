This is my first project in Blitzmax - a remake of the quite simple but very exciting strategic game "Warlords 1" from 1989 :-D Does anybody remember it?



This is only a very basic techdemo for the tilemap display and movement on the map, it creates a INI-File where you can play around with some values. The tiles have an unusual size of 40x40 pixels so I needed to work with meshes and minib3d, press SPACE to see what I mean. The tiles are scalable without seams, only the map increases if you choose higher resolution / other tilesize. I still need to play around with it but looks nice so far. The map has a resolution of 109x156 so don't exceed the values - they will be cut.

I'm not sure if I'll finish this remake but I think I could complete it in Blitzmax (if not I'll release the project as it is). Funny: last year, a remake of Warlords 1 has been published for the iPhone (I don't own one - but the screens look nice). But I never saw a remake for the PC.

OK here we go: Download EXE [1.4MB]

Movement with Arrows / Mouse. You can investigate all terrain, cities and special locations. But no Action so far :-D
