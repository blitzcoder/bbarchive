3 new screenshots showing the revamped SRX track editor - it now has a proper GUI and can load and save to any filename, allowing the creation of loads of tracks.

Start mode placement - currently can place 4 start positions for network / local players -


Checkpoint placement - numbered checkpoints are placed around the track - they extend vertically to infinity so it is impossible for the cars to jump over them - they must just pass within the checkpoint x/z radius at that point -


And finally the track creation itself - the viewpoint can roam around freely, pieces can be rotated, placed and deleted at will -
