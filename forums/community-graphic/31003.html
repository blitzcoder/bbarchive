Converting from 3ds file format would lose the skeleton.

If you export to b3d file format using B3D Pipeline, Ultimate Unwrap can then convert to Milkshape file format. From there it can be exported to Milkshape ASCII, which CharacterFX can import. All while retaining the bones information for people to animate with in Milkshape and/or CharacterFX.
