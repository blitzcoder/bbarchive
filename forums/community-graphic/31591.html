Gabriel(Posted 2004) [#7]
It's not bad, and it could be a lot better if you tweak it a bit. Get a reference pic of a helicopter and tweak the top half of the model. The front is too flat, and the wrong angle. The windows are an odd shape and in the wrong place. With a reference picture, you could line those things up in about ten minutes and it'll look a lot better.

As Puki said, the texture is tiled too many times. It seems to be tiled about twenty times down the length of the vehicle. Unwrap it properly so that it only tiles once down the whole length and again, that'll make a huge improvement.

You'll be surprised how much difference that little bit of extra effort will make to the final result.

Looking at it, I'm pretty sure you made it without using a reference, and considering that, I think it's good work. I couldn't make something that good with no reference. But using references is the key to making those professional quality models.

Apologies, I should have been a bit more constructive here. Anyway, if I had to model a helicopter, I'd go for something like this:

I made this in about 15 minutes, so there's plenty of stuff to tweak here. But I do think that it looks like it could fly.

As Sybixsus said, you should use some reference when creating a model, especially when you've never done a model like that before. Think about the basic shapes. Think about the purpose of the thing (to fly, so you don't want to make it look like a brick). Get some images. For this image I used an image of a Apache and a regular chopper, just so I'd know where the basic parts go and a rough sense of proportion.

Good luck!
