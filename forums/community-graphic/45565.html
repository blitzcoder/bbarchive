No, this isn't some space art stuff, these are graphics from the game I'm working on (yes, holy crap, I'm actually making a game), whose working title is Big Bang -- not because it's big or anything, it's just going to have a lot of explosions.

Anyhow, first sprites I made for it.. (don't worry, I had code done before I started on these ;) ):

The fan goes inside the fan housing, which is attached to the player's ship. The explosion and bolt are weapon effects.

The game is '2D' (technically it's 3D since it's using BlitzMax) and uses a perspective similar to games like Gradius and Raptor (why is it that I'm the only person who remembers that?). Lots of explosions is the goal. Right now I've got a framework up and running and it's not difficult to add in enemies, weapons, etc. thanks to object oriented programming.
