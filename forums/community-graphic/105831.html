cysis(Posted 2016) [#3]
We have thought about making modular packs and the main reason we haven't done so yet is due to the fact that they are actually very unoptimised and can easily have mistakes that leaves holes or overlapping faces. In the end it causes more work for the user to get the built modular model into a decent state.

But we do want to allow for customisability, hense why we included 39 props with this pack. Amongst those, there are 3 variations of wooden walls (can be seen in last picture) so the tavern can be split up into various designs rather than having a pre-built shape. It does include a barebones version of the model which is just the main walls leaving the interior design completely up to you! :)
