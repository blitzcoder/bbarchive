Read the rest of this post if you want to know why I chose this the ARC compression format (long explanation), I just don't want anyone complaining about the ARC format.

ARC is a format that, like WinRAR's RAR format, supports recovery records. These recovery records hold a certain amount of redundant data that, in the event of the archive's corruption ("bad download"), can be used to (hopefully, usually) reconstruct the damaged part of the archive. When you're dealing with vector graphics you don't want any part of the vector graphic to become corrupted, else you might have to start all over again (unlike with text files, or, in many cases, bitmap graphics). I was afraid that if I didn't put it in an archive that it would be regarded as a text file (that is essentially what XML is), and I've had problems in the past trying to upload text files to file hosting websites.
