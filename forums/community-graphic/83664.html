I'll repeat it all here (minus the images):

Contains:
4 Junctions – Straight, Corner, T Section, X Intersection
3 Corridors – Wide, Thin & Asymmetric
1 Stairwell
4 Basic Rooms
All conforms to a grid with x/z sizes in size increments of 20/40 units. Vertical increments of 21 units per height level (for use with stair wells and double storey rooms)

Low poly collision meshes included (make sure they are transparent/not rendered) (in directx .X format)

All pieces are in b3d format and contain both a color map and a light map.

Level editor included which allows you to place pieces in your desired arrangement. Exports to a text file containing x/y/z/yaw values for each piece. Sample included (Test.MAP)

Sample code included for loading pregenerated level (Test.MAP) and moving about within it (blitz3d code).

Download here:

http://home.people.net.au/%7Etmalcolm/contentpack/SciFiContent.zip

Hope you can make some use out of it.

Depending upon how well received this is I may make some more stuff.
