Kryzon(Posted 2010) [#5]
How can you be doing it wrong? host your image on a website like http://www.imageshack.us , grab the direct link and put it inside a tag like this:

[img link]

Replace 'link' with the direct link you copied. It does not matter if you write img, Img, ImG, iMg or IMG, any case works.

If not even pasting the direct link to the image is working, I'm guessing you are not retrieving the correct direct link from the hosting website.
