Rob Farley(Posted 2007) [#8]
Bends are the tricky bit! Well roads that don't follow the grid.

I came up with a system (although never completed it 100%) where you could draw on roads at any angle, which allowed you to create curves (made out of several straight bits).

The main map was a height map with a grid system for the main terrain.

It was pretty straight forward working out which grid boxes shouldn't be there for the road mesh to be inset in.

The first tricky bit was creating the additional triangles to join the grid up with the road mesh, that in the grand scheme of things was pretty easy.

The bit that was hard though which I never got right was working out the uv coords on these extra triangles so that the surrounding grid textures flowed into the road correctly.

And road intersections never worked properly either, again, confused uv maps were the culprit again.

Would have been cool to have got that working though!
