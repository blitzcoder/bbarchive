So I made this:



This is my very first piece of art using a tablet!!! I call it the "Space banana". I practiced shading and highlighting and for my VERY first attempt, i'm happy! I didnt see it as a "3d" object from the very beginning so i didnt highlight it like a round object but rather like an embossed 2d object thats sticking out. Hope you dig it ;)

For those who are experienced texturers and might give me advice on shading read on:
I tried creating a brush (in Corel) that is a circle which adds 1,1,1 (RGB) when painted. I hoped this would make it subtle enough, but when i drew with it, it actually made the pixels darker by about 3 or 4 depending on the colour under it. I tried adding opaqueness and transparency to it but it seems that as soon as i add this, the brush disappears as corel thinks the pixels of the brush are 100+% transparent. If i drew with this i couldn't avoid having clear lines that are visible and show where the shading starts. So i made a noisy brush with varying pixel darkness and set its transparency from 93 to 95% to control the actual darkening. This seems to work well, only that the results are not as smooth as some shading might require. The plan was to shade it very high res and then downscale it and make it more blurry and smooth, but that somehow ruins the artistic value of this particular piece so i didnt do it now. I will probably use the same technique for shading from now on, but also use blur and/or resize with anisotropic filtering to achieve the smooth (not noisy) shading. I know most of you use something other than Corel, but an explanation of how you shade the very fine shades using any tool will be welcome.
