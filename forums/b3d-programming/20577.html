Skysaw(Posted 2003) [#2]
Could be that you're using the same identifier within two nested loops:

For planet.ssb = Each ssb
...
For planet.ssb = Each ssb
...
next
next

There's no way to distinguish the two planet variables. At the very least, change one of them so you can tell which is the problem.
