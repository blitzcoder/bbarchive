Stevie G(Posted 2004) [#2]
Something like this should work - not tested ....


For s = 1 To CountSurfaces( mesh )
For v = 0 To CountVertices( s ) - 1
TFormPoint VertexX(s,v), VertexY(s,v), VertexZ(s,v), Mesh, 0
x# = TFormedX()
y# = TFormedY()
z# = TFormedZ()
Next
Next
