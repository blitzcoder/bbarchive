Ricky Smith(Posted 2004) [#4]
There would be not much point in converting a .b3d to a .3ds unless for use in a non Blitz environment which only supported that format.
The .b3d format is far superior in terms of what it supports and is more ideally suited to Blitz3d.
Some of its features such as skeletal animation could not be converted to a .3ds format which only supports mesh animation etc.
All modellers that have a .b3d export option always have a .3ds option as well.
You can use Quill3d or UUnwrap to import a .b3d and then export to .3ds but only the features supported in the .3ds format will be exported.
