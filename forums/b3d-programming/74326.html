Dreamora(Posted 2007) [#3]
The description does not help much as Blitz3D has its own loaders:

3DS: Only a single texture and UV unwrap + multiple meshes which can change their position.
No bones, no multitexturing, no multiple UV

X: multitexturing but only a single channel of UV. Bones can be in, but bone based animation is not supported.
Shininess is loaded from it I think as well as vertex colors.
Blitz3D does only support the DX7 X format, not the DX9 one!


b3D: bone based animation with vertex weight, 2 channels of UV, multitexturing, can store the EntityFX, TextureBlend etc settings
