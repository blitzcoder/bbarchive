Jeppe Nielsen(Posted 2003) [#6]
Use an array of types, then you can easily add more features to the terrain by adding more fields to the type.


Dim Terrain.tile(Wid,Dep)

Type tile

Field Height#

Field Walkable

End Type


This is what I have done in my RTS
