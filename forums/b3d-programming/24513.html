I just bought Cartography Shop. It's far superior to Quill in my opinion ( that IS just an opinion though ) for level editing, as the interface is very, very intuitive. Quill is undeniably powerful, but it suffers from a cluttered, unintuitive interface and has more features than you need for a level editor. Cartography Shop cost me £40, but it's worth every penny in my opinion.
