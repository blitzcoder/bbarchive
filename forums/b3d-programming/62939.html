kevin8084(Posted 2006) [#3]
Arrays are global. You don't need to explicitly return an array from a function. Changing the array within a function changes it globally. For example:
Dim a(10)
For x=0 To 9
	a(x)=x
Next
For x=0 To 9
Print a(x)
Next
doit()
For x=0 To 9
Print a(x)
Next
WaitKey()

End

Function doit()
For z=0 To 9
	a(z)=7
Next
End Function
