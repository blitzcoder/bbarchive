This is my tokamak command refrence which is up-to date with tokamak v.5. For a walkthrough on tokamak basics, check out my tutorial on blitzcoder - http://www.blitzcoder.com/cgi-bin/articles/show_article.pl?f=bot__builder102082004102230.html .

You may notice it creates two versions of each command. this is because one has a .htm extension for blitz to call for quick help and one is for the index, with a .html extension. Maybe I can request that blitz attempt to open html files aswell. (I was to lazy to redo all the links). This is one of the reasons the zip is so big - 7z looks for interfile relationships.

Put one of these 3 in your blitz folder and extract/run. You might want to backup help/index if you are paranoid. this is a file that came with blitz and is over-written to provide a link to the command reference. Installs right over a previous version.

http://blitzstuff.250free.com/zips/TokamakHelp.7z (25.515K - You only need one of these)
http://blitzstuff.250free.com/zips/TokamakHelp.exe (95.659K - You only need one of these)
http://blitzstuff.250free.com/zips/TokamakHelp.zip (275.579K - You only need one of these)
