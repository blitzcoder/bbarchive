Vertigo(Posted 2007) [#4]
I was able to get blitzbasic to work under Parallels, however, it will not work with blitz3d.. you will get a "Cannot create 3d scene" errors. So as long as your using graphics, and not graphics3d to init the scene then you will be ok. Parallels is simply Virtual Machine software... Its just like using windows Remote Desktop via terminal services... you can get 2d rather very slowly, but 3d doesnt work. 3d will work with open gl somewhat sluggishly however as a side note.
