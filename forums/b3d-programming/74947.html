Dreamora(Posted 2008) [#2]
Use onigirls B3D Export pipeline. See 5 postings below this, posted by Pudding

3DS Files do not store any animation at all so the export actually removes the animation.


YellBellzDotCom(Posted 2008) [#3]
One other option is Ultimate Unwrap 3d, it was cheap, gives you the u3d file export from max. Then you can use Ultimate Unwrap 3d to clean up the file, texture a model, modify a model or just use it as a format exporter. This is the method I use due to the B3d pipelines finicky demands.


Pongo(Posted 2008) [#4]

3DS Files do not store any animation at all so the export actually removes the animation.



Actually, .3ds files DO store animation, but not skin deformation animation. Transformation (Position,Rotate,Scale) animations work just fine.

B3d pipeline is definitely the way to go though,... I just wanted to point this nitpicky thing out.
