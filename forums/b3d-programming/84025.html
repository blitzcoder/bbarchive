Ricky Smith(Posted 2009) [#6]
Pacemaker can do that. First load the model and then from the file menu select "Load Sequence" - select ".B3D" from the drop down and then load the sequence required. It will appear as a new animation sequence. Once all the sequences have been loaded save the model with all the sequences in one file.
