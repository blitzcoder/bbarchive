PowerPC603(Posted 2004) [#5]
Do you clean up after unpacking your stuff with Terapak?

It could be, that a certain file was unpacked, and if you unpack it again (by loading it again), that terapak cannot unpack the file, because it already exists.

After loading an image and/or model without textures, use DLPak().
After loading several files, use PakClean().
This should delete all unpacked files that exist, so you can unpack them again (and reload them).
