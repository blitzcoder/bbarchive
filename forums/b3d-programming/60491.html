it was a .b3d from Milkshape, not .x... it animates ok in Blitz3D, but... you gotta...

1- make sure the Anim button is unchecked when you are exporting from MS (you are not in the anim mode)...

2- make sure that your animate command happens once, if in the main loop, then check...
If Not Animating(stickman)
 Animate stickman,1,1,1
EndIf

3- it won't work (well it didn't work for me) without first calling extractanimseq() before starting an anim... 
ExtractAnimSeq stick,0,22
this way is cool because you can set up different animations for idle, run, etc... and play em back selectively...

hope this is of some help... i'm using Milkshape 3.1.7.4
