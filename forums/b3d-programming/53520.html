Ricky Smith(Posted 2005) [#2]
It depends really on what you mean by edit. If you want to create a .b3d from scratch then many modellers have an export option for the .b3d format - 3ds Max ( for the rich) and Milkshape ( for the poor like me !)are commonly used. There is a free exporter for Max (.b3d pipeline) which is excellent. Milkshape has a plugin exporter for .b3d.
There is also the option to use Quill3d which is an excellent tool for the .b3d format as its written in Blitz3d. It includes an very good level builder.

For lightmapping gile[s] is probably the best.

For animation you can use CharacterFX or Fragmotion - Fragmotion is newer and has an excellent range of import/export options including .b3d of course.

Shambler(Posted 2005) [#7]
Gile[s] is an excellent tool for editing .b3d, you can do everything bar animation and vertex editing.

Yes you can have as small a light as you want and in any colour! =)

As for 3D modellers I would not go with Quill since as far as I am aware it is no longer supported and my own experience with it is that it does have a few bugs.

3D World Studio is a much more powerful and better supported application for level building and exports to .b3d too.
