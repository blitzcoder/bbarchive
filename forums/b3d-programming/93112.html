6(Posted 2010) [#2]
The manual is more of a reference manual, rather then a programming manual. Its about 400 pages in length and the first 100 pages give a very basic overview on programming in Blitz3D; no more than what you can learn just by looking an example code. The other 300 pages are just descriptions of each function in Blitz3D.

Overall I was disappointed in the manual as I was expecting lots of example code on how to develop games, but there isn't much at all. I can't honestly say I would recommend it :(


Kryzon(Posted 2010) [#3]
If you want real programming logic with Blitz3D (well, it's actually in Blitz2D, but it works for Blitz3D nonetheless), buy Krylar's book: Learn to program 2D games in Blitz Basic. It's only for 2D.

If you want true 3D logic programming, search in the old (but gold) BlitzCoder tutorial archive: http://membres.multimania.fr/blitzcoder/articles.shtml - tons of stuff for Blitz3D, I've learned a lot from here.
