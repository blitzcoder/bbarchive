John Blackledge(Posted 2008) [#5]
Resource Hacker:

File / Open - open exe.
Open the icon tree until you see the prog's icon in the main window.
Action / Replace Icon
Click 'Open file with new icon'.
Click exe, dll, ico; click replace.
File / Save.
Exit

It sometimes takes a few minutes for Explorer to realise what you've done - the icon image it displays may be the old one, until you reopen Explorer.
