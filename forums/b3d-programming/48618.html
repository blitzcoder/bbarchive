tonyg(Posted 2005) [#2]
Does this help...
Graphics 640,480
SetBuffer BackBuffer()
While Not KeyHit(1)
   Cls 
   my_key=getmykey()
   If my_key=0
       Text 0,0,"No key hit"
   Else
       Text 0,0,my_key
   EndIf
   Flip
   WaitKey()
; Take out waitkey for keydown for better test.
Wend
End
Function getmykey()
	For x = 1 To 237
	    If KeyHit(x) Return x
;           If keydown(x) return x
	Next
End Function
