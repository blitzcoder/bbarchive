ICECAP(Posted 2006) [#2]
For landscapes, definatly T.ED (its in the tools section)

For trees go for Aliencodecs tree generation software, BlitzTrees is good if you want stuff like wind through your trees. But plantlife and treemagic are more speciffic, create your custom tree/plant/flower/fern ect.
The only thing is there is no animation on them, you would have to manually create the animation in a seperate 3D program.

T.ED:
http://www.blitzbasic.com/toolbox/toolbox.php?tool=137

Aliencodec:
