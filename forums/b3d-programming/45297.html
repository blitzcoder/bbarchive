Craig H. Nisbet(Posted 2005) [#4]
It's crazy fast, and has the best HUD/Layer system I've ever seen! Even has a kick ass, dumb animation system like entity animation! Tell the hud/layers to animate, and forget them. No annoying animation loop coding!! I was using n-sprite before, and it's great for what it does, I don't thing Sprite Candy supports pixal perfect images, and it doesn't support an easy transition from a 2d app to a 3d app like n-sprite, but there are so many more features for use as menus and huds in this system. I actually bought for the font generater and was annoyed that I had to buy Sprite Candy to display the fonts, but I was pleasently surprized to say the least. I actually think I'll continue to use n-sprite for the map generation and Sprite Candy for all the rest.
