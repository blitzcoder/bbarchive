Matty(Posted 2009) [#3]
Particle Candy is reasonably good.
It does have a minor limitation which can be gotten around if you code for it - if the number of vertices or triangles on the single surface exceeds the usual limit (32k i think) then it will crash on renderworld - standard blitz behaviour in this situation, but it is possible to fix this by simply disallowing any more triangles to be created once that limit is reached (unlikely but better than having your simulation crash for an unknown reason and with an untrappable error code)
