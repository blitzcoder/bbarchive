Chad(Posted 2007) [#4]
earth=CreateSphere(32,1)
PositionEntity earth,0,0,0
earthatmos=LoadTexture("earth_clouds.png",2) ; using b32's PNG
EntityTexture earth,earthatmos

Quick code bit =)
