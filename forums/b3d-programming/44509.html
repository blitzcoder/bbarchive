GfK(Posted 2005) [#3]
Something like this:

a = 0
Repeat
	a = a + 1
	If a > 180 Then a = -180
	dir = ((a+180)/45)-3
	If dir < 1 Then dir = dir + 8
	DebugLog A+"  " + dir
	Delay 50
Until KeyDown(1)
