Source code for any major game project is generally a lot to ask for. There are plenty of projects at www.blitzcoder.com you should have a look at.

At some point, you're going to have to just dive in and learn for yourself. If you're not such an experienced programmer, then don't try to start with the ultimate game project. Start with something reasonably simple and develop your skills from there.

You might also want to visit www.gamasutra.com for articles about specific game design subjects, presented by members of the commercial game industry.
