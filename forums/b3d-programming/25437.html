An awsome tool for creating textures is Ultimate Paint, a freeware version is available at the following link.
http://www.ultimatepaint.com/index.html
What sets this paint program apart from all others I have seen is
the ability to load any image up as a brush and then use it to paint with like a brush.
Also you can select any area of the image loaded and use it as a paint brush or save it.
Any image loaded as a brush can be resized and rotated before painting with it.
Ultimate paint is a 32 bit version of the old Deluxe Paint we used to love in the Amiga Days.
I always missed that program. Nothing I found could match it, until now.
