If you are talking about Multi Texturing then besides the ones I already showed you before(in earlier posts), no others. But I can only say what I did: I created one super texture that covers the entire terrain and then a detail texture, plus one texture for lighting / shading / scorch marks(after explosions). This for me gives a satisfying effect, but could be better if one could switch the detail textures in certain areas. For that the same technique as for grass could be applied I guess, but could also kill the framerate.

For the super large texture I used earthsculptor: http://www.earthsculptor.com/

But you are using BlitzTiles right? Doesn't that allow you to add different textures?
