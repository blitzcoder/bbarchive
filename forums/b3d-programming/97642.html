Yasha(Posted 2012) [#2]
There are some shuttle scans of the Earth here: http://www.webgis.com/srtm3.html

The resolution isn't very high (in videogame terms), but I don't think you'll find maps with better resolution that that which weren't created by hand because trees and buildings and so on would presumably interfere with the terrain if the cells got any smaller.

Don't ask me how to load the data into Blitz3D, I have no idea and doubt anyone has done it before.
