For the record,

There WAS a bug in FreeImage's RLE compression for 8bit BMP's...
This bug has been fixed, and the open sourcode updated (specifically PluginBMP.cpp). The distributed dll hasn't been updated/released yet, so you'll have to download the complete source code and compile a new FreeImage.dll yourself.
See http://freeimage.sourceforge.net
