Hujiklo(Posted 2006) [#2]
Can you view a preview of the file in the exporter? If yes and it simply crashes on export then you can have a successful export by doing this:
Simply click the 'preview' in the exporter.
After previewing quit out.
Then find where you placed the B3D exporter 'dle' file (usually in your plugins directory) inside your Max installed folder and locate the file 'temp.b3d.' inside this directory
Rename this file to what you want as it is a perfect exported file.

Sometimes I have the same problems with crashing on export, but if it previews then it's safely copied to you plugins directory as a temp.b3d!
