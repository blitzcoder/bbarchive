GfK(Posted 2003) [#2]
You'll have to export it as either MD2 or B3D format. 3dsmax doesn't do either unfortunately.

There is a Halflife SMD exporter somewhere for 3dsmax. So export to that, load into Milkshape, then export as MD2 or B3D.

Make sure to look for the answer to your question among existing threads. This specific question gets asked a lot so you don't even need to look deeper than the first page. Here's my reply in the thread just below this one:

The ideal path from 3D Studio Max to b3d file format is currently using Ultimate Unwrap. Ultimate Unwrap has a 3D Studio Max exporter; export to UU and then save as b3d.

Several people are working on b3d exporters for 3D Studio Max.
