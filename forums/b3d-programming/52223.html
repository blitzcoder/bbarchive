jhocking(Posted 2005) [#2]
.max is the 3D Studio Max file format. To open them, you need 3D Studio Max, and even then you may not be able to open them if the files were saved in a different version of the software (specifically, if the models were created in a later version than you have.)

.tga is an image file. Open it in an image editor (eg. Photoshop.)

Regardless, yes you do need to convert to a file format Blitz supports. For 3D models these file formats are .3ds, .x, .md2, and .b3d
Note that Blitz supports the various file formats differently however, so depending on what exactly you need you cannot simply convert to whichever.
