Rottbott(Posted 2004) [#4]
Pretty simple.

Function CountMeshTriangles(Mesh)
  Local Tris = 0
  For i = 1 To CountSurfaces(Mesh)
    Tris = Tris + CountTriangles(GetSurface(Mesh, i))
  Next
  Return Tris
End Function
(Untested)
