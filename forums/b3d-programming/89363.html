hi try this:

ExecFile "readme.txt"	;loads the editor associated wiht .txt files
WaitKey

ExecFile "notepad.exe"	;loads notepad alone

Print "fin notepad "
WaitKey


ExecFile "notepad.exe readme.txt"	;load notepad with "readme.txt" as command line
WaitKey
