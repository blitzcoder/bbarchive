It sounds like something you might be able to use aligntovector for...

vx#=EntityX(ship,1)-EntityX(planet,1)
vy#=EntityY(ship,1)-EntityY(planet,1)
vz#=EntityZ(ship,1)-EntityZ(planet,1)
