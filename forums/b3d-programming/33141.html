fredborg(Posted 2004) [#4]
Something like this:
type myglobals
  field var1
  field var2
  etc...
end type

global globs.myglobals = new myglobals

globs\var1 = whatever
globs\var2 = whatever
etc.

And then just access them using "globs\variablename" throughout the code.
