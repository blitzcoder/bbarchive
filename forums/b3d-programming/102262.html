Blitz arrays insist that you do not use the "type tag" after the array is created. Use the name only:

Type auto
	Field  Num%
End Type


Local Auto.auto[10]  

;Auto.auto[0] = New auto ; Error.

Auto[0] = New auto ; No Error.
