big10p(Posted 2006) [#2]
The only way to do this is to store the array size in a variable, when you dimension it. Or, you could use a bank, instead. You can then use the BankSize command to get it's size.

    If PeekInt(bank,i) = 0
        free_pos = i
        Exit
    EndIf
