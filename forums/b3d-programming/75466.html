Never used them myself but I'm pretty sure the RGB components define a vector representing the normals for lighting ...

Nx# = ( Red - 127 ) / 127.0
Ny# = ( Green - 127 ) / 127.0
Nz# = ( Blue - 127 ) / 127.0

There are loads of discussions and code re this.
