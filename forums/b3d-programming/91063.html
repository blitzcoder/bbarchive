Charrua(Posted 2010) [#3]
try this:

Local myArray[9]
Local i

For i=0 To 9
	myArray[i]=i
Next

Print "myArray before"
For i=0 To 9
	Print myArray[i]
Next


AddOne(myArray)

Print
Print "myArray after"
For i=0 To 9
	Print myArray[i]
Next

WaitKey

End

Function AddOne(a[9])
	Local i
	
	For i=0 To 9
		a[i] = a[i]+1
	Next

End Function
