Jeppe Nielsen(Posted 2005) [#2]
use the millisecs() command


;five minutes in millisecs = 1000 * 60 * 5
timer=millisecs()+(1000*60*5)

;main loop
repeat

;check if the five minutes has passed
if millisecs()>timer

;reset the timer again
timer=millisecs()+(1000*60*5)

goto yourlabelhere

endif


until keydown(1)


hope this makes sense :-)
