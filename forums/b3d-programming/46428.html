Beaker(Posted 2005) [#2]
You can pass Blitz arrays like this:
Local myArray[5]

myFunction(myArray)

Function myFunction(thisArray[5])
;blah
End Function


The only way to replicate a STEPped variable is to use a Repeat or While:
i% = 1
While i <= FINISH%
;blah
i = i + VAR%
Wend
