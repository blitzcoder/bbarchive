8 for me. In any case, the return value of GetKey is an ASCII code, not a scan code.

14 is indeed the Backspace key for my keyboard.

Try this code:
Time = MilliSecs()
While Not KeyHit(1)
	Time = Time + 8
	Repeat
	Until MilliSecs() > Time
	Text 2,2,"KeyDown(14) returned "+KeyDown(14)
	Flip
	Cls
Wend
