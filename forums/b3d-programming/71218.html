GfK(Posted 2007) [#2]
I'm guessing you're after the alpha flag (2).

You've tried 12 (easier to express that as 4+8), and 49 (1+16+32). Try stuff with the alpha flag in.

You should probably use a PNG image (with an alpha channel), too, if you aren't doing already.
