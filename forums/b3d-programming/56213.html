BIG BUG(Posted 2006) [#18]
There is a B3D-exporter:
Download for Cinema4D Release 6/7(450kB):
http://www.mein-murks.de/software/B3DExport.zip
Download for Cinema4D Release 8/9(450kB):
http://www.mein-murks.de/software/B3DExport_Rel89.zip

There is no english documentation yet, so here are the most important features:
Texturemapping(UV-Tag has to be assigned to every object), Animation(Position/Scale/Rotation), Bones

Important: Only polygonobjects are exported with surface. The others are exported as pivot.
So use "Make editable" on every object before export.

To use this plugin, just extract the zip file into your Cinema4D\Plugins-directory.
