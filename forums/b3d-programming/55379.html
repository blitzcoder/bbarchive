jhocking(Posted 2006) [#3]
Pacemaker is the only free tool I can think of, and I'm not sure the free version is available anymore. Maybe Slim Shady?

There are plenty of cheap options however. CharacterFX is the cheapest, at $15, but I find the 3ds import kinda flaky. Milkshape is the next cheapest ($25) and it is much more reliable for 3ds import (plus at imports/exports a laundry list of file formats.)

My favorite tool for file format conversions is Ultimate Unwrap. It's $50 and worth every penny.

ADDITION: I suppose you could just use Blitz to do this conversion. After all, it can load 3ds models, and the Specs and Utils link at the top has code for saving b3d files.
