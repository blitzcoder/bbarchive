Using so many functions it really bad practice. This can be really simplified.

Graphics 800,600,32,3

While Not KeyHit(1)
	AskQuestion()
Wend

Function AskQuestion()

	q = Rand(1,20)
	Restore Questions
	For l = 1 To q
		Read Answer$, Question$
	Next
	
	Print Question$
	Guess$ = Input$("")
	If Guess$ = Answer$
		Print "Right"
	Else
		Print "Wrong"
	EndIf
	
End Function
