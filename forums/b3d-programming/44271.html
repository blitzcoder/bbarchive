Beaker(Posted 2005) [#4]
You can do something like this:
Type thing 
	Field Vector%[100] 
	Field blah
End Type
 
th.thing = New thing 
For x = 0 To 9
	For y = 0 To 9
		th\Vector[x*10+y]= Rand(1000)
		Print x*10+y
	Next
Next
