Ricky Smith(Posted 2006) [#2]
You can use the command LoadAnimSeq("file.b3d") to add animation sequences to a model previously loaded with LoadAnimMesh(). This works in the same way as ExtractAnimSeq(). Every time you use LoadAnimSeq() a new sequence is created and the sequence number is incremented.
The file that you are loading from must have the same skeleton structure and joint names as the loaded model.
