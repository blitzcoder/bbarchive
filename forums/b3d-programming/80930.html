There are better methods but quickly ..

PrintText("Are You Sure?" , 0 , 0 )
PrintText("Y/N", 60,20 )

Function PrintText( t$ , x , y )

	For l = 1 To Len( t$ )
		Char$ = Mid$( t$, l, 1 )
		Text x + (l-1) * 10 , y , Char$
		Delay 100
	Next

End Function
