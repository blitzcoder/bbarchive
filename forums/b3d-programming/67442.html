FUI3D is really cool... ultra fast, with a lot of options... the unique same is the poor documentation and lack of examples, but the GUI rocks! it also has a GUI designer tool. This GUI has another 2D version. It's the one I'm using currently.

Another posibility is the Jeppe Nielsen GUI (www.gameart,dk), also very fast and as occurs with FUI3D, with a lot of gadgets, 3D accelerated... pretty cool but not for free.

Jeppe Nielsen has a older one, called AlphaGUI, more oriented to game menus and interfaces, not tools..

Devils Child has another one, free and fast, very prowerful, but as far as I know, without gui designer tool (maybe someone can check this)

You can find more information at the toolbox section, and as far as I can remember, there exist another post with the same title started some months ago... look for it in this forum.
