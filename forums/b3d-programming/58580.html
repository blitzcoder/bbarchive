Its nothing to do with sourcecode, Artemis.

I'm assuming that you've created a model, added bones/animation, then exported to .3DS format. This format does not support boned animation. All you will see is a set of animated bones and a static mesh.

You need to use .B3D file format to animate boned meshes. There is a tool called B3D Pipeline that exports to .B3D format from 3DSMax. Alternatively you can use Milkshape which exports directly to .B3D format.
