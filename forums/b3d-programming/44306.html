Floyd(Posted 2005) [#2]
Graphics 200, 400, 0, 2

For n = 1 To 25
	tx = n Mod 5
	If tx <= 0 Then tx = tx + 5
	Print LSet(n,4) + tx
Next

WaitKey

tx is always in the range 1 to 5, even if you allow negative level numbers.
