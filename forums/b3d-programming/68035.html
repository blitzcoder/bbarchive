Danny(Posted 2007) [#3]
The pause ''feature'' in schnok is a side effect of using WB3D_Notify() - that forces the program to 'stop' until the user hits OK or CANCEL...

WB3D_Notify "SCHNOK!","Schnok is paused, hit OK to continue",0
