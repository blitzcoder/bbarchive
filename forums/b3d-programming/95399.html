Yasha(Posted 2011) [#3]
In case you haven't read the release notes, as of version 1.99 you also have modes 6 and 7, which are the same as 2 and 3 respectively but pause the application if it loses focus. For most games, I would say these are a better option than 2 and 3.

As for the question... yeah, that's an unfortunate consequence of what mode 3/7 does. Unless you actually need to be able to drag it back and forth willy-nilly, using mode 2/6 is much better (nothing - except the expenditure of effort - stops you from resizing the window in mode 2/6; you just can't do it by dragging the edges of the box).
