Byteemoz(Posted 2006) [#6]
Pretty easy:
	tdir = ReadDir("modules")
	Repeat
		f$ = NextFile(tdir)
		If f$ = "" Then Exit
		If f$ <> "." And f$ <> ".." Then
			If FileType("modules\"+f$) = 2 Then
				Game_Mod$ = "modules\"+f$
				DebugLog Game_Mod$
			EndIf
		EndIf
	Forever
