Although a 433 mhz is well obsolete now, I can sympathise.

I'm running a P3 667 mhz pc with a 32mb Matrox G400 card on W95. Firstly, I simply can't run today's games on this rig.

I'm also finding 3d blitz games that have a nice level of detail to them (such as Trium), just run too slow (e.g. 10-14fps). I'm having great trouble getting my own game to run (Tank Universal) at 30fps at 640x480 16bit col, with all the graphical fx turned off. I'm putting this down to the fact that any game written in basic is going to be slower than say, C+ developed programmes.

Additionally though, many Blitz progs in 32bit col depth seem to overwhelm my aging 32mb Matrox vid card. I suspect that these blitz progs are quite liberal with their texture loading (perhaps more so than commercial games would be), so that the min graphics card mem is closer to 64megs or even 128megs to be able to comfortably run the more advanced blitz games being developed.

The point I want to make is that although Blitz can produce commercial quality game graphics, the hardware requirements for these are probably higher than c+ games.
