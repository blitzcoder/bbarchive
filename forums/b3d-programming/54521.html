Beaker(Posted 2005) [#2]
This is the only way:
Global debugfile

Debug "testing"
Debug "testing2"

Function Debug(txt$)
	DebugLog txt
	If Not debugfile
		debugfile = WriteFile("mydebug.txt")
		If Not debugfile Then RuntimeError "Can't record debug"
	EndIf
	WriteLine debugfile,txt
End Function
