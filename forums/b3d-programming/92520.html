Not natively, but FastText ( http://fastlibs.com/libraries.php ) is able to render Unicode from Blitz3D. As it says on that page though, not only does Blitz itself not support Unicode, but the default editor doesn't either, so you'd need to get IDEal (or something else) as your code editor.
