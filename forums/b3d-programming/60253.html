Bouncer(Posted 2006) [#2]
Just play with the teapot demo that comes with the filter source (you can adjust the values with a/z and s/x) and find some values that produce the results you want. Keep blurpasses to default and if you increase passes, lower the values accordingly. Using too big values makes everything go white... You just have to find right values for your environment...
