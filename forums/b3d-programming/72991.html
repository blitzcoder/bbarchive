The shortest way off the screen will always be a horizontal or vertical line.

So, given Width and Height equal to the width and height of the screen...

Lowest = Mx
ToX = 0
ToY = My

If My < Lowest
   Lowest = My
   ToX = Mx
   ToY = 0
EndIf

If Width-Mx < Lowest
   Lowest = Width-Mx
   ToX = Width
   ToY = My
EndIf

If Height-My < Lowest
   ToX = Mx
   ToY = Height
EndIf
