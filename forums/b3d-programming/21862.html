I'm starting to develop a new 3D art vocabulary pet peeve. The whole "vertex-vertice" thing has bothered me for a while (ie. there is no such word as "vertice") but increasingly I am seeing people confuse 3ds with 3D Studio Max. "3ds" the file format is an acronym for "3D Studio," a separate software from "3D Studio Max." Specifically 3D Studio was a DOS app; when Autodesk migrated from developing DOS software to Windows they introduced 3D Studio Max.

In this case it wasn't a big deal (from the subject I thought you were referring to 3ds file format but looking in the thread it was obvious you were referring to 3D Studio Max) but the big problem is how many people think 3ds is the 3D Studio Max file format.
