Email me if you are interested in using the Cartography Shop 5 core for your interface. I am at a stage now where it could be used to make a map or mesh editor, anything that requires multiple viewports. The program uses really fast drawing and has a really light-feeling touch. It's much, much faster than any previous version. The drawing is pixel-perfect and locks exactly to the grid. Grids can be resized, viewports zoom in and out, any number of viewports or layout is allowed, and any viewport can switch back and forth between perspective or any of the 3 orthographics views. Controls are easy to add because I use WorldToViewport() and ViewportToWorld() commands.

I won't charge very much, and will have a couple of stipulations. The reason I am posting this is because I would like to see a low-cost model editor with a fast interface like this, but I myself want to stick to working on a map editor. This is a method that has been worked on and improved over a couple of years, and has been tested by, well, hundreds of people.
