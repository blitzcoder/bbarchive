Floyd(Posted 2004) [#4]
If you have added or changed any userlib decls files try temporarily disabling them.
You can temporarily rename them to anything except *.decls.
If this fixes the problem then something is wrong with a decls file.

Blitz is very sensitive to bad syntax in these files.
I once left off a single " character. Blitz would seem to launch.
But then it would just disappear. Processes were inaccessible, although visible in Task Manager.
