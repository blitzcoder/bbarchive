Ok, i think this is what you want?

Press 1 to enter a position. Then press enter. Repeat as many times as you want.

Graphics 800,600
SetBuffer BackBuffer()

no=-1

While Not KeyHit(1)
Cls

For i=0 To 1080 Step 10
Plot Cos(i)*i/10+160,Sin(i)*i/10+120 
Next 


If KeyHit(2) Then
	FlushKeys()
	no=Input("Enter position:")
	FlushKeys()
End If

If no>-1 Then
	Color Rnd(0,255),Rnd(0,255),Rnd(0,255)
	Rect Cos(no)*no/10+160,Sin(no)*no/10+120,2,2
	Color 255,255,255
End If

If no>-1 Then
	Text 200,0," x = "+(Cos(no)*no/10+160)+"    y = "+(Sin(no)*no/10+120)
End If
Flip

Wend
