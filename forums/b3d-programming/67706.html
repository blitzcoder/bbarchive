fredborg(Posted 2007) [#8]
image = LoadImage("filaxrocks.png")
w# = ImageWidth(image)
h# = ImageHeight(image)
FreeImage image
texture = LoadTexture("filaxrocks.png")
cube = CreateCube()
EntityTexture cube,texture
ScaleEntity cube,w/w,h/w,1.0
Doesn't that work?
