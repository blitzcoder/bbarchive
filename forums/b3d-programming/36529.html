slenkar(Posted 2004) [#3]
the object and handle stuff is for after you understand what you are doing,

you need:
Type object
Field handle
End type

o.object=new object
o\handle=createcube()

for o.object = each object
if o\handle=pickedentity()
moveentity o\handle,1,0,0
endif
next
