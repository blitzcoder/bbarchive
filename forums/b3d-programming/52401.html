KuRiX(Posted 2005) [#5]
People doesn't want to post you an example xDD:

Camera1 = CreateCamera()
Camera2 = CreateCamera()
Cube1 = CreateCube()
Cube2 = CreateCube()

; CAMERA 1 RENDER ALL
HideEntity Camera2
RenderWorld

;CAMERA 2 RENDER ONLY CUBE2
ShowEntity Camera2
HideEntity Camera1
HideEntity Cube1
RenderWorld

Good Luck!
