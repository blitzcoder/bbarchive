Spinduluz(Posted 2004) [#4]
I'm [EDIT*] ->NOT<- exactly sure what you want to do. But if you want to manipulate blitz bank data in C/C++ then this should do it.

C++ part

void MyFunc(int *bank)
{
*bank = 1234567;
}

Blitz

Bank = CreateBank(4)
MyFunc(Bank)
Print PeekInt(Bank,0)
FreeBank(Bank)
