darklordz(Posted 2003) [#2]
Use 2 "Main" Loops

.Menu
While MenuAction = 0
; Menu StuffHere
; If Hit ExitButton : End etc etc etc
Wend

While Not Keyhit(1)
; Real GameSttuff Here
Wend
Goto Menu


i find it easier to build my game first and the Menu/ Interface Second...
