jfk EO-11110(Posted 2007) [#4]
You need to search for the window. Probably you need to use Api_GetTitle(), or so. If you once got the HWND, it's easy to retrieve the full window size using Api_GetSystemMetrics.

If you're not familar with Windows Api calls, and how to add them to the blitz userlibs, then you should get a Windows Api documentation. Also try to get a list of windows constants definitions, example: SM_CYBORDER = 6. They are often required as parameters.
