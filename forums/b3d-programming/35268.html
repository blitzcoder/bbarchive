eBusiness(Posted 2004) [#4]
A Graphics call will do it:
Graphics3D 800,600
texture=CreateTexture(2048,2048)
Repeat
	If KeyHit(28) Then
		Graphics 200,200,0,2
		ExecFile("testi.exe")
		End
	End If
	If KeyHit(1) Then End
Forever
