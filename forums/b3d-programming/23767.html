Well I would say 3D Studio because the arms appear to be seperate objects.

And finally, the fact that you can display a frame number indicates that it is not MD2 because there are no commands to return a frame number for MD2.

Therefore my guess is that it is a B3D file, as B3D can support vertex animation through bones, and it can support selerate entities, and the dots of light on the model appear to indicate the locations of bones or the joints of bones.
