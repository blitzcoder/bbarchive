You will need to check if the mouse is within the camera´s vieport coordinates:
something like:
x=MouseX()
y=MouseY()
If x>=camerax
 If y>=cameray
  If x<=camerax+camerawidth
   If y<=cameray+cameraheight
    ;Do stuff...
   Endif
  Endif
 Endif
Endif
