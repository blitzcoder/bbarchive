Yasha(Posted 2011) [#2]
Draw3D ( http://www.blitzbasic.com/Community/posts.php?topic=78304 ) and Draw3D 2 ( http://www.blitzbasic.com/Community/posts.php?topic=92270 ) both include 2D physics engines. There's also NooNooPhysics ( http://www.blitzbasic.com/Community/posts.php?topic=84769 ), which isn't attached to a rendering engine.

I haven't used any of them myself, so I can't provide any suggestions about their use, but you can find full documentation for all three (in German) here: http://www.blitzforum.de/help/

I dimly recall mention of Box2D wrappers too... Any DLL-based lib like that or Chipmunk should be easy enough to either use directly, or wrap up for use with Blitz.
