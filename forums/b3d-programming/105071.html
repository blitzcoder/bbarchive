Rroff(Posted 2015) [#4]
If you are using TCP streams then at that level any potential performance will likely be swallowed by the TCP protocols that favour compatibility over performance - if performance is a consideration you'd be better off looking at a solution using UDP.

IIRC writeline has some basic formatting included that writebyte doesn't but the performance difference would depend a lot on the level and type of use.
