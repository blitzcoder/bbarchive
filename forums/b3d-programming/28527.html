jhocking(Posted 2003) [#5]
B3d will work but you'll probably lose some detail in the animation. B3d file format supports skeletal animation (eg. Biped skeleton) but does not support blended vertex weights (eg. smooth mesh deformation in Physique.) For a variety of reasons however, like file size, runtime control over animation, and lighting accuracy, I would recommend b3d over md2.
