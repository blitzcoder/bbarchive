At GDC 2013, Nintendo said it will provide an HTML 5 SDK to work with WII U. From what I read the SDK allowed to work with all the features of the device.

I tested some of my applications and everything seems to work .. some app's don't run so smooth .. because the WII U hardware don't have the same power of a PC computer to render a game in JS .. I think we can consider a further platform for creating games.
