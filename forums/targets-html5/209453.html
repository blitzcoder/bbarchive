This post was apparently a thing. Not sure if full HTML5 support is provided by the Wii U. But if it is, your games should run just fine. Nintendo's web APIs are probably a proprietary/closed-off thing that you'll have to sign an agreement for, if I had to guess. So, you'd need to use external JavaScript through Monkey, and compile with the standard HTML5 target. Either that, or you could modify the main target to add controller support and the like. I'm honestly not sure what Nintendo is providing, but it's probably not something that could be supported from a public/default target (NDAs and what-not). I also don't own a Wii U, so take my advice with a grain of salt.
