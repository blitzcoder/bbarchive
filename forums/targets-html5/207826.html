If you are already running / developing a web server on your local host you can get Ted to open the correct url by adding a simple bat file to %monkey/bin for instance startlocalhost.bat:

 
start http://localhost


Then change the mserver.exe reference in config.winnt.txt to startlocalhost.bat so running your monkey app causes Ted to open the specified address on the nearest browser.
