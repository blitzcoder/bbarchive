It's actual for mobile browsers
Add to monkegame.html (production or template) this css rules for canvas element
        -webkit-touch-callout: none;
        -webkit-user-select: none;
        -khtml-user-select: none;
        -moz-user-select: none;
        -ms-user-select: none;
        user-select: none;
        outline: none;
        -webkit-tap-highlight-color: rgba(255, 255, 255, 0); /* mobile webkit */


so get this rule
canvas{
	border: 0;
	padding: 0;
    margin: auto;
    display: block;
	outline: none;
	  
    -webkit-touch-callout: none;
    -webkit-user-select: none;
    -khtml-user-select: none;
    -moz-user-select: none;
	-ms-user-select: none;
	user-select: none;
   -webkit-tap-highlight-color: rgba(255, 255, 255, 0); /* mobile webkit */
  
}
