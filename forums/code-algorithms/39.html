k.i.s.s. toggle by Russell2001
Probably the absolute easiest way to toggle between two numbers. 
;Toggle between 0 and 1:
a = 0
Repeat
  Print a
  a = 1 - a ; <- The clever part!
Forever

; Learned this from waaaaaaaay back in the
; Commodore 64 days...
