Bit Access by itsdanreed2008
This code has many uses, basically, it's 3 functions which allow you to read/write bits in a byte. This code could be expanded to write bits in integers, ect.
No credit necessary, as this is basic programming. 
Const BIT_0			 = 1
Const BIT_1			 = 2
Const BIT_2			 = 4
Const BIT_3			 = 8
Const BIT_4			 = 16
Const BIT_5			 = 32
Const BIT_6			 = 64
Const BIT_7			 = 128

Function SetBit(B:Byte Var, Bit:Int) 
	B :| Bit
End Function

Function ClearBit(B:Byte Var, Bit:Int) 
	B :- Bit
End Function

Function GetBit:Int(B:Byte, Bit:Int)
	Return B & Bit > 0
End Function
