You should probably post this in the PSS forum, but i think your issue is that you need to set the Playstation SDK Target Path to where you have it installed. If you look in the /bin/config.winnt file on your monkey installation, you will see where each of the Target's paths are and then just set it accordingly :

PSS_PATH="<path to PSS SDK>"
