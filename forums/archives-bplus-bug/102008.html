Yasha(Posted 2014) [#2]
Yes, setting it up is exactly the same as it is for B3D, only substitute your BlitzPlus folder paths. IDEal is designed to allow for multiple Blitz installations simultaneously (even multiple versions of B3D/B+), and it will remember settings and things like which userlibs are activated on which compiler.

All you have to remember is which one you set as the default (but you can change compiler with a drop-down on the main screen).

_PJ_(Posted 2015) [#4]
Open IDEal
Open SETTINGS Pull-Down Menu
Select "Preferences"
In first tab of Preferences (Compilers) CLick on AddNew button
Enter name (for example "Blitz Plus")
Highlight "Blitz Plus" in compilers window
Under "Blitz Path" link to blitzcc.exe location (i.e. "C:\Program Files (x86)\Blitz+")
Under "Help Path" link to Help directory location (i.e. "C:\Program Files (x86)\Blitz+\help")
Click "Refresh Definitions"

To change compilers in use, use the listbox found within the "Compiler Toolbar"
