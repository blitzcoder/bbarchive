TEXT input Lib by skn32002
USefull if you are going to make a text input box, if you called this everyloop, it will retun the apropriate character or COMMAND of what you pressed.

Takes into account buttons second functions,
Returns thing like TAB,DELETE,BACKSPACE etc 
Repeat
a$=get_keyinput$()
	If a$="ENTER" Then 
		Print
	Else If a$<>"" Then
		Write a$
	End If
Until KeyDown(1)=True
