Scancode constants by WendellM2003
I'm posting this at the request of Outcast who created it. He writes: "Maybe there was something already like this, or it was just so trivial that no one cared, but I got tired of seeing magic numbers and having to look up magic numbers whenever I wanted to read a key with blitz. So, I took the scan codes and created constants for each of them. All you need to do is copy the below text into a scancodes.bb file and you will be set to use the scan codes without ever having to memorize magic numbers again!"
