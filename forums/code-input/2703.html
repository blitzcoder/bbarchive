Well, Just realized there was already Scancodes constants after I did all the job. Just posting it here because I also made the Mouse buttons and ASCII codes constants too. Boring to do, but will definaly help to make programs and test programs faster. Just save as KEYS_Include.bb and then put it on the "temp" folder or in each project folder. Then Include it, and save time (:
Just ignore the side comments. If you are gonna use some of theses keys, you will probably know what they mean.
