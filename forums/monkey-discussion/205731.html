MServer constantly exiting - fixed!

Monkey Archive Forums/Monkey Discussion/MServer constantly exiting - fixed!

rIKmAN(Posted 2013) [#1]
This has been doing my head in for the last 2-3 weeks, although it only happened once my project reached a certain size asset wise.

MServer would load up, start loading the assets and listing them in the console window, and then just vanish - leaving me with a html5 error as obviously the server had stopped "serving" halfway through.

Maybe 1 in every 15-20 times it would load fine, and then I would be able to just 'build' and hit 'refresh' in the browser to test again without having to re-run MServer.

I did this for a few days, but as I added more stuff it became ridiculous, I counted 50 times the other night and every one failed the same as above.

I tried various versions of Monkey - it happened in all of them - but small programs were fine, and I was pulling my hair out.

Then when testing v73b my anti-virus popped up saying mserver was dodgy...which of course I knew was false (unless Mark is upto something lol) - and then my lightbulb switched on!

I disabled my AV and what do you know - perfect every time!
I have since added my Monkey/Project directories to the exceptions/whitelist in my AV and no more problems.

So, this is for future reference if anyone has this problem and decides to do a search.
If I can save them the hair pulling it caused me then it was worth the 2mins it took to post.
