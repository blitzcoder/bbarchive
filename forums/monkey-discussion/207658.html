James I'v been meaning to ask you what you do with your testcard, like what information can you gleem from it or what do you actually do with it.


ElectricBoogaloo(Posted 2013) [#5]
1. Credit screen! Does credits. No need to faff about each and every game, when there's a suitably nice little credits screen available for all my games!

2. Learning experience. Learning to code something that suitably rescales to multiple orientations, but with the same end result. Even in portrait, it still looks like a test card, still has the same info in the same areas, and even has the same text in the same circle in the center.
Was VERY good practice!!



3. The top left is the current resolution, and Update rate. The top right is virtual resolution, and Render rate. (Albeit slightly low, 'cos I can never quite get that exact 60fps figure using my botched maths!)
These numbers aren't always useful, but if someone ever finds a weird issue with elements popping out of the screen or something, a simple screen grab of the testcard will usually help me track down any resolution specific problems with a game.
