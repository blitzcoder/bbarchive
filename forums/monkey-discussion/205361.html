Hi all.

I am working on an IDE for Monkey name JumpIDE. My idea is to create a very practical and clean IDE to Monkey. I try out several IDE out there, all of them is great piece of software but I kind of have a different taste. Thing like.. bigger working area, simple custom folding, Jump to several place easily and also come back more easier, easy command references and a lot more thing base on practical stuff.

Its still in beta but please download it if you interested to try.



JumpIDE consists of:
-Standard editor features
-Syntax highlighting
-Custom folding
-Procedure browser
-Custom browser (jumper list)
-Interfacing with Monkey compiler
-Search/replace
-Auto Cursor history
-Easy Back/Forward of cursor history
-Bookmark
-Inline auto-complete for Monkey functions and methods reference
-Customize inline auto-complete
-Zoom feature for faster scroll
-Syntax themes

You can download at here: http://nvgamepad.com
