dawlane(Posted 2015) [#3]
I reckon Mark threw it in as a quick way to write and compile the tools for Monkey. TED is functional to edit and compile a BlitzMax application, but there is no real control on configuration and no real text highlighter. If you want to try it, then set the BlitzMax path in the options dialog and open a BlitzMax sample directory as a Project and double click a BlitzMax source file. You will find that the target will change to "BlitzMax App".
