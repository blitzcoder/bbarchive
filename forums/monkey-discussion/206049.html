Gerry Quinn(Posted 2013) [#7]
Best pick a fairly recent version and stick with that until you need a new feature, or just want to catch up a bit. Monkey is pretty consistent and it is very rare for an existing feature to be changed or broken by an update. The exception might be if you need minib3d as that is a fast-changing module that can need a certain version - most modules will work fine on any version that was current when they were developed.
