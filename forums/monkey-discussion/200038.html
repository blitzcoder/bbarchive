Monkey has escaped!

The new massively multi platform language Monkey is now available.

An evolution of the popular Blitz range of compilers, Monkey can generate code for the following platforms:

* HTML5
* Native OpenGL/OpenAL (Windows + Mac)
* Android
* Flash
* iOS
* XNA

Grab the fully functional html5 demo from the store now and give it a whirl!
