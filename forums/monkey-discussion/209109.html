dawlane(Posted 2014) [#2]
If you have purchased the pro version, then you should have received the registration code. Click your user name in the upper right of the web site to register the product. You will also find you can download all of the versions of Monkey Pro via the Products updates in that list.
All versions of Ted (Window/Linux/OSX) are included in both the free and the Pro version. To run Ted and build Applications for OSX and iOS you have to install Monkey onto a Intel Apple Mac. You will also need an iOS device (a long with being in the iOS developer program) for actual testing as the simulator that comes with Xcode is inaccurate.
