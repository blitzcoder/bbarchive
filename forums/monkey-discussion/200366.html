matt(Posted 2011) [#3]
IMHO size alone shouldn't be a factor here, but rather suitability of the file format for the images being presented. Surely the goal is to reproduce the screenshot accurately?

Any "space savings" gained by using JPEG is artificial as the image is not an identical representation of the original. PNG is the best file format widely supported for saving low file size, and more importantly lossless, versions of images such as screenshots.

More info:
http://en.wikipedia.org/wiki/Portable_Network_Graphics#Comparison_to_JPEG
http://info.eps.surrey.ac.uk/FAQ/standards.html
