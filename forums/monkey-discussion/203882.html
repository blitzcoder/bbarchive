I'm sitting here and was about to upload some early code of Terminal 2 my main project at the moment, as I wanted to get some alpha feedback on the ui and basics that as in the game at the moment.

but then I stopped because I realized that it would be public..

I think it would be cool if we could edit an app and set it as Public,Private,Password or closed.

Public of course would just be what we have now.
Private would require a direct link to the page, and the app would not show up in any charts or lists.
Password would require a password, it would show up in charts etc.

With a few basic options like this, people could work on their games in private and then give people here in the community direct links, or a password so they can beta / alpha test stuff without making it all public.

I can and will be doing the above with my game on my own host, but not everyone has their own host so thought I would suggest this to help those people out.
