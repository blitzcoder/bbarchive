Thanks for helpful comments and ideas.

I find a method to bypass "cannot find file ?3ds" message.
(Sorry for mistyping ?3ds -> ?.3ds : As looking carefully, there is no period mark.)

It seems that the installer is trying to find files of pattern ?3ds. But the created files have names like a.3ds b.3ds etc. ; pattern ?.3ds .
So I simply add a file named a3ds that matchs searching condition after the incomplete installation. And try 2nd installaton. Happily! it proceed bypassing that error, and installation is ended. Icon is created,viewing tutorials becomes possible.
