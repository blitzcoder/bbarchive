SebHoll(Posted 2008) [#4]
Yeah, you can typically view the SVN Commit log for a repository, although to be honest, most of Mark's commits don't have any comments in them, simply labelling them as "lazy commit" :-P. You can still see what files have changed though...

Regarding changes since the last v1.30, there haven't been many - a few doc tweaks, a few fixes to Pub.FreeProcess and of course a few fixes for the maxide.bmx source.

Over on the MaxGUI SVN repository however, I've started to specify detailed commit information to each revision so that you can see at a glance what has changed, and in what module.

If you are using TortoiseSVN, you can see the log by right-clicking (in this case the maxgui.mod folder you use to checkout the MaxGUI SVN to), and choosing "Show Log" from the TortoiseSVN sub-menu. You should see something similar to:



I try to keep the MaxGUI repo as up-to-date as possible, so there are usually commits quite frequently.
