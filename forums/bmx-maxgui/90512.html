Logic Gui Professional Version 5.2 build 432 has been released!

Those who don't know Logic Gui yet, it's an easy to use, feature rich visual Gui Designer and source code creator for MaxGui.


Major improvements:
Version 5.2 brings the ability to create independent types from panels and it's gadgets on top. This allows you to create your own set of closed types, ready to use in any new project or reuse of one type again and again in your application.
This extends the feature from Version 5.0 which introduced type creation for your form and adds much more flexibility inside the form itself.

Another great improvement was made to the SmartButton Editor and Logic Gui which allows much easier exchange of SmartButtons outside of Logic Gui forms and also better project handling.

New session backups helps you to get back with whatever you started and gives more safety. While the undo is unlimited during a session you are now able to go back even further. You can go back, up to three Logic Gui sessions.

And more...

A lot of work has gone also into a completely new LogicZone website!

The new site is based on Joomla and offers more flexibility and looks hopefully more professional than before. Also the forum is now more integrated. If you miss something, please let me know.

Registered user can find their login now under Support/Customer Login

Key features and key benefits are available from the main page.
Interested users can download demo versions or tutorials in the download section.
The user manual is updated for version 5.2!
