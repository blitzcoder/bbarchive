rs22(Posted 2008) [#5]
I believe events are the best way to go when dealing with input in MaxGUI:
WaitEvent()

Select EventID()

  Case EVENT_KEYUP
		
    Select EventData()
			
      Case KEY_LSHIFT
				
        ' Do whatever

    EndSelect

  Case EVENT_MOUSEMOVE

    MX = EventX()
    MY = EventY()

EndSelect()
