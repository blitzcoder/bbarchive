jsp(Posted 2007) [#1]
SmartButtons, a new sort of image buttons for MaxGui are done.
Based on panels it is a flexible system using hotspot regions and actions which run on defined triggers.

A demo can be downloaded here:
http://jsp.logiczone.de/downloads/SmartButtonDemo.zip

For those who are interested in how it works in detail, you can check out the docs here:
http://jsp.logiczone.de/downloads/SmartButton1.0.pdf

While those SmartButtons initially are written to support Logic Gui in the next release, i can make it a standalone application if there is enough interest for the module and editor.
