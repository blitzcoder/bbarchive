Amon(Posted 2007) [#4]
I think i found them. You need a dedicated app to view the icons in shell32.dll. Beleive it or not the icons are sorted in directories within the dll.

The app you need can be downloaded from below.

http://www.softpedia.com/progDownload/ResBuilder-Download-10850.html#download_locations

Install it and open shell32.dll with it. Browse to the bitmap folder in the dll using the pane on the left.

All the icons are in there.

[edit] There are more icons in the icon folder. I think every icon used by standard xp apps and windows are in this dll.

Hope that helps.
