Dreamora(Posted 2006) [#4]
MouseDown isn't the same as mousehit.
mouse hit counts the number of hits since the last call of it (it does not say if it has been pressed as some assume until they run into the well known "why does it return 0 on the second call" problem) so what you would need to do is raise a variable each time a mousedown is registred ... but thats a little pointless as this is exactly what BM itself already does with the mouse and keyboard buffer arrays for hit and down ^^
