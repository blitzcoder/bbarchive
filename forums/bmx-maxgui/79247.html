Is the recommended size hard coded?

For Windows, it isn't hard coded as such. It crops the pixmap to whatever..

GetSystemMetrics(SM_CXMENUCHECK)
returns, which is supposed to be the correct size of the check/menu icon. This varies from Windows XP to Vista, and from theme to theme, between (12-16px iirc) and I think I remember OS X having quite a small size constraint, so a size of 12 as it produced consistent results and no clipping.
