Beaker(Posted 2005) [#3]
For normal arrays use:
Dim ship(0)
..which makes it very small.

For 'Blitz arrays' use something like this:
Function blah()
	Local ship[19]
	;..do stuff to ships
End Function ; array is killed here

OR

Type arraycontainer
	Field ship[19]
End Type

ac.arraycontainer = new arraycontainer
;...
Delete ac
