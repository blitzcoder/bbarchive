Gauge(Posted 2005) [#5]
Heres some basic code to get you started. What kind of features do you want to add btw? This works in blitz3d so no special 2d functions, just draws the text on the screen.
Print "TEXT EDITOR STARTING"
Global x
Global Y
Global key
x=10
y=10

While Not KeyHit(1)
key=0
key=GetKey()
If key<>0 Then
Text x,y,Chr$(key)
x=x+10
checkline
EndIf

Wend

End

Function checkline()
	If x>390 Then
	y=y+10
	x=0
	EndIf
End Function
