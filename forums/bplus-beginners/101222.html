Yasha(Posted 2013) [#2]
There is no official support for threads.

If you're feeling adventurous, you can use FastPointer (free to download from here: http://www.fastlibs.com/index.php ), but it's not for the raw beginner, and is missing some important features (i.e. no synchronization).

BlitzMax does have some official support for multithreading, if you end up deciding to use that. That's a different product though.
