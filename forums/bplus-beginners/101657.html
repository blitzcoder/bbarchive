GfK(Posted 2013) [#3]
Graphics 640, 480, 32, 2

The 2 forces windowed mode, IIRC. Replace this with 1 for fullscreen.

The 32 is the color depth, which has no effect in windowed mode (it uses whatever your desktop is set at) so it doesn't matter what you put.
