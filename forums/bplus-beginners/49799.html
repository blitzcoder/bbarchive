Blitz will auto-convert between variable types for you, most of the time...
a$ = "10"
b = a$
c# = "42.5678"

print a
print b
print c#
print int("30") + int(a$); Notice I force the conversion to integer here
                         ; to ensure an addition rather than a string concatenation!
