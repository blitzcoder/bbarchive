rdodson41(Posted 2007) [#2]
Function getstring$()
    Return "This is a string"
EndFunction

Function names are followed by the type they return:
% or blank for integers
# for floats
$ for strings
.TypeName for user defined types

getint() or getint%()
getfloat#()
getstring$()
getalien.talien()
