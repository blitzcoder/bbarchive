Floyd(Posted 2013) [#4]
Start looking at the first position after the substring you just found.

s$ = "HelloGoodbyeHelloGoodbyeHelloGoodbyeHelloGoodbye"
t$ = "Goodbye"

p = 1

Repeat
	p = Instr( s, t, p )
	If p = 0 Then Exit
	Print "Found " + t + " at position " + p
	p = p + Len( t )
Forever

WaitKey
