WolRon(Posted 2006) [#2]
Files made in ANY program can be used in BlitzPlus (or any blitz product for that matter).

What, of course you do with it (when you say "USE"), is up to you...

If you were asking if a maplet model could be displayed in BlitzPlus, then No, that can't be done (at least not natively). There are no 3D commands in BlitzPlus.


Pineapple(Posted 2006) [#3]
WolRon's right...

Say you export you maplet model in .X format, to load it into Blitz+, you'd have to wrap a large amount of Direct3D in a DLL, then import the DLL as a userlib!

Not impossible, as I once used a DirectX SDK example (The one with the rotating mesh) as a spinning logo effect for a B+ splash screen!

I'm working through DirectX myself (SDK 8.1), just for 2D games, which ironically, uses Direct3D... And there's a lot to get to grips with!!!

As I said, not impossible, but a lot of hard work!
