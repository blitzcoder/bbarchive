GfK(Posted 2014) [#4]
Ditch Replace() - it's not going to do it for you.

Here's what I use (Blitzmax code, mind. Shouldn't be too hard to get it going in BlitzPlus).
Const ENC_KEY:Int = %01101010101011111101000101011101

Local s:String = "Hello World"
s = encryptString(s, ENC_KEY) 'encrypt it
Print s
s = encryptString(s, ENC_KEY) 'put it back
Print s

Function encryptString:String(txt:String, key:Int)
	If txt.Length > 0
		Local out:String
		For Local n:Int = 1 To txt.Length
			out:+Chr(Asc(Mid(txt, n, 1)) ~ key)
		Next
		Return out
	EndIf
End Function
