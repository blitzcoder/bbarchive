well... the REPEAT...UNTIL is the games main loop. The first statement inside should always be a CLS, which clears the screen. The last statement in this block is always the FLIP command, which shows what was painted before. The KeyHit(1) quits the game when pressing the ESC key.

The interesting part is the FOR EACH loop. This is a possibility of stepping through all the elements of an type list. Each time you create a new element of the type BALL it will be added to the element list of this type. With FOR EACH you can call them all... and do whatever you want to do with them:

examples
==========

show values to the debugger console:
For locBalls.ball =Each ball
	Print locBalls\Color + "   " + locBalls\X + "     " + locBalls\Y
Next
WaitKey()



add something to the values:
For locBalls.ball =Each ball
	locBalls\X = locBalls\X + locBalls\mx
Next


find certain elements:
For locBalls.ball =Each ball
	If locBalls\Color = RED
		locBalls\Y = locBalls\Y + locBalls\my
	Endif
Next
