Tommo(Posted 2009) [#3]
You need to import font modules too, like BRL.Freetypefont.
Using Framework will only import BRL.Blitz. If you don't specify a Framework, all modules form BRL and PUB will get imported.

Some modules are setup to load with "factories". This allows a more pluggable framework, but of course you need to remember to import the specific modules from which you require that functionality.
One other example is the need to import BRL.RamStream if you want to use IncBin.
