klepto2(Posted 2007) [#4]
The actual module version of minib3d 0.4 (similar to simons version) is available in the thread 'minib3d 0.40 module Version' : http://www.blitzbasic.com/Community/posts.php?topic=66746

this is prebuild for win32, but you may have to run docmods from the console:

YourBMaxPath/bin/docmods sidesign.minib3d

and to build the module:

YourBMaxPath/bin/bmk makemods -a sidesign.minib3d
