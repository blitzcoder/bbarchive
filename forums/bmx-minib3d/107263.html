With this module, OpenB3D commands can be used in Python. With small changes it should work under Windows, too.

In my opinion, one great feature of Blitz3D was the immediateness: one could type few lines, press a key, and see the 3d instantly, and that made it a great learning tool. BlitzMax+minib3d/OpenB3D was not as immediate, it is good for people who started with Blitz3D and want something more complex, but not as good for people who starts from scratch and want to learn 3d programming. I wonder if a python module (when commands can be executed immediately) could encourage some beginner to try.
