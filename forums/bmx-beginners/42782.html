or just use something like this:
file=writeFile("myfile.dat")
If Not file RuntimeError "could not open file myfile.dat"
	writeline file,"Hello"
	writeline file,2046
	writeline file,3.14
	writeline file,"Goodbye"
	Closefile file


file=OpenFile("myfile.dat")
If Not file RuntimeError "could not open file myfile.dat"

	text$ = ReadLine(file)
	Print text$

	integer% = ReadLine(file)
	print integer%

	real# = ReadLine(file)
	print real#
	
	text$ = ReadLine(file)
	Print text$
	
	Closefile file

Bot Builder(Posted 2005) [#5]
They are necessary because they tell blitzmax what type of variables they are. % isn't actually needed because if not declared a variable is assumed to be an integer. Floats are for numbers that aren't counting numbers like 12.0564 . Strings are for text like "Hello World".

Just in case you didn't know before.
