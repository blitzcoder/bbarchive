When using SuperStrict ALL variable must have a type, even function results ;)


Yep, you need to specify the return type in the method

SuperStrict

Type TClient

	Global list:TList = New TList

	Method New() 
		list.AddLast(Self) 
	End Method

	Method clientsCount:Int() 
		Return list.Count() 
	End Method
End Type
