tonyg(Posted 2006) [#2]
Cast value back to folder...
Type tfolder
	Global tfolder_list:tlist=CreateList()
	Field x
	Function create(num:Int)
		Local a:tfolder = New tfolder
		a.x = num
		ListAddLast tfolder_list , a
	End Function
End Type
For x = 1 To 10
	tfolder.create(x)
Next
test:tfolder = tfolder(tfolder.tfolder_list.firstlink().value())
Print test.x
