grable(Posted 2015) [#15]
If your going to use an updated MinGW you allso need to copy over various library files and some binutils executables into blitzmax (the same ones that are already present).

Otherwise you will have version mismatches.. Oh, and if your using tdm-gcc 4.8.1 or above, you allso need to add -lpthread to the gcc commandline used in bmk.
(in file bmk_util.bmx in function LinkApp() around line 337)
