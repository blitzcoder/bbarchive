kaisuo:

It works nearly the same:
1 there's the . seperator instead of \
2 there can be type-functions (global)
3 there can be type-methods (attached to an instance)
type bleh
  field a:int

  function blah()
  end function

  method blow()
  end method
end type

local MyInstance:bleh=new bleh
