tonyg(Posted 2008) [#9]
Anyway, you have BlitzMax 1.18.
Go to the Accounts tab and, under Product Updates, download and install 1.28.

Next, run Syncmods from within the IDE.
This should download the latest modules including the (now) old version of MaxGUI.
To get the new version you will need to install SVN and follow the steps in the Module Tweaks forum.
You will also need to install MingW and follow the instructions for building modules in the same forum.
Finally you need to rebuild docs as well.
Good Luck
