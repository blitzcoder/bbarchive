2)One copy of PL that is refferenced in both lists
Type rect
  Field x:Int
End Type
Global list1:TList=CreateList()
Global list2:TList=CreateList()
my:rect = New rect
my.x = 15
ListAddLast list1,my
ListAddLast list2,my
For all:rect = EachIn list1
  Print all.x
Next
For all:rect = EachIn list2
  Print all.x
Next
For all:rect = EachIn list1
  all.x=55
Next
For all:rect = EachIn list1
  Print all.x
Next

  Print all.x
