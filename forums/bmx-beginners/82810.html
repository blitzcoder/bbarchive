Has anyone got a function that replaces DrawRect with an option to have the rect solid or not?


GfK(Posted 2009) [#2]
Something like:
Function DrawRect2(x:int,y:int,w:int,h:int,filled:byte = true)
  Select filled
    Case False
      DrawLine x,y,x+w,y
      DrawLine x+w,y,x+w,y+h
      DrawLine x+w,y+h,x,y+h
      DrawLine x,y+h,x,y
    Case True
      DrawRect x,y,w,h
  End Select
End Function
