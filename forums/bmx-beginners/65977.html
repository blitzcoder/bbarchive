GfK(Posted 2006) [#2]
MaxGUI is a GUI module which allows you to create forms/windows/gadgets/buttons etc. Its available to buy from Blitz Research and costs $30.

BLIde is an editor for Blitzmax code (with or without MaxGUI). Its available free from www.blide.org

If you see any Blitzmax code which states that it requires MaxGUI, then you can run the code in any editor provided that you've also bought the MaxGUI module from Blitz research.

Is the MaxGUI module mandatory? No.


Amon(Posted 2006) [#3]
MaxGUI is just a module provided by Blitz Research which enables you to make applications eg. Tilemap Editors or small software.

BLide is an IDE that you use to replace the max ide. An IDE (Intergrated Development Environment) is used to program your apps and games in.

When you start BlitzMax you are presented with the IDE and you can then start to code. Blide is the same but with a ton of features.
