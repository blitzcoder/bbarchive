Midimaster(Posted 2012) [#2]
there is a special command to use the system monospaced font on each machine:

Local Mono:TGuiFont=LookupGuiFont(GUIFONT_MONOSPACED,12)
SetGadgetFont view_ram_ascii_text_area, Mono
