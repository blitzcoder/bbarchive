GaryV(Posted 2010) [#2]
MaxIDE? Do you mean BlitzMax? If so:

1. Click on the forum tab that says "account".
2. Click on "Product Updates".
3. Right next to where it says "BlitzMax 1.41", click on where it says "Windows".
4. Download the file and remember the directory you download it to.
5. Once the file has been downloaded, go to that directory and click on the file which is named: "BlitzMax141_win32x86.exe"
6. Follow the instructions for the installer.

If you do not have anything listed under "Product Updates", you need to click on "Register Product" on the "Account" tab. Select "Blitz Max" as the product and enter the registration code you were given when you purchased the product and received your email. Then follow the previous instructions.

If you do mean MaxIDE and not BlitzMax, please ignore my post.

If by MaxIde you mean the community edition of MaxIde, then you should be able to just copy it into the BlitzMax directory.

If you are meaning BlitzMax, follow the instructions in the previous post. You should be able to start the installer and follow the instructions shown on each screen and accept the default options. It will put an icon in your start menu that you click on to run it. Installing BlitzMax is not really different from installing most programs as BlitzMax's installer conforms to the accepted "standards".
