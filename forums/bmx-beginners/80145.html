In my experience most .PNG's are slightly smaller than a zip-compressed true-color bitmap, so the end result is pretty much the same. Now add to that that a PNG can have an alpha layer which you can use for smooth antialiased edges, and there is next to no reason to pick BMP...

I use PNG's for most things like sprites,but use JPG for large background photo's which can be compressed further.
