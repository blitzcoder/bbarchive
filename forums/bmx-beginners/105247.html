Brucey(Posted 2015) [#2]
It is probably not starting because you are missing libraries.

From the Terminal, you can retrieve a list of required libraries by running the following command from your BlitzMax dir :
ldd ./MaxIDE

You will be presented with a list of .so files. If you see a message such as "not found" next to one or more, you will need to install a library.

See the Linux forum for scripts that will fetch required libraries, or a list of them that you can try to install yourself.
