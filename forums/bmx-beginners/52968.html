Brucey(Posted 2006) [#19]
AppTitle isn't a GUI command. It's part of the Blitz runtime module - BRL.Blitz.
Therefore, as long as you set it *before* you start your Graphics(), it should be shown in the window title bar.

With MaxGUI, you set the window title when you create the window...
