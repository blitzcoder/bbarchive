Perturbatio(Posted 2005) [#2]
try using this code to find the scancode for it:

Graphics 640,480,0,0

While Not KeyDown(KEY_ESCAPE)
Cls
For Local a:Int = 1 To 256
	If KeyDown(a) Then DrawText(a, 10, 10)
Next
Flip
Wend
End
