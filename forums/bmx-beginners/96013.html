No problems here. Is your BlitzMax folder happen to be in the Program Files folder? Vista and Win7 enforce no altering the Program Files folder once a program has been installed. You either need to run MaxIDE as administrator or turn off UAE.

To run as administrator, just right-click the IDE icon and choose "Run as administrator."

It is better to install BlitzMax in the root folder (i.e. C:\BlitzMax)
