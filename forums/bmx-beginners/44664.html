One easy way of finding out the framerate for any OpenGL or DirectX application under Windows is with Fraps, http://www.fraps.com

It will run in the background, and can automatically put a little FPS counter in a corner of your screen when you run OpenGL or DirectX apps, without having to add any code to your actual program. Very convenient for quick tests.
