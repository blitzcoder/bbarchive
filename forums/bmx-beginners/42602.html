EOF(Posted 2005) [#2]
For joysticks:
If Not JoyCount() Print "No joysticks found!"


For graphics see:
GraphicsModeExists()
Although that does not guarantee the display will be opened.
You'll likely get a 'Unable to open graphics screen' message and the progam will terminate.

If no sound card exist the program just terminates if you do something like:
sfx=LoadSound("test.wav")
