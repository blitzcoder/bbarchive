markcw(Posted 2014) [#3]
BlitzMax allows you to import C, C++ ObjectiveC and Assembler and declare them with extern. See the section in the docs "interfacing with C and other languages" under "advanced topics".

If you want to use another language then you can do that by compiling as a library and then loading it and getting the function addresses. For an example you can look at my library wrapper for Openb3d (angros.openb3d.zip) here.
