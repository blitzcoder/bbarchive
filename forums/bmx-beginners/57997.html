TomToad(Posted 2006) [#11]
You can press SHIFT-Down Arrow to highlight the line with the error. As for the selected item color, MaxIDE uses the Windows setting. You need to go to Display properties (right-click the desktop and choose properties), click the Appearance tab, select "Advanced." select "selected Items" in the Item combobox, then pick a color.
