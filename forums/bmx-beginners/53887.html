EOF(Posted 2005) [#3]
To add an icon using Reshack:

1) File > Open .... your Max executable
2) Action > Add new resource

Browse for and select an icon (*.ico)
The resource type should appear as ICONGROUP

3) Enter a resource name (I use 101)
4) Save the file. ResHack will save a backup too
