I'll take a stab at this one. Plus even if this isn't the problem you're having Falken it IS a problem.

If you syncmods BEFORE you update to the latest core update (or any of them I'd assume) the update wipes the doc folder but since your modules are up to date you can't sync and fix up the doc directory.

Quick fix - drag any official module to the trash and sync again.

I'm sure there's a way to call the doc generation from terminal but I can't be bothered looking it up and the above is an easier fix anyway.
