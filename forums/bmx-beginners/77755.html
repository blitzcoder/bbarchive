Hi Ryan

You could use Var to return the position...
Local x:Int, y:Int
arraythingy(x,y)
Print x+", "+y

Function arraythingy(x:Int Var,y:Int Var)
	x=2 ; y=3
EndFunction


or even an array...
Local pos:Int[] = arraythingy()
Print pos[0]+", "+pos[1]

Function arraythingy:Int[]()
	Return [2,3]
EndFunction
