BRL has moved on to Monkey 2, and stopped working on B3D.

Ploppy had been working on a 64-bit version of Hybrid, which is based on B3D. Unfortunately he recently passed away, and I do not believe anyone else is working on this.
(Complicating things slightly is that B3D is based on DirectX7, which apparently doesn't have a 64-bit version -- so any 64-bit B3D also would need to be rewritten to use DX9 or newer)
