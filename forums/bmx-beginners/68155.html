Gabriel(Posted 2007) [#2]
For counter = 0 To 99
star[counter].x = rand (1, 1024)
star[counter].y = rand (1, 768)
star[counter].draw()
Next


Should be

For counter = 0 To 99
star[counter]=New tstar
star[counter].x = rand (1, 1024)
star[counter].y = rand (1, 768)
star[counter].draw()
Next


You've initialized the array, but not each object.
