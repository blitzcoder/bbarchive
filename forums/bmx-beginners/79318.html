CS_TBL(Posted 2008) [#5]
Those html files are enough to learn file I/O however.

Anyway:
SuperStrict
Local a:TStream=WriteFile("c:\b00h.txt")
WriteByte a,65
CloseFile a

Local b:TStream=ReadFile("c:\b00h.txt")
Print ReadByte(b)
CloseFile b
