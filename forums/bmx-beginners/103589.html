Goto or
Gosub?


Neither, both goto and gosub are horribly outdated legacy basic commands, you'd want to stick it in a function and call that instead.

To change resolution, you'd have to do an EndGraphics(), and then use Graphics() to create the new resolution graphics window -- but be aware that you'll have to reload all your images because they'll be flushed out of memory when you terminate the existing graphics window.
