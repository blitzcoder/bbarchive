_Skully(Posted 2010) [#2]
For ct:int=0 to lines.length-1
   lines[ct]=""
next



plash(Posted 2010) [#3]
If you want the array to stay the same size, do
lines = New String[lines.Length]


If you want a new, empty array do
lines = New String[0]
