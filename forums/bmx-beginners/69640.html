Sinus wave? Wouldn't have thought so.

A Sine wave, on the other hand...
Graphics 640,480

X = 320
Y = 240
xAngle:Float = 0
yAngle:Float = 0

While Not KeyDown(key_escape)
  Cls
  xAngle:+2.8
  yAngle:+1.6
  If xAngle>=360 Then xAngle:-360
  If yAngle>=360 Then yAngle:-360
  DrawOval Cos(xAngle) * 100 + X, Sin(yAngle) * 50 + Y,2,2
  Flip
Wend
