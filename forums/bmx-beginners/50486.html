Shambler(Posted 2005) [#2]
something like

Local a:Int=2
Local pad:string=a

While Len(pad$)<4
    pad$="0"+pad$
Wend

Print pad$

or this...

Local a:Int=2

Print Format(a,4)

Function Format:String(n:Int,p:Int)

Local pad$=n

While Len(pad$)<p
	pad$="0"+pad$
Wend

Return pad$

End Function
