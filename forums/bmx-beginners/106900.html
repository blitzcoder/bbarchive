Henri(Posted 2016) [#3]
Hi,

a way to iterate all the values in a multidimensional array:
Strict

Local ar:Int[3, 4]
Local dim:Int[] = ar.Dimensions()

'Populate
For Local x:Int = 0 Until dim[0]
	For Local y:Int = 0 Until dim[1]	
		ar[x, y] = Rand(0, 10)
	Next
Next

'Iterate
For Local x:Int = 0 Until dim[0]
	For Local y:Int = 0 Until dim[1]	
		Print "[" + x + ", " + y + "] = " + ar[x, y]
	Next
Next


-Henri
