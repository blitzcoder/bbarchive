Beaker(Posted 2005) [#2]
This works:
Strict

Local myarr[5]

myarr = [22,32,15,100]
myarr = changeArr(myarr)
Print myarr[9]
End

Function changeArr[](anyarr[])
	anyarr = anyarr[..10]
	anyarr[9] = 105
	Return anyarr
End Function
