AdamStrange(Posted 2012) [#2]
first thing is the wav file itself. Blitzmax only supports 8/16 bit, mono/stereo files. if the file is 24 bit or above it will not work.


xlsior(Posted 2012) [#3]
also note that some "wav" files really are MP3 files, with a .wav extension. Windows Media player plays them without problems, but not everything will.


xlsior(Posted 2012) [#4]
Note: You can try a program like 'audacity' (free) to load the file, and export to wav again. At least that way you know for certain what the filetype really is.
