Yeah the '..' is a nice hidden feature of BlitzMax, I use it a lot when doing large IF statements:

If ((x = 1 And y = 2) ..
   Or ( z = 3 And q = 1)) ..
   And MouseHit(1) Then

   Print "Hello World .."

End If
