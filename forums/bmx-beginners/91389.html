therevills(Posted 2010) [#2]
1. Download the maxgui_mod_142_beta.zip file from: http://sourceforge.net/projects/maxgui/files/

2. Extract the zip to /BlitzMax/mod so it looks like: /BlitzMax/mod/maxgui.mod

3. Start BlitzMax (rebuild modules if needed - you need mingw set up re xlsior post below)...

Done ;)
