iprice(Posted 2010) [#2]
LOCAL film[4]

However, that would actually hold five numbers, not names -

film[0]=54
film[1]=32
film[2]=23
film[3]=19
film[4]=67

If you want to hold the NAME of a film in an array you must use -

film$[4]

notice the "$"

film$[0]="Star Wars"
film$[1]="E.T."

etc.
