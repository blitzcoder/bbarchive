GfK(Posted 2007) [#2]
Something like (untested):
Local scale:Float = 0
Local angle:Float = 0
Local img:tImage = LoadImage("pic.png")

Repeat
  scale:+0.01
  If scale > 1 then scale = 1
  angle:+10
  angle = angle Mod 360
  SetScale scale,scale
  SetRotation Angle
  DrawImage img,x,y
  Flip
Until scale >= 1
