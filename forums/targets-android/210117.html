Ole JR(Posted 2015) [#2]
Probably missing Android SDK 3.2 (API 13).

Fire up Android SDK Manager, and check if it's installed. If you can't find it in the list, a checkmark in 'obselete' down left should reveal it..

According to this post you might also need Android 4.4.2 (API 19).
