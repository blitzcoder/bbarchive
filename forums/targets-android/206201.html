Ironstorm(Posted 2013) [#2]
Jep.

#ANDROID_SCREEN_ORIENTATION="sensorLandscape"


"sensorLandscape" was added in API level 9 (Android 2.3).
For more possible screen orientation settings just read the API documentation.
