Skn3(Posted 2013) [#1]
So I was temporarily without an android phone to test on and I was getting sick of the incredibly bad performance of the google android emulator. I tried a few other solutions but ended up finding this:

http://www.genymotion.com/

It automates installation of various android images in virtualbox.

I had some issues booting the emulator through the genymotion GUI, but I don't know if that is due to my crappy temporary graphics card? Anyway, it adds a virtual machine to virtualbox automatically, so its possible to boot it up from virtualbox.

Genymtion connects via ADB, so if you have done it manually from virtualbox then you would have to connect ADB manually.

The performance seems nice!
