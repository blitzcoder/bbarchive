JazzieB(Posted 2003) [#3]
Another point about the scaled window mode is that the image it contains still only consists of an image of the pre-defined dimensions you put in the Graphics() command.

Basically, Graphics 640,480,0,3 will create a scaled windowed mode, but the image will always be 640x480 as far as the program is concerned. The graphics card will just scale it to whatever dimensions the user drags the window to.

I've always found it useful if I've been chatting over the net at the same time as working on my game/program! Of course, I could also do that with the non-scaled windowed mode - depends on the resolution used within the game and the size of desktop you use really. Not found any real use for it though.
