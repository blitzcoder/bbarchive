SopiSoft(Posted 2004) [#6]
You can use the CommandLine$() function in main.exe to check if a parameter was passed to it e.g.:

If CommandLine$() <> ""
  ;run the program
Else
  Print "Sorry, to start this program you need to use Launcher.exe"
  WaitKey()
  End
EndIf
