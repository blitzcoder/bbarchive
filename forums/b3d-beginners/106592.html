NRJ(Posted 2016) [#2]
You have to use the textureblend command as shown in the code below - 
 

tex0=loadtexture("planet.jpg") 
tex1=loadtexture("normal.jpg")

Planet=createsphere(32)
ScaleEntity planet,5,5,5
Positionentity planet,0,5,0

Entitytexture planet,tex0,0,0
Entitytexture planet,tex1,0,1

Textureblend tex1,5


Check out the blitz3d command reference for textureblend command for more details.
