The shininess info should carry over; could you describe exactly how you defined shininess in Ultimate Unwrap?

Transparency is not carried over. You will either need to set the textures in the b3d model to alpha transparency using B3D Tweak or setup a texture filter for loading textures with alpha transparency. The latter technique basically involves using the TextureFilter command to establish that any texture with the term "alpha" (or whatever you setup) will be loaded with alpha transparency on.
