morduun(Posted 2004) [#2]
The Milkshape exporter is severely crippled in all sorts of ways; unfortunately, one of them does not allow you to model child meshes and export them as such.

To solve this, you'll want to create the model (without bones) in Milkshape, import the .ms3d file into Ultimate Unwrap and then export as .b3d with multiple meshes.

If you're going to do any serious work with Milkshape, you'll need to use Ultimate as your export tool; the Milkshape exporter is just not ready for prime time.
