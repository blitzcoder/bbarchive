Kev(Posted 2007) [#8]
in = OpenFile("test.txt")
If in > 0 Then
	
	While Not Eof(in)
		pos$ = ReadLine(in)
		file_offset = file_offset + Len(pos$)+2
	Wend
	SeekFile in,file_offset
	WriteLine in,"new line"
	CloseFile in
EndIf


not sure if this would be the correct way but it does work, create a .txt file add some text to it.
