GfK(Posted 2012) [#6]
Could be a .FON file as well, if I remember it.
You may be right, there.

This type of file is a bitmap font, not vector (whatever the hell that means). You might have to convert it to .TTF with some free tool.
A bitmap font is constructed of pixels. A vector font is constructed of vertices and splines, so can be scaled to any size without loss of clarity. A small bitmap font will not translate very well to a TTF as there is not enough data there to begin with.
