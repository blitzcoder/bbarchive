I'm sure this question was only just asked a few threads down, but still..

Blitz3d does not have any of the GUI functions that are in BlitzPlus. Blitz3d DOES have all the functionality of the ( now dormant ) Blitz2d. It can do all your DirectX 2d graphics stuff for games, but it doesn't have the GUI stuff you might want to use for apps.

On the other hand there are various alternatives which you can use with Blitz3d to add all those GUI functions. I'm using XLnt which imo looks better than the Windows API anyway.

BlitzPlus has no built-in 3d functionality, although you can use OpenGL with Peter Scheutz's userlib ( though you need to understand OpenGL to take advantage of this. )
