Shifty Geezer(Posted 2005) [#3]
You shouldn't run an old Windows install on a new motherboard. If you switch mobo's you ought to do a clean Windows install with the right Mobo drivers. That might be causing your problems. I believe the correct order is clean install, then add mobo drivers, then video drivers, then everything else. Remove all extra cards and peripherals when you install and only add these after installing the other drivers.

I'm not sure how robust XP is and maybe this rigmarole isn't as important, but it's certainly the recommended practice for earlier Windows versions and I have known not following this process to cause problems with the system.
