Stevie G(Posted 2006) [#5]
Array of types ..

Type MyType
   field x#, y#, z#
end type

Dim MyArray.MyType( 1, 10 )

MyArray( 0, 0 ) = new MyType
MyArray( 0, 0 )\x = 10


etc....
