Try adding the whole address for the texture... ie. instead of ("my_texture.bmp") add something like ("C\Program Files\Blitz3D\my_texture.bmp")or where ever you've saved the texture.

I've managed to run programs with textured meshes using this method, but only when I havn't actually saved the program yet.

It's best to just put everything in the same directory. It'll save hassle later on.
