Neo Genesis10(Posted 2003) [#5]
Here's a quick code example showing how you could put it together (just a guideline).

point = CreateImage(1,1)

...

If MouseHit(1)
	If ImagesOverlap(point, MouseX(), MouseY(), menu_item, 400, 300)
		Do_menu_item()
	Endif
EndIf
