Kuron(Posted 2004) [#4]
B+ is a complete rewrite. It is faster (logic speed wise, and in graphics speed). Has limited, but useful GUI system. Ability to use normal DLLs. B+ lets you load images to system ram or video ram. B+ can use a built in software driver for rendering (slower than Jamagic's software mode, but perfect for apps), it can use DX1 for rendering (This and the software driver allow it to run on virtually ANY 32bit Windows system}. B+ also allows for using OpenGL for rendering, and it is blistering fast. B+ supports event-driven programming and allows you to interact with the OS in a more natural and cohesive way.
