You can use an array of types, like so .

Type MyType
  field x#, y#, z#
End Type

Dim MYarray.MyType(100)

For l = 0 to 100
  MYarray(l) = new MyType
  MYarray(l)\x = blah
  MYarray(l)\y = blah
Next
[\code]


To get the y value for the 53rd instance use:

[code]
Print MYarray( 52 )\y
