RemiD(Posted 2014) [#9]
Fragmotion is really easy to use to make a mesh, make a skeleton with joints, set the influences of some joints over some vertices, make animation frames.

About

The program fragmotion just gets d3d9_43.dll can't be found errors.


Maybe try to install DirectX 9.0c

There is also Character fx which is free but you can only rigg, skin, animate with it.
There is also PaceMaker which is free but you can only rigg, skin, animate with it.
So you could use Wings3d to make the mesh and then use CharacterFx or PaceMaker to make a skeleton with joints, set the influences of some joints over some vertices, make animation frames.
