Who was John Galt?(Posted 2007) [#3]
Blitz 3D has just about everything that Blitz plus has for games, so no point changing to that IMO.

Max can draw 2D images with realtime rotation, colouring and transparency effects but I think you can get packages to allow you to do the same with B3D.

Max is certainly the most powerful of the Blitz languages but it's really a matter of preference. Some people seem to hate it. I love it. The docs can be a bit daunting (+they're not that well written), but if you don't use all the bells and whistles, the code is hardly any different than doing 2D with B3D. The only way to find out if it is for you is to try the demo.
