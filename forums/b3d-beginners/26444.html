Rob Farley(Posted 2003) [#12]
I don't think spamming forums is a good idea, it tends to annoy people...

Maplet is fine if you want plenty of pitch detail, however, you'll trade off against polycount.

The flipside is create 1 texture with all the detail on and use 2 triangles, but you are restricted by texture size and therefore lose detail on the decals.

Sticking decals on the top of a single texture you'll get z buffer issues so I'd avoid that.

Personally I think the decals are always in the same place create a model for the pitch and do some clever UV mapping. So the bulk of the pitch will be one surface and very few polys, then add a few extra bits of detail in the model and UV map those onto different surfaces so you can have full detail with those too.

You could do this in maplet too, take a look at my tutorials about texture patterns on the floor, then create some textures that have the basic ground with the decals on then and add these into the ground.
