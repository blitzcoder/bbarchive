I don't understand fully what you wrote but if you search on the forum you will find that if you can not detect a collision or a pick on an animated mesh because only the TPose is considered.
In order to detect what is the current pose of your mesh during an animation, you have to set some collidables/pickables meshes as childs of the joints (=bones) so that each collidable/pickable mesh will turn/move with the body part.
