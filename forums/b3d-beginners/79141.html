2) Gnet is, as far as I know, a server that you can use to connect to and search for same games. If you make a multiplayer game with a Gnet base, your hosted game will be listed on the Gnet server and if your looking to join a hosted game, then your game will search the Gnet server for the same kind of game. It is internet based only. Non-Lan only.

3) The same way we use anything Blitz3d related... Do a search on these forums and in code archives for Gnet related stuff. Youll learn everything you need to know about it.
