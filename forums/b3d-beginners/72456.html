Pete Carter(Posted 2007) [#2]
blitz3d doent do lightmapping unless you use a lightmapper like YAL in the code archive. normally you would use a modelling program to lightmap your level.

The blitz3d format (.b3d) supports 2 UV sets these hold the information about how the textures are displayed on the model. the first one is used to display the texture and you can use the second to store the shadows.

its easyer if you use something like 3d world studio because it does it all for you.
