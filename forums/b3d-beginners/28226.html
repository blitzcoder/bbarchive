it depend, what you really want

this function return the distance beetween 2 coordinates
Function calcdist#(x1#,y1#,x2#,y2#)
	distx#=Abs(x1#-x2#)
	disty#=Abs(y1#-y2#)
	Return Sqr((distx#*distx#)+(disty#*disty#))
End Function
