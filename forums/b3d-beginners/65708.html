To answer your initial question: b3d to 3ds

If a b3d file is using boned animation then this cannot be converted to animated 3ds. The model may be converted as a static mesh.

The only tool that properly inmports B3D that I know of is UltimateUnwrap3D, or UU3D. It allows to im/export in many formats
