ardee(Posted 2009) [#5]
Normal workflow is to bake an Ambient Occlusion pass to image and save the UV face layout (UVs-->Scripts-->Save UV Face Layout) and load these as layers in Gimp, set the opacity and use these as guides for painting the texture. I find running an edge detect filter on a layer copy of the AO map helps immensely as well.

This approach is common to most 3D software where texturing is involved.
