skidracer(Posted 2015) [#9]
whateveryournameis: That sounds a lot like the problem when you put bad files in the userlibs folder.

I would

1. Uninstall
2. Make sure the Blitz3D folder has been in fact removed
3. Make sure you have no other versions of Blitz3D installed
4. Restart computer
5. Install using official 1.64 release and no others until you solve the issue.

To summarise remove all installations, restart, and then use ONLY the version in accounts->product updates that is labelled -

Blitz3D Version 1.64

If you continue to have problems then we have at least confirmed you are using correct / latest / best / official / full version before installing the 107 upgrade.

The only XP issue I can recall that can cause serious issue with Blitz3D is those caused by the wacom tablet driver.
