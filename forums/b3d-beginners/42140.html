Floyd(Posted 2004) [#7]
The order is Y,X,Z and subsequent turns are about the newly rotated axes.
Graphics3D 400, 300, 0, 2

e=CreatePivot()

TurnEntity   e, 0, 20, 0
TurnEntity   e, 10, 0, 0
TurnEntity   e, 0,  0, 30

Print " Angles are  "+EntityPitch(e)+" "+EntityYaw(e)+" "+EntityRoll(e)

RotateEntity e, 10, 20, 30

Print " Angles are  "+EntityPitch(e)+" "+EntityYaw(e)+" "+EntityRoll(e)

WaitKey()  :  End
