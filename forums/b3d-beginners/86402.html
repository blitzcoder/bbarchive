Warner(Posted 2009) [#2]
Data fields are stored in the executable. That is why they should be constant. Basically, you can't use variables as a Data field. You could however, use a String:
Data "test", 0, 0, 1, 0

For i = 1 To 5
   Read d$
   Select d$
   Case "test"
       val = test
   Default
       val = d$
   End Select
   Print "value:" + d$
Next
