Floyd(Posted 2008) [#2]
Some things to try:

1. Reboot. That never hurts.

2. Try again, i.e. uninstall Blitz3D, reinstall 1.64 and then update.

3. A bad userlib can cause mysterious problems. You may not have any userlibs with a fresh install, but you might try disabling them anyway. They are in a folder called \userlibs. Rename this to something else, such as \userlibs.bak. You can always change the name back later.

To save time, you need only run the latest update. You don't need all of them. The older ones are still available in case a newer version causes trouble. You can then "downdate" by running one of the older updates.
