Make sure the textures are inside the same folder as the mesh. Probably convert the X to B3D using something like Milkshape, Fragmotion, UU3d.
Make sure the textures load automaticly. Applyng them by code complicates things (identification of surfaces, or the use of LoadAnimMesh).
