xlsior(Posted 2015) [#5]
(it wasn't working with the BMP's, but It is ok with PNG format?)


It should be working with 24-bit BMP as well -- but PNG's would typically be smaller in filesize.

But note that the LoadAnimImage is expecting an image strip -- e.g. if you have three 64x64 frames that you're loading, it will expect a 192x64 pixel image strip that contails all three frames side-by-side.
