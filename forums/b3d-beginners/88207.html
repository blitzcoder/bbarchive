I'm nearly 100% sure blender has a .b3d exporter... I don't know if it's community produced, but i'm pretty sure there is one.

I'd go against creating a new model format tbh. Your just causing yourself far more work, before you even begin, and it may sour the whole experience.

As for point 2, i've used a few free apps.

Wings 3D
Milkshape (small fee for this) - supports .md2 and .b3d
FragMotion is free i believe and supports .b3d (and .md2 if memory serves me)
Amapi version 4 is free. UI is a bit different, but i really like it. No texturing export option though...
