Stevie G(Posted 2006) [#2]
Like this?

Graphics3D 320,240,0,2
SetBuffer BackBuffer() 

camera=CreateCamera() 
CameraClsColor camera,0,0,127
PositionEntity camera,0,0,-5 
light=CreateLight() 

cube  = CreateCube()

Dir = 1

While Not KeyDown(1) 

z = z + Dir
if abs( z ) = 20 Dir = - Dir

rotateEntity cube,0,0,z 

RenderWorld 

Flip 

Wend 

End

GfK(Posted 2006) [#4]
Alternatively, if you require smoother movement:
Graphics3D 320,240,0,2
SetBuffer BackBuffer() 

camera=CreateCamera() 
CameraClsColor camera,0,0,127
PositionEntity camera,0,0,-5 
light=CreateLight() 

cube  = CreateCube()

Angle = 1

While Not KeyDown(1) 

Angle = Angle + 1
If Angle > 359 Then Angle = 0


RotateEntity cube,0,0,Sin(angle) * 20

RenderWorld 

Flip 

Wend 

End
