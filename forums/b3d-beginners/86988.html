This is easier...

Graphics 400, 300, 32, 2
SetBuffer BackBuffer()

While Not KeyHit(1)
    If KeyHit(57) Key = Not Key
    If Key Text 150, 150, "HI BRAZIL!"
	Flip
    Cls
Wend
