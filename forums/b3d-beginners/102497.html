AngelOnFira(Posted 2014) [#7]
Ok, I had this problem the other day and here is how it works. When you make a new file, it automatically tries to temp-save it to that folder, hence the error. Therefore as you were saying, it needs admin privileges.

The easiest way to fix this;

1. Install Blitz anywhere, that doesn't matter.
2. SAVE THE FILE YOU ARE WORKING ON IN A LOCATION THAT IS NOT USER-RESTRICTED (such as the desktop, or documents)
3. Profit

That fixed my problem and it WILL fix yours.
