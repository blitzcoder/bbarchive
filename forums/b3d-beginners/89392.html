_PJ_(Posted 2010) [#4]
Spacebar has a 'scancode' of 57,
so:

KeyHit(57)

or KeyDown(57) will return True for differing sues of the Spacebar:

KeyHit returns how many times the spacebar has been it since the KeyHit was last called (unless FlushKeys was called meanwhile)

KeyDown returns True if the Spacebar is currently held down. (be careful, since frames are faster than the average keypress, so your program will react to x milliseconds worth of holding down Space.)
