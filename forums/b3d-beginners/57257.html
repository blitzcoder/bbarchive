Unfortunately, as far as I know, the 'groups' you have in milkshape don't export as children /parents when exported to the b3d format. Obviously bones have parent child relationships but I don't think that will help much if it is meshes you wish to have child/parent relationships with.

Since you are using milkshape, the only advice I can offer would be to set the parent child relationships in blitz when you load in the models.

I could be very wrong though, and if so I'd be interested in hearing if anyone else knows how to set parent child relationships in milkshape.
