SopiSoft(Posted 2003) [#3]
here's an example for 3D:


Type enemy
  Field entity
  Field x#
  Field y#
  Field z#
End Type

Global alien=LoadMesh("alien.b3d")

;Make 50 objects
For i=1 To 50
    e.enemy = New enemy
    e\entity=CopyEntity(alien)
    e\x#=Rnd(-100.0,100.0)
    e\y#=Rnd(-100.0,100.0)
    e\z#=Rnd(-100.0,100.0)
    PositionEntity e\entity,e\x#,e\y#,e\z#
Next


i hope it makes sense ;-)
