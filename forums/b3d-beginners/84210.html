Stevie G(Posted 2009) [#5]
As well as the ID per car, you could also create an array of types ...

Type CarT
  field Entity
  field Blah
End type

dim CAR.CarT( 99 )

for l = 0 to 99
  CAR(l) = new CartT
  CAR(l)\Entity = Copyentity( CAR_MESH )
  CAR(l)\Blah = ???
next


Then to point to car 4 do this:

pointentity CAMERA, CAR(4)\Entity
