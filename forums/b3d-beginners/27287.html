Your code logic is incorrect, the last line

Graphics vh_scrwidth,vh_scrheight,vh_scrdepth,vh_scrmode

will always be executed regardless of the result of the test.


vh_scrwidth = 641 
vh_scrheight = 480 
vh_scrdepth = 32
vh_scrmode = 0 

vh_scrcheck=GfxModeExists(vh_scrwidth,vh_scrheight,vh_scrdepth) 

If vh_scrcheck=False 
Text 0,0,"error" 

Else

Graphics vh_scrwidth,vh_scrheight,vh_scrdepth,vh_scrmode 
Text 0,0,"success"

EndIf

While Not KeyHit(1)
Wend
