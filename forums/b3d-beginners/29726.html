b3dWriteFloat( VertexV#( surf,j,0 ) )

; b3dWriteFloat( VertexW#( surf,j,1 ) )
Next
Next
b3dEndChunk() ;end of VRTS chunk

first_vert=0
For k=1 To n_surfs
surf=GetSurface( mesh,k )
n_tris=CountTriangles( surf )-1

b3dBeginChunk( "TRIS" )
b3dWriteInt( k-1 ) ;brush for these triangles (surf -1 !!!)

first_vert=first_vert+CountVertices( surf )

Next

b3dEndChunk() ;end of MESH chunk

End Function

Eigther study the B3D Format and learn how to nest NODES to save multiple Child-Meshes inside one File, or, maybe easier, use the AddMEsh Command to add multiple MEshes together before you save it.

x1=loadmesh("chair.x")
x2=loadmesh("table.x")

x3=createmesh()

positionmesh x2,213,34,anywhere

addmesh x1,x3
addmesh x2,x3

freeentity x1
freeentity x2

now start with the saveB3D code to save x3
