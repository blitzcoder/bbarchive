This
It would be nice, if it could erase words, like from a song, but still leave the music intact.

is something that cannot be done in 100% certainty. It may be easy for human to recognise singing on top of a music, but not so for a computer, which knows nothing about human voice. The current solutions only notch the frequencies of the human voice and if you have ever done sound editing, you know that doing notches doesn't leave the music intact. Actually it rapes it pretty badly.

Audacity. If not too big budget means more than usual, options are Sound Forge, Cool Edit, Wavelab...
