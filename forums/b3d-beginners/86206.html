Terry B.(Posted 2009) [#4]
Function fillRect(startx,starty,width,height,r,g,b)
    color r,g,b
    for a=0 to width-1
        for b = 0 to height-1
            plot startx+a,starty+b
        next
    next 
return
end function

That should work.
