Akira(Posted 2004) [#5]
Yep, if you purchase the Mac version you get full access to the BMax forums which includes the ability to post, as well as access to the 'private' forums in which the link to the Win32 and Linux betas can be found.

You may feel it is silly purchasing the Mac version when you don't even own a Mac, but once you own a license for one version (ie Mac) you will be entitled to download the Win32 and Linux versions for free when they are released. Plus you also get access to the betas.

Be aware that while the current version of BMax does support OpenGL commands it does not contain a 'BlitzMax' 3D module. The 'BlitzMax' 3D module is apparently scheduled for release next year (2005) and will need to be purchased by existing BMax owners if they wish to make use of it's features.
