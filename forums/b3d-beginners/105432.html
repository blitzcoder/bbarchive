Matty(Posted 2015) [#3]
If you are after a cheap modeling and animation tool then milkshape is well worth it. It is designed for low poly modeling for games and although quite old is still okay.

The interface on the more expensive applications (3ds max, maya etc) is usually much easier to use and more powerful but often lack the easy export path (and are hideously expensive).

You can do some things in code as well, but I'm not sure if that is what you were interested in.
