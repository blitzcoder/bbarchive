Floyd(Posted 2003) [#2]
How about this?
If Abs( MilliSecs() Mod 10000 ) < 5000 Then useRed Else useWhite

The Abs is there in case Millisecs() has 'rolled over' to negative numbers.

There are other ways, but this doesn't need to remember previous results.
