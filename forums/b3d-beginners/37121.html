BlitzSupport(Posted 2004) [#4]
You could just define as many of those temporary Tvectors as needed as globals, outside the function.

Type Tvector
	Field x#
	Field y#
	Field z#
End Type

Global u.Tvector = New Tvector

Function Acc.Tvector ()
    u\x = 0
    u\y = 0.1
    Return u
End Function

test.Tvector = Acc ()

Print test\x
Print test\y
