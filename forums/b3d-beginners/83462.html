dont know about activity STATUS persay, but you can load with the old openfile("pathname.ext") and read em.

like:
file = openfile("pathname.ext")
if file <>0
   while not eof(file)
    dat=readline(file)
   wend 
endif



And writefile
file = writefile("pathname.ext")
if file <>0
    writeline file,dat
endif


hope that helps.
