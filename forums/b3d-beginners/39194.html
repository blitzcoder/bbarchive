Tiger(Posted 2004) [#2]
Yes you can try this:
Type Colum
 Field Y[4]
End Type

Type Types
 Field X.Colum[4]
End Type

Type StoneInfo
 Field Mode.Types[4]
 Field Colour%
 Field pX%
 Field pY%
 Field pStart%
End Type

Local Test.StoneInfo
Test=New StoneInfo
Test\Mode[0]=New Types
Test\Mode[0]\X[0]=New Colum
Test\Mode[0]\X[0]\Y[0]=3
Print Test\Mode[0]\X[0]\Y[0]
;Delete them all
Delete Test\Mode[0]\X[0]
Delete Test\Mode[0]
Delete Test
WaitKey


But it gets a little bit complicated after a while. :)

 Field Y[4]
End Type

Type Types
 Field X.Colum[4]

Type StoneInfo
 Field Mode.Types[4]
 Field Colour%
 Field pX%
 Field pY%
 Field pStart%

test.StoneInfo=New StoneInfo
