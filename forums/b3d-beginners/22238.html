It's a new feature called "insert-o-matic", which randomly toggles insert/typeover mode as you code. The faster you code, the higher chance it will activate.

It's a new feature Mark put in becuase there were complaints that programming in Blitz was too easy. And that seemed like the best method to artificially generate occasional syntax errors and the like to make it a little more challenging.
