How do I specify in my custom type that one of the fields contains a handle to a mesh that will be loaded and returned later ?

Thanks


GfK(Posted 2003) [#2]
m.MyType = New MyType
m\Entity = LoadMesh("Mymesh.3ds")

Type MyType
  Field Entity%
End Type
