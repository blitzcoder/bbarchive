IPete2(Posted 2005) [#2]
Rantz,

You need to either hard wire stuff in place or use something which has the ability to place objects and name them in the map.

Gile[s] has this place and name ability. It also has a fantastic lightmap capability.

Some code exists in the Roman Baths demo which takes objects placed and named inside Gile[s] and attaches some particle systems to them, for torches. This is what you need I think.

Otherwise try the demo of HomeSpin or WorldSpin, alternatively write an editor for yourself, this will make levels for any game you are doing really quick and easy to develop. You will also have 'real' control over them in your own code too.

Good luck.

IPete2.
