cyberseth(Posted 2003) [#4]
Flynn,

Firstly, as skidracer said, BlitzPlus can't do Input in the graphics window. You have to use something like WaitKey instead:

Text 10,10,"Really Quit? (Y/N)"

Repeat   ; loop until Y or N
    key = WaitKey()
    If chr(key)="N" or chr(key)="n" Then Exit   ; exit loop
    If chr(key)="Y" or chr(key)="y" Then
        iGameinProcess=False
        Exit   ; exit loop
    End If
Forever


Secondly, the concept of Flip is that you draw EVERYTHING you want to put on the screen, and then you Flip it. Until you flip it, the screen will not change. (Unless you SetBuffer FrontBuffer() but you don't really want to do that.)

The reason we Flip is because of animation. You don't want the user to be able to see the screen in the process of being drawn (even if it only takes 2 milliseconds), it looks unprofessional and flickery. You just want to draw the screen in secret while no one's looking, and then when it's all done you Flip the new frame to the front which is what the user can see on the monitor.
