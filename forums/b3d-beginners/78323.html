How do I Synchronize 2 Global Variables with each other?

The game has a global variable to keep track of the weather.

When the game starts that variable should = 0, and then begin
counting upward to determine the weather.
At first the weather will be clear, but then will get snowy ect, as
the variable counts upwards.

When a second player joins the game, his Global Weather variable
will = 0, since for him, the game just started.

So then the first player might be in the middle of a snow storm

Cp(Posted 2008) [#2]
Client and server. The vars like that are in the server, and the players connect to the server.The clients contain individual player data, while the server contains data that is the same for all players.
