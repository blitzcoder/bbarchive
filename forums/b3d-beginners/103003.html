xlsior(Posted 2014) [#7]
Is it possible to make a cellphone game using blitz3d? I think the only thing holding me back is not knowing the numbers that are assigned to the buttons.


No, the main thing holding you back is that the platform simply isn't even remotely compatible.

Blitz3D only runs on x86 windows devices with DirectX.

Despite the similar name, even the 'windows' phones aren't x86 devices, they run a different processor architecture altogether (e.g. the Lumia 630 uses an ARM-based snapdragon processor, not intel x86)

You can create games for those platforms, but not with B3D.
