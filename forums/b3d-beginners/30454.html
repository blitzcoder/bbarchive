Stevie G(Posted 2004) [#2]
I'm not sure if I've picked you up right here but you would need to use 2 x renderworlds for this like so ..

showentity object
showentity camera1
hideentity camera2
renderworld()

hideentity object
hideentity camera1
showentity camera2
renderworld()

flip

You should then only see the object in camera1 not on camera2. Hope this helps.
