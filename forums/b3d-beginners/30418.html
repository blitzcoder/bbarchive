DarkNature(Posted 2004) [#2]
Here you go:
;outside of main loop
delayTime=1500 ; 1 1/2 secs
time_to_wait=millisecs()+delayTime ; use 0 for no initial delay

;in main loop
if millisecs()>=time_to_wait

 doThings()
 time_to_wait=millisecs()+delayTime

end if


Have fun.
