octothorpe(Posted 2005) [#2]
Yes. No. It's clear.

Blitz's internal object lists always contains all the objects for each type. Even if you have no pointers to an object. This is because Blitz keeps its own pointers secretly.

Type foo
End Type

f.foo = New foo
f.foo = New foo

i = 0
For f.foo = Each foo
	i = i + 1
Next

Print i
