AbbaRue(Posted 2003) [#2]
Might I suggest you go to help, select home and select the language reference area and start reading the language reference.
Then Read the Basic Tutorials section, after that try the 3D tutorials. Look at the source code it gives you and follow it through until you understand it.
Once you get an understanding of the 3d tutorials, find a 2D or 3D sample that does approximately what you want to do and start modifing the code yourself, to see what happens.
First save the sample under another name so you don't loose the original.
Programing in Blitz is like speaking to the computer in English, most commands do what they sound like they do.
