big10p(Posted 2005) [#8]
If it's only a smallish portion of the screen you want to zoom then it's possible. I have this ability in my current app where I have a zoom window ~3rd screen size. I do it by manually drawing the zoomed pixels to an image buffer, using WritePixelFast - it's fairly quick. I tried a version using solid Rects to draw the pixels but that was too slow.
