Zethrax(Posted 2005) [#3]
Since you mention Blitzbasic (2D) and Blitz3D I'm not sure if you're talking about the difference between 2D screen coordinates and 3D world coordinates, or the difference in the 2D screen coordinate systems of both programs.

3D coordinates are as Shambler said. 2D coordinates use the x (horizontal) and y (vertical) screen axiis which start at zero at the top left corner of the screen and increment positively as you move right and down, respectively. This works the same for both Blitzbasic and Blitz3D's 2D drawing, as far as I know.
