Zethrax(Posted 2012) [#2]
I think Wings3D uses a different coordinate system layout than Bliz3D. From what I can recall, when you export a model in Wings3D you have an option to change the coordinate system layout (or transformation matrix) used. Have a close look at the Wings3D export options and try experimenting until you get the model how you need it to be.

Also take a look at the LoaderMatrix command in Blitz3D. This command can be used to change coordinate systems when loading specific file types.
