Kryzon(Posted 2012) [#3]
It helps if you keep a browser page opened at an online Hex color viewer.

I find Blitz3D's original color-scheme just nasty...


I use the classic Blitz Basic color-scheme which tastes like homemade apple pie to me:


Colors for classic BBasic:
rgb_bkgrnd	206060
rgb_string	ff00
rgb_ident	ffffff
rgb_keyword	ffe75f
rgb_comment	ffff
rgb_digit	c8f0ff
rgb_default	fff0c8
In case anyone is curious, these values are placed inside the 'Blitz3D\CFG\blitzide.prefs' file - make sure to backup the original first.
