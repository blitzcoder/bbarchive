Gabriel(Posted 2009) [#5]
A game state system would be better IMO.

Something like

Select GameState
   Case GAMESTATE_MENU
      clearworld()
      while 
         if race_start then State=GAMESTATE_RACE
         menu loop
      wend

   Case GAMESTATE_RACE
      clearworld()
      load ode world and ode things
      while (race)
         race loop
      wend

      destroy ode things
      GameState=GAMESTATE_RACE

End Select
