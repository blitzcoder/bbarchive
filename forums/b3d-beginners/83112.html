Mahan(Posted 2009) [#4]
@chrisnorris007: Sprite Candy does a lot for the money. It includes a GUI, tools to merge icons like inventory buttons etc. to a texture (to minimize system resources). The GUI is using the Sprite Candy lib itself so it's 3D based and you can do the same things with the GUI as you do with other GFX with the LIB, like rotate, alpha, different movement effects etc.

But no GUI i know of is "for MMORPG", they are 2D-overlay GUI systems for 2D over 3D, as simple as that, and can be used for any game.
