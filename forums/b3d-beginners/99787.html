RemiD(Posted 2013) [#2]
If you want to make a 2D game with a side view or a top view, you can draw each 2D tile by hand with :
Photofiltre Studio 7
or
Paint.net
or
Gimp

Then to select, position, save the tiles, you can do it easily in Blitz3d.

If you want to make a 2D game with an isometric view you can use 3DWorldStudio to create 3d tiles.
