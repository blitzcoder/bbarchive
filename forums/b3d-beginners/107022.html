Function InterpBetween(X,Y1,Y2,R1#,G1#,B1#,R2#,G2#,B2#,H1,H2)
Local Y
Local Dist#=Float(Y2-Y1)

Local GradR#=Float(R2-R1)/Dist
Local GradG#=Float(G2-G1)/Dist
Local GradB#=Float(B2-B1)/Dist

Local GradH#=Float(H2-H1)/Dist

For Y=Y1 To Y2

Color R1,G1,B1
Plot X,Y

R1=R1+Float(GradR#)
G1=G1+Float(GradG#)
B1=B1+Float(GradB#)

H1=H1+Float(GradH#)

Next
End Function
