makakoman(Posted 2003) [#4]
Terragen CAN export a heightmap as a bitmap. You just need to download the F.E.O. - Windows .bmp plug-in. Then, select the Accessories... button from the terrain group. Don't forget to resize the .bmp file to 256. It exports as 257. You can also export as raw format and use a program like Paintshop Pro to load raw file, resize to 256x256, and save as .bmp.

I don't think you can use the output commercially though unless you register the software. I think terragen is such a great tool for terrains that I registered it.
