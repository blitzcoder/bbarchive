RemiD(Posted 2016) [#9]
Btw, if you use bilinear filtering, i suggest to use 1 texel each 0.1unit or 1 texel each 0.125unit. With bilinear filtering activated, the shades look better with less texels per unit (with reason), but without bilinear filtering activated (or any other texture filtering) it looks better with more texels per unit.
