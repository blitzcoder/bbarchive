I have an object that I have two textures for - a colour map and an alpha map. I've been experimenting with the texture flags, but I haven't got very far.

killertomato(Posted 2007) [#2]
You don't load them as two seperate textures. Tga texture format supports the alpha channel. In your paint program load the color map and then load the alpha map into the alpha channel and save it as a targa. Then in Blitz, just set the alpha flag on.
