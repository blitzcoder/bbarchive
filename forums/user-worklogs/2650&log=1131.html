FireFighters - B2D Game(Posted 2002-08-06)
This is my first entry, so I'm kind of excited :)

FireFighters is a simple solo strategy game. Like many strategy games it is all about resource allocation. You command four fire sqaud cars fighting a forest fire. Your objective is to vanquish the fire before any of the houses scattered through the forest burns down.

What was done so far:
Ingame engine. Calculates fire behaviour, squad car behaviour etc.
User input handling.
Pathfinding. based on Patrick Lester's AStar code (great job, Patrick) modified for different terrain cost.
Minimal graphics.

What needs to be done:
Improve graphics (greatly!)
Add sound/music.
Add Front end.
Add victory/lose conditions.
Add more maps (currently only 1 created).
Tweak the parameters.

I would also like to get a feel as to the speed the game runs on other machines. The game is geared for 30fps (there is a counter at the lower right corner of the screen). Results from other machines would be appreciated.
