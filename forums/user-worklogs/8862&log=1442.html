welcome(Posted 2007-04-11)
Ok let's try this again, my first log seemed to have been eaten...

Anyway, 'Zero' (working name) is a top-down, 3D space combat game inspired by Wing Commander I, Asteroids, EVE Online, and a touch of the movie Aliens. Although the primary gameplay will be focused on arcade action, the game will feature a rich universe and emerging storyline that can be experienced in a quasi-RPG style. Blah blah blah, onto some wip screens:









Keep in mind, these screenshots are way early in development, but the fact the games is both playable and actually kind of fun is inspiring. Considering I've been using Blitz for about 4 years now, this is a pretty large step for me. Zero is built atop the rubble of many a failed project before it, however, in that time I've learned a great deal about programming, game art, and modularity.

I'll try to keep this worklog current with new screens as much as possibly. If you are interested in the minutia, check out my blog
