BlitzSys DLL(Posted 2002-02-01)
Good progress made on BlitzSys DLL this week. Although finishing up the Font Requester was a total git..

In short, you now have (almost) 100% complete control over the requester (minus a few hook bits and pieces)..

- Enable any of the font requester features..
- Set the initial Face and Size of the font..
- Set the min and max size allowed to be selected by the user.
- Set the initial Colour of the font, and get its colour if and when the user changes it.
+ Much more.

Path requesters are also in now.. Not a lot to talk about here.. However, I cant seem to find the comctrl way to do it, anyone has any info on opening a common control folder requester, mail me please...

Checksums will be in the next version too.

Version 1.04 will be released in a few days.
