Tenok Image Manager 1.0(Posted 2005-05-05)
A simple filter based image editor. Includes the following features:
* BMP font creation
* Filters (Blur, Sharpener, Seamless, Noise etc)
* Image loading from the harddrive and by URL (bmp, jpg, png, tga, ico)
* Image saveing (bmp, tga, ico)
* And more (coming)
