3DS MAX 4/5 plugin(Posted 2002-08-03)
Been working on the 3ds max plugin for blitz3D. It's not a 100% high priority for me but it is shaping up to be the best plugin out there for max export! supports absolutely everything Mark has put into the Blitz3D file format.

The first release had bugs on half the test platforms, and I think this was due to the way it was called and set up.

I need to do a little more tinkering to make it stable for all, then I will be releasing another version - hopefully bug free!
