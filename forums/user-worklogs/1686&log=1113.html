Putting it all together(Posted 2002-03-22)
I'm currently focusing on my ball game, which uses some of my physics code, including my fast collision code - sphere(moving)-to-poly(moving) collisions with optional sliding. I'm quite proud of the results, only one level but its fun, easy to pick up, and technically there.

Level design is easy - all done in any 3d application and optimised in Blitz.

The graphics need some work and I'm thinking of removing the bits of Blitz collision I do use in favour of something smoother, but, overall this project has legs.
