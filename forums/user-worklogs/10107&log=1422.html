Dev Update(Posted 2007-02-23)
Work on SWARM is going well. I'm currently drawing the shapes for the enemies with Inkscape and thinking about how I want the levels to be. At first I wanted to have vertical scrolling levels but now I'm thinking about maybe doing 4-axis levels like you can roam around in any direction. I think this would suit the feeling of the game as I want it.
Oh: and I'm listening to the Rocky Balboa soundtrack while working on SWARM =) One of the best OSTs in the past 2 years imho.


It'll be called SWARM(Posted 2007-02-12)
1. I now got my iMac. It's a beautiful 17" all in one computer system with the most modern operating system on the harddrive that is available today: Mac OS X. I won't rant about how archaic Windows is (even Vista) compared to OS X. Just wanted to let you know that I'm on Macintosh now (important since it's my development machine).

2. My website ( http://www.moonshape.net ) is ready and 100% created on my Mac using iWeb. It was very easy but I must say it would be better if iWeb would create PHP based websites rather than simple html. But nevertheless it's pretty easy to use, I like it :-)

3. My game will be called SWARM and it will be a 2D shoot em up merged with a snake like gameplay where you must build up a swarm of wingmen. Did you never wonder how a single spaceship is able to defeat all those bad guys in the normal 2D shoot em ups? SWARM will give you the opportunity to send a whole army to kick the bad boys asses =)
