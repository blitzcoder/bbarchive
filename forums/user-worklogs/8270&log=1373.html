Devil Particle System

Return to Worklogs

1.0 Final(Posted 2006-09-29)
The Devil Particle System is a particle engine, designed to create particle effects such as fire, smoke, spark or explosions. The point of this system is, that it is a so called 'single surface' particle system! This means, that all particles of one emitter are in one blitz mesh, which is a lot faster than Bblitz3D's sprites. Also, this system comes with a large particle image collection of approximately 100 particle images (see the fourth screenshot).

Download the latest versions on devil-engines.dev-ch.de
