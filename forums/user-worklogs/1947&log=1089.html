ResED!(Posted 2003-04-01)
Hi everyone guess I should have done this a while ago, I started a resource editor based upon one I used on the Acorn computer over many years. It is availible from http://www.btinternet.com/~nigelibrown in very early stages but I will be working on it when Im not updating websites for people etc....

It is not restricted to a single window, and now saves your files into it's own format that I will publish later I also want it to export code that can read the files rather than generate Blitz+ code to store all the info then when you update the .res file your app should also be updated :-) It uses a popup menu over the window objects that it creates to add/delete/edit gadgets of most types.

Please remember this is a WIP and not really ready for use but I am very open to suggestions for features...
