Well well(Posted 2004-10-06)
Many moons has passed since i started this project. I have the project draft finished and will send it to interested parties.

In short:

Mixed 2d/3d game
mp3 music (your own or the bad in game music)
"Massive" Enviroment
Module based
Ships with Editing tools
The game will not feature any data (Due to copyrights from owner of the game its based on, so you need a copy of the books to add it)
There will be a few things though:
The rule system will be incorporated
Pictures of common things.
Basic skills
No Equipment
No weapons
1 adventure (based upon the rulebook data)

For the rest you will need at least the rule book to add data into the database.
