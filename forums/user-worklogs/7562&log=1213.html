Alpha Release(Posted 2004-06-23)
Game Manager is an Explorer/IDE for managing assets during the game development process. Written in C#, Game Manager provides a grouped tree view of all the files in your project, with configurable groups so you can customize the file listing to suit your work style. External apllications such as editors, convertors, and compilers can be configured and made available for fast access when you right-click on any file in the file list.

In addition to the usual Explorer-like file management, Game Manager provides a plugin system for attaching automated processes to specific files. Plugins can be external applications linked into Game Manager via the Plugin Editor, or integrated C#/VB.Net DLLs. For linking in an external applications, the built-in Plugin Editor allows you to define the settings, actions, and file types that the Plugin will handle. DLL style plugins can access the full feature set of the .Net system to perform any task fully integrated with Game Manager.

Future versions will include a build system which will scan all files in the project and perform specified plugin actions automatically on any files that have changed. For instance, if you edit a map, GManager can run the lightmapper program automatically with presets that you define for the specific map. Other plugins will perform file conversions, generate script includes from model and map data, GUI editors, and other applications.
