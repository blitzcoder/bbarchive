First working version ready(Posted 2005-04-04)
DCFG is a program that gives you the ability to easily generate frontends for commandline-based programs. You specifiy the possible settings and switches in an editor, and load this into the program which generates a frontend.

MoPS-Editor(Posted 2004-10-02)
I'm currently programming a Particle System called MoPS. Some of the features are:
Single Surface per emitter
Various settings
Four types of emitting(point, sphere, line, cube)
Event-based sounds(eg. sound on creation, deletion, or after certain time-span)
Bouncing planes
