Getting there...(Posted 2005-11-24)
Well I have re-started my ongoing attempt to generate a computer game and instead of going for the next 3D killer app. I've started off with something a little bit simpler...

Candy Cascase (yet another Tetris).

So far I've got the basic game running, and some nice 'sweet' graphics for the blocks, even basic sound just some polishing and tidying up then package it up and put it online.

Shouldn't take long, however it could be a case of 99% there 99% to go.

Things left to do:

1. Restart Bug needs fixing, one play is never enough for your average Candy Cascade addict. [Done]

2. Leveling up screens [Done], with appropriate rewarding soundfx.

3. Scoring to increase based on the speed of the player. [On Hold - Could unbalance scoring]

4. Hi-Score table. [Done]

5. Special effects e.g. line removal explosions using particles? [Done]

6. Tidy up background graphics [Done]

7. Wrap up into standalone exe and test on other PC's [Done]

8. Package as single exe and place it online as freeware.

And I need to find a nice object that can convert strings to a nice bitmap using a tiled alphabet... [Done]
