Introduction of BlitzCoop(Posted 2008-03-27)
When I discovered a free hosting-site with database support, I wanted to create a website. And I got this idea:
A website to meet other game-designers(programmers, modelers, componists) to cooperate with. You can submit an 'idea' for a game and other people can join you. They can submit the content you ask and you can create a freeware game an put in on BlitzCoop.
I hope the idea is clear. I don't yet give the URL, because there is still a long way to go...
