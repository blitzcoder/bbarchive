Word Munch Starting to take shape(Posted 2007-08-04)
This is game where you basically have a snake and you have to collect letters to make words that come up in a list you need to complete for each level.

Think snake, but with words instead of pills.

You'll gain points for getting letters you need for the word, lose points if you get letters you don't need, and gain bonus points for how long it takes you to complete the word.
Can't see it taking more than a few weeks to write, assuming I manage to actualy spend any time on it that is.

Initial thoughts and reasons(Posted 2007-06-14)
I've seen that a lot of simpler type games now, such as those you find on miniclip and other online game portals. It looks like most of these games could be written by one person. Maybe just outsource the music and some graphics.

I'm pretty sure I could write a couple of thse type of games in order to build up the start of a portfolio.

I'll look at doing some word games and some number games. Mainly adding a new twist on established games.

The first few games will be freeware, just to get my name out there a little, but then work on more professional style larger projects for shareware release.
