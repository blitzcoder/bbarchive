announce(Posted 2007-02-24)
First worklog of Dreamotion3D, a 3D engine programmed in C/C++ and Direct9c for BlitzMax.

Languages :
- BlitzMAX
- Purebasic
- C/C++
- C#
- Other...


Features :
- Based on DirectX 9.0c
- Flexible entity system
- Flexible brushes system
- Script material for total exploit DirectX State Render
- Texturing Basic, Multi-texturing, Bumpmapping, Lightmapping
- Lighting multiple lights
- Meshes files : 3ds, x, md2, b3d,ms3d,...and more in a future
- Textures/Images : bmp, jpg, png, dds
- Physique engine can be integrated (example : PhysX module)
- Shadow Volume, shadows mapping
- Collision detection.
- Terrain system (Brute force, Quadtree, Octree, Splatting, LOD,...)
- Special Effects (Billboarding, Particle System, Sky, Fog,...)
- 2D functions (sprites, images,...)
- Occlusion system (Fast frustum culling, AABB,...)
- Pixel shader, Vertex Shader, HLSL
- And more....

Available only on Windows system.

ENGINE Status : 65%, beta2 version v2.0.5, available during March 2007

also exist for PureBasic (already available), and will be soon available for C/C++ - C#.

Team: TerryMyke and Seyhajin
