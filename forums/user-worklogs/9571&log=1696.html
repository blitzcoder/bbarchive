First log...(Posted 2009-07-27)
* Here I come !

This entry is THE entry :)
FastGui is expected to be a very fast Gui using FastImage library ( cf : MikhailV's dll @ FastLibs.com )

It is Fully skinnable, and supports many gadget .
Currently, here is the list of the supported gadgets:
- Window
- Label
- Panel
- ComboBox
- TextField (InputBox)
- GroupBox
- Button ^^
- RadioButton
- CheckBox
- Tabber
- Slider (spinner)
- TrackBar

Still missing gadgets ( I really missed those gadgets... ) are :
- Menus
- ListBox
- TreeView

It comes with MaxGui-like commands and a library for Event handler
Also provided with FastGui :
* TList ( for blitz3d )
* TMap ( for blitz3d )
* Ini File Library ( for blitz3d )
* Registry Library

Except for the Registry library, every lines are copyrighted ©Bobysait.

The library will be released as shareware when I'll come up with a stable version wich supports Menus and ListBox ( plus, a template for FileRequester and ColorPicker ).

/!\ FastGui can't work without Fast-Image
WhatEver, it works with the demo version of Fast-Image. So You'll be able to test it before getting a full copy.
