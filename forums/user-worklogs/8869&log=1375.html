The Start (and still not)(Posted 2006-09-30)
Archon 2160
A tribute to the wonderful smashhit, Archon, from 1983.

If I would compress all my designdocuments, visions and ideas into a few sentences I would say that this is all about creating the gameplay and multiplayerfun that Archon did deliver but still taking the concept into todays technology and graphics.
It's all about learning to play the 12 different units and dive into several clashes with the opponent on the arena and still playing tactical on the "gameboard". Well, if you want to know more please visit the Archon 2160 wikipage at: http:\\archon2160.pbwiki.com

So here we go:
Archon2160 as an idea has been there for a long time but we started the development in Spring, 2006. We were a small dev.team where I was the Project Manager and Lead Artist. As time went by I realized that I was the only one contributing to the project I was burning for and in August, 2006 I decided to stop the project and try to find a whole new team. Now we all know that finding dedicated teammembers is hard so eventually I decided to start programming it myself. After a short while I decided to use Blitz3D (as Im a pretty lousy programmer) and this is my very first project in B3D so PLEASE be gentle with me and don't hammer to hard cause Im actually a little bit proud of achieving this much already.

I've now been working on this for a week (yes I didnt find the Worklogs until now :P) and this is what Ive got so far:

2 Playable units (to be able to play 2player you need at least 1 gamepad/joystick)
2 Playable guns (Flamethrower and Grenadelauncher)
2 different controllingtypes (8-way and carsteering)
Working HUD (at least the energy and weaponstatusbars)
Working Effects and particlesystems
Working Collisionsystem
Animated obstacles on the arena (nades still go through them (?) )




Thanks for reading this first post and I hope there will be a lot more soon. My intentions is to release a first Demo within the next few days.
