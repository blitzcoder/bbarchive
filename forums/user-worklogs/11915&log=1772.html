presentation(Posted 2010-09-03)
Hello everyone.

a small presentation of my staff Wrap Ogre.

He is currently a project in development phase, but the features are already interesting.
This wrap is not specifically designed to bmax, and therefore the set of instructions is simple,
without internal BMax architecture OOP. Nothing is stopping you think I write as a wrap for Blitz3D for exemple.

The basic functionality offered by Ogre are already available.
There are still a lot of work, and many elements to include, either of Ogre itself, or at the physics engine.

The documentation is also still basic, but is expected to grow steadily.

Some examples (from some of the last package ogre) are provided.

So I work quietly to ensure that the project is expanding over time and become a serious alternative for 3D
programming bmax, or any other language BASIC.Be kind, this project is in the early stage of its creation, so
it is certainly filled with a lot of imperfections.
