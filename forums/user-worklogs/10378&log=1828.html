Initial status(Posted 2011-08-06)
Work has begun on BTHINM. Yes that's the worst working title ever, I just needed a name for the project folder. :P

I got the inspiration for the game while playing "Realm of the mad god", a superb mini-MMORPG in flash (http://www.realmofthemadgod.com). I will probably need to change the visual look of the character since it is merley identical to some characters in ROTMG.

The object of the game will be to defeat animals, NPC bandits and such with friends. I got most basic things working, like moving, animation, collision, shooting. Netcode is almost complete as you will see in the next worklog.
Oh and I almost forgot, the character is built up from pieces of clothing so changing clothes will also change appearance. You can remove the hat, change the vest but not the legs and so on.
