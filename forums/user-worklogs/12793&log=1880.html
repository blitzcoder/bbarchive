First post(Posted 2013-09-13)
OK, this is a work log for My new project, a text based and graphics assisted game set in the universe of Mana Khemia with mostly new characters, although some will be brought over from the original games to be teachers and of course Principle Zeppel will be in there cause I think he is funny.

This project is already at Alpha about .10% complete. I have a working character creation and most of the basics are in place. I've already set up the 15 students you can choose from to join your workshop and I'm working on the conversation menus as well as the list system that allows me to make one menu serve as many.

Anyways this game will feature a new alchemy system that borrows allot from the Mana Khemia games but also introduces new methods and ingredient altering. The combat will be a little more tactical cause its turn based in a text game. The game should still be fun though especially if graphics don't mean everything to you.

I will post a screenshot soon.

I have a working word wrap function. If anyone is interested in using it I will be glad to share the code, although it requires a type I made or slight modification.

If you are interested in the project let me know, email info on my account page.
