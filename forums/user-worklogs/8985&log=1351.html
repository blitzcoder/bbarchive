First Screenshot(Posted 2006-07-11)
The old Cyberscape 3D has been cancelled, because of some performance problems, feature limits and a lot of bugs, so i have now started a new version of Cyberscape. It uses WinBlitz3D to access the Win32 API.
It features:

- Multi-layer Terrains
- Instancing of mesh objects
- Very easy to create, manipulate and texture objects
- Unlimited of terrains
- Animated models
- Cubemapping, both on objects, and in the build-in feature for water
- Scripting in pure Blitz3D language
- Realtime-lightening
- 3D & 2D sound effects
- World-over-net feature: Connect more Cyberscape computers together, and work with more computers on the same world
- Imports: DirectX .x, Blitz3D .b3d, 3D Studio .3ds, Lightwave .lwo, Milkshape 3D .ms3d & .txt
- Exports: Blitz3D .b3d, Cyberscape World .csw (specs included with the release)
