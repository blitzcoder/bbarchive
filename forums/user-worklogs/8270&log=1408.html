Devil GUI

Return to Worklogs

1.0 final released(Posted 2006-12-16)
The Devil GUI (former XGui2) is the successor or the XGui. This GUI lib has been supposed to be called 'XGui2'. But later it was called Devil GUI and put to the other Devil-Libs. There is a skin SDK included. This is for very advanced users only! This GUI it made for Windows and Mac skins. The *.skin file is very flexible, but not enough flexible for a complicated skin like Windows Aero yet. All the skin *.png and *.ini files are packed into the *.skin file. The skinwriter.bb packs it, so you can basically create your own skins.

This GUI supports following gadgets:

Window
Tab
TabPage
GroupBox
Button
CheckBox
RadioButton
Image
Slider
Spinner
Label
ComboBox
ProgressBar
ScrollBar
Icon
ListBox
Edit
Menu


Following dialogs are supported:

Message box
Color picker
