Music to your ears(Posted 2005-12-28)
Successfully rewrote the Sound Lib to allow for short loops to be repeated in patterns, this allows me to build whole new audio tracks from sample components as the game progresses. The patterns can also react to game events allowing for more organic sound tracks, indeed the sound track will be different everytime you play the game as it is constructed from the events you trigger.
