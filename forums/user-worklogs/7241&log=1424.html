Earth Defender(Posted 2007-12-16)
I am currently working of the kind of game I normally like to play myself:

- it can be played without having hours of time
- simple to learn, hard to master
- challenging strategy wise, not randomness dependent
- offering different challenges that use the same base "ruleset"

All in all: It is basing on concepts of tower defense mechanics, tech war mechanics, dota and a few others related to this kind of games and some mechanics I believe that always were missing in this genre.

The core mechanics of that are all fully integrated (loading of XML levels, game object definitions etc etc), what I now need to do is design the needed graphics and texts to fill it with life.

I was able to implement this quite fast actually thanks to Grey Aliens great framework in the 1.05 prerelease version and a few other modules and libraries I either had or wrote in the past like my priority queue module and the like.
