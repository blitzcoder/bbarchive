Introduction (15 march 2008)(Posted 2008-03-15)
Game Designer is a program to create 3D-games easy and fast.
You can load meshes, heightmaps, sprite images and a lot more in GD.
It also has a built-in terrain editor and a treecreator.
It automaticly applies high-quality shadows and physics.
