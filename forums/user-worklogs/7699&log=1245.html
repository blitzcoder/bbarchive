Working ...(Posted 2005-02-27)
Yup yup, altough I dont tell much about it im still working on it.
Tonight I managed to finish the portal systems (to go to a map to another) wich isnt as simple as it sounds since it requires player disconnection from mapserver of origin, redirection to the mapserver managing the map he is going to, and player information transfert ...
Ok actually it is supposed to be simple, as it should be just like a reconnection, but somehow I managed to make it harder to do. <__<;

I also finished a first version of something 'quite' important wich is the player infos recording system ... you know, the thing that makes the server remember that you wandered arround and gained xp & items. Not the kind of thing I could delay much longer in the schedule (what schedule ?!).

So to recap, so far theres :
- Players movement & chat (public, private),
- Basic player-monster fighting managment,
- Gates allowing to go to other maps,
- Basic game interface (text input & function, message log, windows system, almost minimap ...)
- Mapserver begining of graphical interface & tools for gamemastering (so far monster adding / removing at points or in mouse-drawn zones)
- Main server basic fonctions (whois, kick, global notices & stuff ...)

Wich means it approching the point ill do some private beta test sessions. But before that, I need to find myself a second PC to do my own basic multi-clients tests, since theres a while I couldnt do any real tests (for now I can only run the client multiples time on my own computer. Needless to say having the main server, mapservers & 2 clients runing on the same computer isnt the best testing condition I can hope for).

Thats all for now =)
