I was asked to do a page on setting up BMax on Mepis, and I thought it was a pretty good idea, so here goes:

1) Open up '/usr/lib' in a terminal.

2) Check for GL dev support files. (should end in dev and have gl in the name.)

3) If they are missing or don't work anyway (you get some error message about libs,) go to www.mepis.org/node/10128 for a list of libraries.

4) STEP 5 IS DANGEROUS! BACK YOUR SYSTEM UP BEFORE ATTEMPTING!

5) Remove all GL related libs from your lib directory. Go download a new set of packages from the list in step 3.

6) Use KPackage to install them. Make sure they are all from the same family to avoid dependency problems.

7) Fire up BMax and see if it works! If it doesn't, sorry, I really don't know what else to do. Consider Ubuntu Linux.
