Um... well I'm a dumbass, but considering there's been no replies...

The .tar.gz file goes to: /var/cache/apt/archives

Where the actual files that are unzipped go to can be found in the synaptic package manager. Find the package you installed, right-click on it and select properties and then go to the installed files tab.
