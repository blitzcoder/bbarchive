I needed a bit of a break from the GTK module, so I thought I'd throw together a module for RealTimeBattle, which allows you to program robots in different languages and compete against other such robots in various game arenas.

As usual with these modules I do, it seems to take longer to put together the documentation than it does to write the code :-p
Anyway, I've tried to make it as Max-friendly as possible - you can create a robot in only a couple of lines of code - but I've not tested it to death (only spent a day and a half making it...), so any problems with it, let me know...

You can download it via syncmods it with :
syncmods -u your_user_name -p your_password bah.realtimebattle

Apparently it can also work in Windows, but that needs cygwin installed - which seems like a lot of effort to me.

Enjoy :o)
