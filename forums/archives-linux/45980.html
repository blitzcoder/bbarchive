Hi Noel,

The source code is the same, it is just the binaries that are different.

Syncmods as usual, copy the contents of your 'mod' folder to your BlitzMAX Linux folder, and reboot in Linux. Select the Build Modules option from the IDE and it will generate the BlitzMAX Linux binaries for you.
