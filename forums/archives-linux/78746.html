Canardian(Posted 2009) [#10]
openSUSE 11.1 runs fine on PS3, and as far I know BlitzMax runs on openSUSE 11.1 too.

openSUSE is a bit more hardware closer than Ubuntu, especially on PS3, where it can utilize the video RAM using its built-in driver.
It's also easier to setup on PC and PS3, since everything can be done from the graphical KDE 3.5/4.3 UI.

However, you need to cut down the list of installed default programs, since openSUSE is quite bloated if you let it install all programs by default, and then the PS3 is swapping memory a lot.
