dawlane(Posted 2012) [#6]
Open a terminal and type the command 
sudo rm -f /usr/bin/g++
sudo ln -s /usr/bin/g++4.4 /usr/bin/g++
to make a system link, but you have to make sure that you have the g++4.4 package installed first.
