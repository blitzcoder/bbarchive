How to get Puppy Linux 5.1.1 to work with BlitzMax

Hi there

This is just a note to you, if anyone who is having problems installing Blitzmax to use with Puppy Linux, it is very simple, After a day or researching, Puppy doesn't use apt-get, so the script wouldn't work.

Instead you need to download the following:
http://distro.ibiblio.org/pub/linux/distributions/puppylinux/puppy-5.1.1/lupu_devx_511.sfs

Copy this file over to /mnt/home

Now go to "Menu | System | Boot Manager Bootup", select the 1st option and adding the dev, reboot puppy and it will have all the development packages added in. Thats it

Of course, I'm using Puppy Linux on USB install so it saves all my session now.

Hope this helps someone

Also remember, if your sample are sluggish, then install the proper linux driver then it works flawlessly
