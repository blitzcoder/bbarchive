Mandriva 2009 is much better than 2008. Here's a quick howto (see here for an older one).

HOWTO: Mandriva One 2009

First, here is a general package list based on this one.

GNU C++ compiler
GNU Standard C++ Library
X11 client-side library
X11 XFree86 video mode extension library
Mesa OpenGL library
Mesa OpenGL utility library
X11 FreeType-based font drawing library
FreeType 2 font engine
X11 pixmap library

To install these packages with urpmi, enter 'su' and then copy to a Terminal.

urpmi gcc-c++ libx11_6-devel libmesaglu1-devel libxft-devel libxpm-devel

Package dependencies are shown below. Note you can use urpmq --fuzzy word to find a word in package names.
