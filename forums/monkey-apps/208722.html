mjcamerer(Posted 2014) [#7]
Thank you for the feedback navyRod, I will keep what you said in mind.

Now to answer the questions about frameworks / modules I used. For the physics I used muddy_shoes Box2d module. For the font I used Ziggy's FontMachine module. Finally, I am using DruggedBunny's Autofit module for scaling the game for all the different devices. Everything else was pure Monkey v78b, written in the default Ted IDE.

The game is not done. It just had to be submitted in time for the competition. I am working on some bug fixes and improvements and will hopefully have updates soon!
