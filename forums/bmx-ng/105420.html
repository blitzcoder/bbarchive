Brucey(Posted 2015) [#3]
This works for me :
SuperStrict

Framework brl.glmax2d
Import brl.standardio

Graphics 800, 600, 0

While Not KeyDown(key_escape)

	Cls

	If AppSuspended
		Print "Suspended!"
	End If	
	
	Flip

Wend

When the app is not active, the text is printed until the app is active again.
