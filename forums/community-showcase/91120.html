Well, here it is! ... Again!

For those who don't know, AnimImage will assemble frames into one image that later can be loaded with LoadAnimImage(). Pretty simple stuff.

Features:
* Specify the number of frames per row (0 = all one row)
* Loads BMP, JPG, PNG, and TGA
* Saves PNG
* Load single image
* Load entire folder

This is the beta release. Any bugs, suggestions, etc. that you have to report please post in this thread. I will fix any bugs ASAP, and I will try to implement any suggestions.
