MegaKerma Game Maker

MEGAKERMA Game Maker is my new and free 3D game creation system.
You can create puzzle solving, adventure, explorer style games or 3D presentations,
exhibitions or any other interactive applications.

MEGAKERMA is a complex system. In the Game Editor you can create your 3D world,
add textures, models and sounds. You can place static elements,
make them pushable or collectable, define dialogs, set up animations,
add your game logic via a very simple scripting language called KermaScript.

Thanks everyone :)

but why are you giving it away for free?

I started this tool as a small and basic game creation system with some nice but limited features. Previously I played with the Sherlock Holmes games and I thought it would be great to have a tool which you were able to create adventure or puzzle games with.

This tool still have limitations and there are more advanced game creation systems for free. I love programming and I would have created this even if nobody uses it. Also it is nice to have a small community tweaking my tools (www.silentworks.hu), so I decided to release it as a free tool. There is a registered version available, which allows you to create games for commercial purposes, if you really want to.

I also have the luxury to spent as much time with this as I want to, which would be difficult to achieve with a full commercial product. :)
