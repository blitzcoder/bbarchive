Action Adventure takes shape (images)

Sunteam Software(Posted 2003) [#1]
After spending a week visiting Paul our action adventure game is now starting to take shape. Here are some images to show progress:


This shows the new test level Paul created where we now have a proper scale to work to (in 3D terms). Also shown is the animated water effect and off to the right you can see a red arrow which is a working placeholder for level exits for which the code is now up and running to read the destination and automatically load up the next level.


Here you can see a place holder enemy which is fully animated and the player can now attack and kill them.


This shows off some of the new lightmapping we've managed to achieve with our test level.


Finally a quick show of the character interaction dialogue box in action. This is currently being tied in with our scripting system which will manage all character interaction and dialogue with full access to all aspects of the game, which basically means it will be capable of allowing full interaction so that NPC's can ask questions and react based on the response as well as giving/taking items, furthering the plotline, animate and move characters around the screen as well as play sound effects.

We've still got a ways to go yet but progress is good and with a working title ('White Morning'), and a fully worked out plot, this is shaping up quite well (we think).
