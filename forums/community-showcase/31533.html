Wandering Knights 0.3.2 has been released!




Wandering Knights is a hack'n'slash RPG granting unique gameplay that I have been doing over the time! It is getting really close to completion.

Features:
-Unique worlds, a new one every time you play
-The world is created as you play, based on random generation philosophy
-Huge world, 10,000 zones to explore on every game
-Skill-based system for character improvement
-This is a real RPG, you are free to do what YOU want
-NPCs interact with the player with group factions

March 11, 2004 *Beta 0.3.2 Release & Patch
-Fixed a color problem with the Logger
-There is now a NICE skybox, making graphics much prettier! The skybox includes moving clouds, a gyrotating sun (with an accompanied lens flare) and moon. Really a must see :)
-You can now set up the camera distance
-Added a sun hardware light that affects the environment
-There is now a prettier menu screen with light effects and a background
-The quest number is now displayed on the map and radar, so you can quickly see where to complete quests
-Added/changed code to be able to set the world size (normally 100, meaning a 100x100 zones world: 10,000 zones total), cleaned and optimized code for drawing maps
-NPCs now have a portrait in the target window instead of a special camera
-There is now sounds on actions upon quest obelisks (accepting and completing a quest)
-There is now less unique NPCs, they spawned far too frequently
-You can now run faster by pressing and holding the SHIFT keys
