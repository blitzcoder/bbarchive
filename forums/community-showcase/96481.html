I just released my screensaver framework for Unity, available for $49. The reason I'm mentioning it here is because most of the system is actually written in BlitzMax. Blitz handles the custom builder utility which embeds Unity data into a BlitzMax exe. The blitz exe handles stuff like launching the screensaver, interacting with the o/s to set wallpaper, install files, show splash screens and an EULA window, and to embed the Unity screen into the Windows screensaver dialog window. It's a pretty nice marriage between Unity and Blitz :-) Blitz provides the foundation and system interaction while Unity provides user input and special effects.

http://forum.unity3d.com/threads/81345-Unity-screensaver-framework-for-Windows-soon-to-be-released!?p=758796&viewfull=1#post758796

It comes with a couple of free screensavers, and you can try a demo here: http://forum.unity3d.com/threads/81345-Unity-screensaver-framework-for-Windows-soon-to-be-released!?p=680294&viewfull=1#post680294
