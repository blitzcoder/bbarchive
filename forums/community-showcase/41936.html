:o) Hey everyone. Some code to morph one mesh, into another. Both meshes have to be constructed the exact same. ie. they both must contain the same amount of vertices and created in the same order. Best way is to get your first mesh, save it, then deform it, then save the second mesh.

Left and right arrow keys to turn camera. Adjust morph_time to speed up/slow down the morph. I'll wrap it into some functions and stuff when i get time :o)
