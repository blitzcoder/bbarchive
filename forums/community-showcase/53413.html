Well as the title says, here is an .exe of a current in house development tool im working on for my racing game - "Last Lap". The long and short of it is;

Lets me define what type of terrain can be found on a 2d race track by placing polygons on the image (and verts).

Its no where near finished at the moment, but I only wished to show you lot what MaxGui can be used for, so have a play! :)

Download the win32 .zip here.

When it loads, you will need to load a graphical file, then the data file. When editing the data in the tool, general rule of thumb is; LMB = adds stuff, RMB = removes stuff, MMB = drags the screen, MouseWheel = Zooms!.

There are a number of bugs im aware of, but im always intrested in learning about more!.
