Boiled Sweets are proud announce the initial public release of their new screensaver!





Watchaball is a truely mesmerising and watchable 3D screen saver!

Watching Watchaball whilst it constantly spews balls made of various materials onto platforms, spring boards, rotating paddles and other animated objects never becomes tiring especially as they all behave with totally accurate physics.

Bouncy balls bounce, metal balls skid and fall with a thud, even tennis balls become fun to watch as they collide with each other, everything around them and sometimes even explode in a burst of smoke and flame!
Watchaball is highly configurable and can even run in interactive mode! What are you waiting for? Start watching balls with Watchaball...

Version 1.01 is out...

• When the ball type is set to random, generate mixed random balls unless the “No mixed random balls” option is ticked.
• The ‘crash test balls’ size was set to 0 so they were too small to see!
• The default value for the ball lifetime has been upped from 12 to 20 seconds.
• Tweaked the sizes and textures for a couple of the balls.
• The ‘drops’ theme balls were all the same size.
• Added display of selected theme/background to the F11 display
• Reduced the chances for the exploding and fire balls.
• Fireballs are now yellow again as they should be.
• Background image is now set to random by default.
