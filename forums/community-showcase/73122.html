Picklesworth(Posted 2007) [#20]
Looks good. The grass is flawed at the moment, though. As with LoD's grass, it is distributed in straight lines, so the pattern is completely obvious (thus, very ugly looking).

I suggest that you throw in some random numbers to mix it up a little, so no boring pattern can be seen in the distribution of grass sprites. Have each sprite offset within some limit (grid cell size), and maybe have the chance that a grass sprite does not appear at some places.

Finally, those grass sprites all seem to be the same colour, which again looks boring and out of place. It is possible to get the colour of the underlying texture at a particular point so that the sprite has the same colour as the grass texture it is meant to be an extension of.
