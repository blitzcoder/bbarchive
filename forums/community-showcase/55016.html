We represent your attention library SingleMesh AMT.
Library SingleMesh AMT is intended for acceleration of job Blitz3d of programs, at use of a plenty of objects.
This library creates virtual Entity in structure of one meshes from Entity - a sample.
The library allows to spend dynamic operations (positioning, moving, rotation) above virtual objects in realtime.
The library is based on technology of direct access to memory Blitz of the program, therefore is very fast

Now you can use in the projects of tens thousand objects.
The basic difference of this library from analogues, is an opportunity to move and rotate objects in real time without fatal falling FPS
