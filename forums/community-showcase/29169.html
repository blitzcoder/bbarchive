The new demo is available for download.



http://homepage.ntlworld.com/jm.keay/sare102.exe

WASD to move + mouse look
Space to jump
Left shift to crouch
Escape to access incomplete menu system =)
In menu use cursors/enter to select/change items.


Improvements made,

Jumping/Falling...

Much improved ( still not 100% happy with it though )

[Bug fix] Can now jump when in contact with a wall.

Jumping up into an obstruction now results in pain animation just like with the lift.

[Bug Fix] Jumping while on a lift stops the player moving/rotating with the lift.

Graphics Modes...

[Edit] Put 16 bit modes back in, demo requires either 16bit or 32bit graphics mode to run.
[Edit] Now only displays 4x3 ratio modes.

Miscellaneous...

Door Entity added!

Player height increased to improve sense of scale.

Camera cls no longer clears the colour buffer.
