Back from the dead, I've ported my 1st ever blitz3d game ( from back in 2001 ) to run under blitz 1.99. Strange how you pull some code out of the archives from years back and it just won't run under the latest compiler! Well this time it was due to terapacker and the way i handled some type lists, oh and for some reason the Animation Sequencing within X files was off, all my sequencing numbers had shifted UP one.. After a days work i got it working under the latest blitz3d build and I present it to you in it's original form ( again )

Some may remember this game from the competition blitz3d had in the early blitz years. Tranz3D and myself ( formerly SabatraRH ) put this together.
