Bukky(Posted 2006) [#7]
Hey Grey Alien, thanks for checking out our game!

We thought about the mouse issue, but we figured that it would be best if we had a consistent control scheme throughout the whole game.

There is a windowed mode which can be invoked if you press Alt+Enter. We should have made that clearer though.

Regarding the anti-aliasing...we had to cut out a lot of features to get the RAM consumption as low as possible. We'll try and fix that up in future revisions.

About the levels, it goes by point value. After you have a certain amount of points, the difficulty incements (i.e. the speed increases, you have to contend with more block pieces, etc.) I agree that a counter would have worked here as well.

Thanks again!
