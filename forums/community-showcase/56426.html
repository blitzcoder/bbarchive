Hi fellow blitzmax users...

I have reached version 0.001 of my first Blitzmax game. It is called "Su Doku Live". Right now only the Mac OS X build is ready, Windows and Linux will follow.

Why the heck create "yet another Su Doku computer implementation"?

- Because I was intrigued by the game (tried it recently for the first time) and did most of the code just to understand how to solve the puzzles. Looks like the game wrote itself... :-)

- I wanted a small project to take a break from a bigger one (a "Settlers of Catan" clone I have been coding for nearlly a year). Not releasing something was getting on my nerves...

- I liked Su Doku and I wanted to play on-line. Not against a clock (i.e. not really interesting), but against live players, in real-time, having some sort of hints regarding if I was in front or lagging... and I did not find any other game with that feature... so I found another reason to go ahead with it.

- I like solving hard/evil puzzles with my wife. In my game I can do it in Co-Op mode, I see her game, she sees mine, and we can share our efforts in order to solve these little nighmares in So Doku form...

Anyway, this pre-pre-pre-alpha release is on greedy.mrnet.pt/sudoku/distro/SuDokuLive.dmg for Mac Users to try out. It will be shareware eventually but right now it is fully functional, does not expire, does not nag anyone, so go get it and give it to as many mac users as you want.

Have fun and drop me a line if you liked it!

PLS

ps: uses MaxGUI and GNet :-)
