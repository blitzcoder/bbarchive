Here's a demo of Maptrix, a 2d tile layer based map editor with built in 2d lightmapper.

Click here to download.(2.9mb)





The full version can save projects(Including all tilesets) and export maps.
The exported maps can be loaded directly via a supplied 2d map engine.

The map engine/editor supports real time scaling.

Controls are,

Lmb paste active tool.
Rmb context menu to add lights and pivots. Right click where you want the light or pivot and click on the requried menu item.

Mouse wheel scales the map bigger or smaller.
Holdin the third mouse button and move to scroll the map or tileset viewer.
Everything else is gui driven.

Will be going on sale shortly for the low price of 15 Pounds.
No part of the editor or map engine requires the max gui to run.
