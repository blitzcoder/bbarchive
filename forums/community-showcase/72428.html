I had a similar problem where the vegetation didn't show up with an exported b3d scene. I posted in the gile[e]s forum about it and Fredborg said to change the Align property in the Instanciator Manager for each vegetation to "Vertical" instead of the default which was "Shader Controlled". This worked for me.
