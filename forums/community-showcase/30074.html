Texture Creator


This allows you to use small tiles to create larger textures. Small tiles without much detail I find much easier to draw than whole textures in one go. Using 32x32 tiles you can build up a texture using the 10 layers available.

Hitting the render button will then cast shadows on the layers below giving the texture more life. On top of this there is a fx stack that allows you to add, multiply, or 2x Multiply in any amount or quantity onto any over the layers below at any level. This gives you massive power to creating some very unusal effects as well as realisic looking dirt and grime.

Using the stack editor you can rearrange, change the magnitude, change the texture of each of these effects. Or indeed pile more effects on the top to give an even more rich texture.

Currently you can't save out the final textures (although you can do a print screen if you want!) and strength settings don't work on the multiply blend modes.

Let me know how you get on with it.

** Remember, this is a beta so it might blow up for no good reason at all, I've tried testing most things but you know how really obvious stuff falls through the net. If you can try and remember exactly where and why the error happened, let me know and I'll do my best to fix it!



Beta Download Here

Hints on Usage:
* Hide all the layers you're not using. It will render all visible layers regardless of if there is anything on them or not.
* Higher blur levels slow it down more so keep the blur setting low for getting it to look about right.
* Using the Right mouse button on buttons will auto repeat it (so you don't need to keep clicking)
* My textures are crap! I'd recomend building your own. Mask colour is 255,0,128, this will be definable at some point!
