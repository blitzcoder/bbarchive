This is an alpha release of our (me and Metalfusion) BlitzMax game Spacebuilders. You need to build a spaceship out of small blocks (the game provides a default ship, but that's not good). Press Space to pause the game and edit. Notice that the enemy ship will start to kill you immediately after starting the game. You need to bind keys for the thrusters and weapons manually. The physics are fairly realistic (with inertia, center of mass etc).

Make note of the power level, it should not be negative if you wan't your spaceship to function properly.

Keys:
Space - Pause
Mouse Wheel - Zoom
Left mouse click - Rotate parts
Delete is the default shoot button, but can you bind it.

Note that this is non-complete game, but the basic functions work okay.
Binaries are included for Linux and Windows and so are the sources. It's naturally in the public domain so you can do what you want with it.
Download (2 MiB): http://piracyfaq.com/files/spacebuilders.zip
