IPete2(Posted 2005) [#10]
Jeppe - I loved this.

You have obviously spent a lot of time and hard work on this game so far...it really shows, split screen , track editor, variety of tracks, weather effects...it has got fantastic potential.

The controls are my main source of concern, they really need altering to make the game much more fun to play. I think a lot of it has to do with the speed of the game itself.

The game does not need to be so fast, slow it down and give players the chance to play the game with the other competitors not racing off too quickly making it seem like an impossible task. Taking part is not fun at the moment.

Developing a game is an organic process, with iterations - each getting tighter, better and more enjoyable... this game is no different. Just now it needs more iterations, tweaks, additions, balancing and it will be GREAT!

You need to get the controls working far more in the player's favour... at the moment they feel out of control, which is unfair. Give players the control to keep their kart on the track and able to compete with the computer karts, as that is a major part of a racing game! At the minute it feels very unfair, players are fighting with the control system which is wrong. BUT there's nothing to prevent you from adjusting it to improve it enough to make the potential of this game emerge.

Introduce kart engine sounds, get some bouncy music in there and more sound effects, on the buttons and everywhere.

Amon and Smiff are both right - the potential for this game is there - it could be a FAB game if you can get the balance of the gameplay right.

I love the track designs and colours, I think the graphics style is fine, I just want the game to emerge from what looks like a project which has been a long time in the making.

KEEP IT UP Jeppe! Really great work so far!
