Code Counter version 0.9.8
Developed by Sean Karol in BlitzPlus 1.39
=====

------------
Description:
Code Counter is a small program used to count the amount of BB files in a folder, the lines, the size of all of the files and optionally you can choose to count the constants, globals and DEFINED local variables in the BB files. Another option is to include sub-directories of the main directory you chose.
DEFINED locals mean, only if the file has "Local" beside the variable, it is counted.

----
Use:
Simple run the file "Code Counter.exe", click "Browse" and select the Folder you want to count. Select any options you want. Click "Start" and it will generate a report that will be displayed very shortly.
