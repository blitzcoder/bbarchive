Hi all,

The Tattoo mesh skinning and Texturing tool is up to Version 1.15

This version adds several minor features:

Added the ability to cycle the background color (F2)
Added the ability to toggle brush outlines on and off (F3)
Added CPU Throttle when inactive (Thanks Fredborg)
Improved Seam detection to avoid lines jumping at seams
Added User Brush Folder. You may place any user defined brushed here for inclusion in the Tattoo brush list.
They need to 32 bit .PNG files in solid Black (color 0,0,0) with a variant Alpha Channel. Example file provided.
Added Visual Brush list.
Moved Lighting Panel to a separate window which can be activated from the View Menu
Numerous other little tweaks.

Check it out

TeraBit(Posted 2003) [#8]
Usually this means that the model does not have any UVs set up. (The pulsing is the brush outlines) Press F3 in the lastest version it should stop since this toggles on and off the outlines.

If you go to the view menu and select 'View Warnings about this Model' it should be able to tell you if the UVs are amiss.

Models need to have properly set up UVs prior to being loaded into Tattoo so that they can be painted.
