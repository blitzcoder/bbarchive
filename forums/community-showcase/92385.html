GaryV(Posted 2010) [#36]
[quote[Not quite, I believe it just needs to detect the word 'setup' in the filename, like 'BlitzGameSetup.exe'.[/quote]I was giving a possible reason as to why he named it what he did. ;)

If you are not using a manifest, to determine if it is a setup file UAC's heuristics look at the file name, the version fields and certain sequences of bytes within the exe. The latter two are only of a benefit if those are on file with MS to be recognized by the UAC.

So, yes "setup", "install" or "update" does need to be in the file name if you are not using a manifest to request the necessary privileges. Having your setup program digitally signed does not hurt either.

MSDN has detailed info on this.
