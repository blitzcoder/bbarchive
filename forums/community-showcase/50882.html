Boiled Sweets(Posted 2005) [#1]


Vorboils is an awesome 3D screensaver. It can generate endless, beautiful, random wormholes, is highly configurable and can generate random wallpapers for your desktop too!

You can also select an image to display on the spinning cube to advertise your business. You really have to see it in motion to fully appreciate the funky effects it produces.

You can also display an analogue clock and sync it and your PC to an online atomic clock!

To download a demo, view screen shots or purchase click here...

www.boiledsweets.com/Vorboils

Whats New in Version 2?

* Added wireframe mode!

* General bug fixes / code tidy

* Added extra stats / config display (added pages to stats display)

* Added pop up message to show you where it has created the screen shot

* Added option to allow Vorboils to start in 'locked' mode

* Added 'disable interactive mode' - this means that any key press will terminate the screen saver

* Added 'News' button in configuration. This automatically downloads news about new versions, WIP etc.

* Added check to ensure that INI must be found rather than just loading default values

* Increased the possible palette of colours for the checkered layer

* When forcing new texture with the "X" key ensure we have new scaling and motion too.

* Enabled the max textures option. This means that you can limit the number of texture layers on at the same time. So you can force Vorboils only ever to allow 3 texture layers on at the same time. This creates more diverse patterns!

* Modified the 'dots' texture layer to produce small crosses too

* Added option to allow unfilled rectangles for texture layers 1, 2 and 3

* Changed the tunnel segs option to allow a selection of segs, i.e. 32 AND 64, instead of 32 or 64. Added a 48 segs option too. This allows for more 'flexible' tunnels

* Added option to allow for 48 and 64 sided tunnels (enable only on top spec gfx cards)

* Added X and y axis oscillation intensity - this allows for more 'wriggly' tunnels!

* Added X and y axis oscillation intensity modifier - this allows for more 'wriggly' tunnels!

* Added option to allow flattened tunnels - and chance factor of flattened tunnels

* Added a 3D clock. This can be synchronized to the NIST Caesium clock and update your PC time. Choose from several styles, plus more! Send us your themes!

* Added a themes option, this allows you to save/load/download/share your favourite Vorboils patterns!

* Increased the max and default rotate speeds for nice falling down the tunnel effects

* Updated the 'thanks to' dialog to match the Vorboils theme

* Added preview of cube image in configuration

* Now use a XP style controls where possible

* Tweaked the Vorboils logo and added copyright information

Features:

* Option to generate and automatically update desktop wallpapers!

* Option to choose your own logo to display on the logo cube (supports masked images too)!

* Sync the clock and your PC time to an online atomic clock (accurate to 1 second in 6 millions years)!

* Loads more wacky clock faces to choose from!

* Ability to take screen shots!

* Save, load, download & share your favourite themes (tunnels)!

* Ability to run in interactive mode!

* Fully configurable - 194 configurable options!

* Show realtime statistics of all configuration parameters!

* Supports resolutions right up to 1792x1344 in glorious 32 bit colour with anti-aliasing!

* Install / uninstall support!

* Free updates for life!

* Lifetime technical support!

* Warm satisfaction that you are supporting an independent software developer!
