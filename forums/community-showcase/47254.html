Imagicon is a simple freeware graphics utility designed to convert multiple images from BMP, JPG or PNG image format into BMP, JPG, PNG or ICO image formats.

Windows Icons are created using 32-bit colour depth. Options are available to enable alpha transparency of icons and to define the mask colour used. (Only single image icons are currently supported)
