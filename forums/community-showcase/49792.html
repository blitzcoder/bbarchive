What is Vorboils?
Vorboils is an awesome 3D screensaver. It can generate endless, beautiful, random wormholes, is highly configurable (187 configurable options) and can generate random wallpapers for your desktop too! You can also select an image to display on the spinning cube to advertise your business. You really have to see it in motion to fully appreciate the funky effects it produces.
Not only that but it has a clock that syncs (and can update your PC time) to the NIST Caesium atomic clock which is accurate to 1 second in 6 million years!

What's New in version 2?

Added wireframe mode!

General bug fixes / code tidy

Added extra stats / config display (added pages to stats display)

Added pop up message to show you where it has created the screen shot

Added option to allow Vorboils to start in 'locked' mode

Added 'disable interactive mode' - this means that any key press will terminate the screen saver

Added 'News' button in configuration. This automatically downloads news about new versions, WIP etc.

Increased the possible palette of colours for the checkered layer

When forcing new texture with the "X" key ensure we have new scaling and motion too.

Enabled the max textures option. This means that you can limit the number of texture layers on at the same time. So you can force Vorboils only ever to allow 3 texture layers on at the same time. This creates more diverse patterns!

Modified the 'dots' texture layer to produce small crosses too

Changed the tunnel segs option to allow a selection of segs, i.e. 32 AND 64, instead of 32 or 64. Added a 48 segs option too. This allows for more 'flexible' tunnels

Added option to allow for 48 and 64 sided tunnels (enable only on top spec gfx cards)

Added X and y axis oscillation intensity - this allows for more 'wriggly' tunnels!

Added X and y axis oscillation intensity modifier - this allows for more 'wriggly' tunnels!

Added option to allow flattened tunnels - and chance factor of flattened tunnels

Added a 3D clock. This can be synchronized to the NIST Caesium clock and update your PC time. Choose from several styles, plus more!

Increased the max and default rotate speeds for nice falling down the tunnel effects

Updated the 'thanks to' dialog to match the Vorboils theme

Added preview of cube image in configuration

Now use a XP style controls where possible
