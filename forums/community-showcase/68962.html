This is a BETA of a project I want to complete. As I miss blitzmax wiki I decided to create something like that...
At the moment is a simple database of the official commands (only the one defined by rem..bbdoc..endrem).

What works:
- search engine
- you can find word with wildcards in description, mod, type, command
- comment system (with rating)
- tag

What doesn't work:
- authentication: all people (and bot I think...) can write/destroy message at the moment
- translation
- data export
