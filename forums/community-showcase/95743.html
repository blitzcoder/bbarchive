Pacfic Battles - TBS Game

Community Forums/Showcase/Pacfic Battles - TBS Game

Sung(Posted 2011) [#1]


Pacific Battles is a turn based strategy game für 1-2 players.
The current demo 2 is restricted to play only one scenario against the computer.

Every player can move one unit per turn (similar to chess). The unit can be selected with the
left mouse key. On the left side there are additional information about this unit (as mouse over
as well), which are shown in the following way:

CA 1/2 25

"CA" is the short name for a cruiser. The values "1/2" have the meaning that the unit can move
1 field maximum and has an attack range of 2 fields. "25" means the attackpoints the unit makes
under normal conditions. More on that later.

After the move the player can attack an enemy ship if there is one nearby. After that the computer
starts its move. The game is finished as soon as one side hasn't any units anymore.

Attack computation: Beside the attackpoints there are some more things which are calculated in.
There are critical hits (1% probability) as well as misses. Plus there are combinations where
the attacking ship makes only a damage of 50%. The following list shows the main targets where
a damage of 100% will be made (attacker -> targets):

CV -> DD, CA, CL
BB -> CV, BB, CA, CL
CA -> CL, BB, CV, DD
CL -> DD, PT
DD -> SS, PT
SS -> CV, BB, CA, CL
PT -> SS, CV
