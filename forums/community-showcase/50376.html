Hello all,
I'd like to invite you to help us test Control Monger which is due to be released in roughly three weeks.


Background:
Control Monger is a 100% free online multiplayer FPS (first person shooter) where you take control of a bot and fight in teams to win a match.
Each bot has different weapons and its own special abilities. For example the Construction bot has a machine gun, side arm and laser and has the ability to build or repair objects on the map (doors, bridges, etc) as well as place auto firing turrets.
There are seven types of bot and two main game types (Death Match and Capture the Flag). We also have more game types coming soon such as Objective and King of the Hill. Each map type also supports both team and free for all modes.

Games can have (in theory) up to 255 players (although graphics will get way too laggy with current CPU/graphics cards - however in years to come it may be possible :))
We will also be allowing players to easily create their own maps and will be creating some tutorials shortly.

More info can be read on the official site: www.controlmonger.com and on the forums at forum.freeworldonline.com

Blitz related info:
The game was written entierly in BlitzBasic3D. Dedicated server runs on the same code but is optimized by not rendering any graphics and cutting out any visual only code (such as particle effects).
We made use of GLET for the map editing and Particle Candy for the particle effects (although both were modified a bit to get exactly what we wanted). Otherwise all the code (with the exception of a couple of userlibs such as stickkeys, blitzclose and blitzsys) was written from scratch.
All the media (maps, bots, sounds, etc) were made especially for this game.
