My little game Sheeplings is up to v0.5, and this is also a release candidate. Please feel free to give it a go!

Description
Wolves, eagles, bandits - everyone is after your cute little friends! Protect and guide the sheep through more than 80 wool-raising levels in this thrilling arcade game, until you finally reach your goal: the sheep-jumping competition in Woollyville.

Main features
* Adoringly cute sheep
* 84 wool-raising levels
* Three quests
* Non-linear gameplay
* Mouse, keyboard or joypad control

System requirements
??? MHz, 256 Mb RAM, DirectX 7 or later, and 16-bit or 32-bit color mode. Tested on Win2k, XP and Vista.

Please note
* I'm afraid that I'm fresh out of older computers, so I'd much appreciate comments in case you have a <= 1 GHz computer to try this on. Also, BlitzMax is supposed to run on Win98 but the game didn't run on a VMWare installation I did; I have no idea if it works on a real Win98 comp.
* I'm aware that some production values are less than stellar. I hope that the game is decent (and fun) enough despite that.
* For testing purposes there are some cheat codes that you can enter on the title screen:

"endgame" - Displays the endgame credit sequence.
"setec astronomy" - Unlocks all locked stages.
"warp speed" - Fills up the dog's speed gauge.
"bring the noise" - Likewise for the dog's range.
"dirty cheater" - Grants 50 tokens (also counts toward the total number of collected tokens).

Any feedback is much appreciated!
