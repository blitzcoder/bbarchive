It's funny to play. Although a simple if not the most simple game idea ever (excluding maybe Guessmaster 2000), I did some similar games too in the past, even Ads Games for respectable phone companies. Regardless of this Simplicity, it can be really addictive. Some of the graphics need some polish (eg. the mart logo the middle, with the photoshop canvas texture...). I guess this is some kind of theme game, you can't be too serious about it, are you? The key controls make it rather hard until one gets familar with it. Maybe rotation should me smoothed too, like the motion speed. Contrary to the name that associates something dark, sinister, the game is featuring a light and funny ambience. FX are done well. I like the shopping mart background sound and the newspaper made me laugh. :)


Yahfree(Posted 2010) [#3]
Hey thanks for the review JFK. I was aiming for simple gameplay.

The graphics were done with haste, I was on a deadline. I'm not too serious about the game itself. I just wanted to write another 'complete' game, not end a half done project.

I found the keys quite comfortable. I used this game to showcase a simple particle engine I wrote. Hence the effects everywhere.

I wanted the user to take something away from it too; so I found facts about black friday, and added some historical humor in there.
