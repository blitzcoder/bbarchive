Hi,

I'm pleased to announce the release of the next version of my psychedelic screen saver - V O R B O I L S.

Vorboils generates endless beautiful wormholes and can create eye popping wallpaper!

It's still in development but is almost finished, just a few more configuration options to get working.

Go grab a copy here www.boiledsweets.com/vorboils

What's new?
* Improved installer
* Fixed couple of windows bugs in vb
* Added background text colour option
* Added display background text colour option
* SUPER improved the texture generation/motion!!!
* The general and camera config options now work!
* Added option to disable fog
* Added "Vorboils" logo stamp to screenshot/wallpaper images
* Reduced the bb dat file by 75% approx
* Now can create automatic wallpapers!

* Added option to display FPS
* Added cube config options
* Now pass the desktop res depth to bb code -- n need for blitzsys.dll file now
* Added extra texture options (WHITE) for texture 4 (will expand on this for some nice effects)
* Moved FPS to separate display to try and get more accurate FPS stats

* LIGHTS options now working
* HELP tab improved
* Added option To prevent the cube from being drawn on the wallpaper/screenshot image files.
* Fixed bug whereby cube was ALWAYS visible!
* Fixed bug whereby if the cube image filename was actually just a directory then we hit a problem
* Fixed bug whereby if you took screenshot when the cube was disabled you got crash
* Added code to handle it if there is no ini file
* Added a basic texture twist function

Now we have 86 configurable options!!!
