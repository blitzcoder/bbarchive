PyroParticleSystem is a single surface particle system for blitz3d.

Screenshots, click to enlarge:

[img
http://www.gameart.dk/pps/Gfx/Screen2_s.png]
The full library features:
- Single surface for optimal speed.
- Create templates, to define particle effects.
- Split templates into different parts, for easy effects creation.
- Create effects using logical modifiers.
- Many logical modifiers available
- Update the library with just one command
- Treat emitters just like blitz entities
- Create any effect imaginable
- Simple and logical commands
- Easy to integrate into your game
- More
Visit the PyroParticleSystem website, for more information:
www.gameart.dk/pps
