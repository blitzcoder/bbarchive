Hi folks,

I've just released (should be available in several hours) on the Google Play store an app that is useful for pen and paper rpg campaigns.

It is a world campaign mapper.

It generates a random world map, populates it with cities, villages, keeps and dungeons with a random description for each one and allows the user to annotate these locations themselves if they wish and save the map.

It allows for update from the web with extra information about locations, characters, items and the like.

It was put together fairly quickly - over about 3 days (started work on it about Friday after I wrote my map generation algorithm which I shared on the code archives).

It's mainly for use with pen and paper role playing campaigns.
