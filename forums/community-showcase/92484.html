Howdy folks!

Here's a spiffy little game I started this month, with the help of my new game
design partner, Ash. He's done most of the graphics, which are, I'm happy to say,
way better than anything I've ever done ;)
This will be WerdnaWorldGames second commercial project, so please be
as scathing as possible with the critique.

However keep in mind this is an extremely 'alpha' version of the game. I don't
even have a name for it yet!

There are a lot of place holders present, but to my knowledge this demo is glitch
free.





It includes 4 different levels, one for each race. Each level is complete with
it's own unique boss, and there are 2 kinds of weapon for the player to use.

Use the Arrow Keys to move, and Spacebar to Shoot(You can change these
in the options menu). Also you can press 'Q' to display armor and FPS count,
and 'T' or 'Y' to speed up scrolling if needed.

Have fun!
All suggestions, input, feedback, and criticisms are welcome.
