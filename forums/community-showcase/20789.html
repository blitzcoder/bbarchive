Well, it is so far that I brought my first Blitz game, SokoBlobz, to it's completion ! :-)

It is a animated SokoBan clone, and even if my aim with this first production was rather to discover Blitz and its main function I find it amusing also.

The game comes with 937 integrated levels, built by well-known Sokoban levels authors, wich I thank here again for this (I thank them again in the ReadMe file also, and in this file I thank you all also here for your help and patience with me ;-)

I have no website right now, so I put the game in the showcase.

Thanks a lot in advance for all of you that will take the time to DL it and give it a try (the puzzles are pretty hard I think).

And thanks even more for the people that will give me any feedback (corrections to be made, bugs reports, suggestions,... ALL your comments are welcome ! :-)
