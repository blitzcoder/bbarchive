It's very beta as you can imagine reading here:
www.litobyte.it/madrace.htm

Download it from that page. It's quite old as you can see from the date of the files.

Is compiled with blitz3D compiler v.1.83

It uses ODE.DLL and wrapper by P.Schutz
It uses wiebo de wit "mini" particle system
It uses Phosphore compiled city blocks level meshes
It uses 256 texture flags "in-house method" to realize dynamic generated environmental mapping(yes 2 renderworld each pass) Unless you disable it pressing F3.
It uses BMW Z4 model and some textures courtesy of Puzzle LTD.


All the code is not more usable under blitz v.187 for the way how the .b3d level meshes were saved(for 1.83).

I will update it hopefully for a tokamakized version compiled with 1.87 and with sounds, ragdoll and all back in.

If anybody has got a better model for the player, should be a .b3d boned and animated model, with 1000-2000 poly versions (if possible for Fake-Dynamic-LOD).

It's quite a bit I don't come on these forums, so I'm not aware if there is SOME plug-in which exports characters with shared weights (now 1.87 supports it), but it would be very nice if the model had this peculiarity.

Thanks for any feedback
