I have included , well I wont call it a demo because its far from complete, it's the first 8 or 9 levels, but I have a game play issue that I would like some feedback on before I continue work on the project.

In a nut shell, any triangle that can see another facing back toward it will die, any same colour tiles in between theme will also die, but if another color blocks these two facing triangles from facing each other then no line will be made.

Keys
Arrow Keys :- Use the arrow keys to move your selection tool around the map.
Space :- Hitting space will pick up or let go a tile while holding a tile you can move it around.
L :- will reload the level if you fail.

My Issue : - I'v had this concept in my head for a while , but I am finding it far to easy to play, I'm also finding it hard to come up with puzzles to challenge the player, so what I need from any of you that try it is what you think of the first few levels, and if you have any suggestions on what I could do to add that challenge factor that I feel its missing.

I should say that there are a few other tile types not on show which might ramp up how the game plays but iv not coded them in yet so :/

Anyway thanks for checking it out.
