After 3 months of hard work, Cod2Doc is near completion. To get more details on Cod2Doc, please go on my worklog. In short, Cod2Doc takes your .bb source files and creates an html documentation (but also it can create help files, web pages).

To see what is able to do Cod2Doc, I have created two example files. You will be able to see the source file from which was created the documentation as well as the resulting html documenation.

- BlitzBasic style
- Cod2Doc style

There is still some minor layout bugs that I have to correct, but the general idea is there!
