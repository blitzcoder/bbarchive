A game that i am currently working, still in alpha development, but playable.

Its a 2D platformer based around one huge map - no more then one map.

The objective is to get to the end, which i have yet to make, still alive.

Controls:
- 'X' action button
- 'Z' Fire
- Arrow keys to move + jump
- space to fly, when you have the jet pack

This game was just going to be a test for HighGUI so if you press '9' you will be able to edit the game.

if you use the editor the controls are:
- mouse scroll to change lift hights.
- left click to place an object - right click to delete it.
- 'o' & 'p' to change zombie styles.
