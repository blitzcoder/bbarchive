A big update this time folks - and I am one revision away from a full blown public release

There is still some stuff in the the config to implement but if you take a look you can see what sort of stuff will be configurable.

Lots has changed but the biggies are:
* Save the mouse position prior to running the screensaver (in full mode or preview mode) and restore it's position afterwards
* Added option to reset all configuration options or just the current tab page's options
* Tunnel options now functional
* Disable the unavailable options when in DEMO mode
* Added basic validation to ensure that daft config options cannot be set
* Added nice icons to each tab page
* Now uses custom rounded frames instead of boring square ones

* Auto generate wallpaper is now off by default
* Added 5 second delay when exiting demo version nag screen!

* Corrected the max values for the camera yaw max and change chance
* Show Background option was not being read in from INI file
* The 1280 * 1024 * 32 video resolution option was read in from the INI file correctly

* Now seed the camera with random values at startup
* Fixed horrid clipping problem that occurred when the tunnel wall got too near to the camera
* New super duper 'ripple' effect!

Now have 101 configurable options!!!

Please try the F1 key for help.

Please let me know how you get on.

THINGS STILL TO DO...
* Allow 6 sided cube
* Get texture options working
* Tidy stats/config display

Please keep those ideas coming!!!
