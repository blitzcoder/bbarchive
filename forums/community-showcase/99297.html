Htbaa(Posted 2012) [#1]
Maximus is a Module Manager for BlitzMax. With Maximus it's possible to easily install and update BlitzMax modules which are provided by 3rd party module developers.


Today I released Maximus 1.1.2 which fixes a Memory Access Violation bug which would occur when installing a module that didn't have a dev-release.

You can get the updated version from http://maximus.htbaa.com/client . For Windows there's a ZIP-file and an installer. There's also a targz for Linux.
