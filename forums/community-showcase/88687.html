Hi all,
your comments are very appreciate because at the moment I feel a little sad.


The game is considered the most awful match3 game ever due its strange approach to the match 3.

When we started developing this game we consider that all the match 3 games were clone each other. So we decided to create something different, where the focus was the playability and not the extern part of the game (menus, story, fairys and special effects).
So we realized a game different from any other based on speed, rithm and strategy.
The game has a lot of features that can't be evidenced directly...
- For example, maintaining a costant rithm the game awards you with extra scores and multiplier.
- The gameplay is continued and not interrupted like any match3 where you have to wait that all the pieces are placed..here you can continue in matching also when pieces are falling
- there is a strategy for each round to avoid unuseful match not required and to finish before the time is over and to avoid the drunking time.
- the music is synchronized with the rithm of matches you need to mantain
ect..etc..
- each tile is a single object that has its sound, its grafix, animation and can be scaled in any dimension with a pixel perfect 3d rountine..so is possible to have from 6x6 to 25x25 tiles on the stage!

All That is exactly what all the other match3 games avoid.
Match3 is considered only a relaxing game.

Cocktail mania is an hard core match3 for its mature contents and playability.
