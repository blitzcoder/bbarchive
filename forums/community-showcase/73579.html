What is qoob?

qoob is a game of skill, cunning and strategy!

You are trapped in the qoob hexiverse. The only way to escape is to decrypt the 36 bit qoob hexiverse code. To decrypt the hexiverse code you have to visit every qoob world and obtain it's 1 bit world code. The world code can be obtained by locating the code key and code terminal.

Sounds easy but you are up against the clock and there are many puzzles, tricks and traps to prevent your success.
