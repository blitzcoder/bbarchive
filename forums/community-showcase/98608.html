This is actually a continuation of my original Tevada Trigger thread but since Taron has done an immense amount of work on this that I feel that he should get his own thread since it almost feels like a new game altogether :).

I present to everyone Tevada Trigger Taron Edition, an incredible mod of the original game that features : updated graphic renderings, tweaked gameplay, sounds, and of course new the devastating new Ring Turret enemies!







Controls :
AWSD - Left, Up, Down, Right
Left Mouse Button - Shoot!
Right Mouse Button - Switch Bullet Color Mode between Gold and Silver
Space Bar - Bomb!

Download :

Happy shooting, and good luck :)!
