Not sure if I can even post it here (it might be frowned on) but I made a GUI for get_iplayer - get_iplayer is a tool to download bbc iplayer shows resulting in a .mp4 file.

**** I DID NOT CREATE GET_IPLAYER ****

more details here on get_iplayer http://www.infradead.org/get_iplayer/html/get_iplayer.html

Currently it only has a command line tool so I made a GUI to make it simples.

Download it here (inc source bmax): http://www.getperplexed.com/eye_player/eye_player.zip

Instructions:

1. Install get_iplayer - direct link: http://www.infradead.org/get_iplayer_win/get_iplayer_setup_latest.exe (windows)
2. run eye_player
3. choose where to save your downloads and where you have installed get_iplayer
3. navigate to the show you want to download using the inbuilt browser
4. press download and convert
5. cmd window will popup showing the download progress - let it finish
6. folder will popup showing your DRM free .mp4 video when done
7. Enjoy

Tested with windows 7 x64 and winXP.
