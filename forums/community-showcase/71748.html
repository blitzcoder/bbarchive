- Note :

"Brium XG-72" is a game with 50% of action and 50% of logic. Indeed, you must recover the weapons in a precise order to be able to cross the level and to destroy the enemies.

Even with the mini-map, you'll have to memorize each sector to avoid the traps and other dangers!

============== FRENCH ===============
- Jouabilité :

Afin de ne pas vous faire repérer trop facilement par les engins mécaniques de la météorite, vous larguez votre matériel avant de débarquer dans un secteur(arme, piles d'energie, mega-bombes).

Pour détruire un ennemi, vous devez utiliser le même code couleur que son arme. Par exemple : un ennemi bleu ou qui tire bleu est sensible uniquement à un tir bleu. Un ennemi orange ou qui tire orange est sensible à tous les tirs.

Code couleur de vos armes : arme 1 (tir simple) : bleu, arme 2 (tir double) : bleu, arme 3 (tir multiple) : rouge, arme 4 (tir laser) : vert, arme 5 (lance mine) : gris et arme 6 (lance missile) : gris.

Vous commencez le niveau avec 2 mega-bombes (explosions electro-magnétiques). Lorsque vous les activez, elles détruisent tous les ennemis et tous les tirs ennemis à l'écran dont vous possédez le code couleur.

Vous avez une barre de charge pour votre arme. Chaque tir consomme de son énergie, mais elle se régénère automatiquement. Vous devez attendre que la barre de charge soit complète pour pouvoir tirer.

Vous avez 3 vies pour terminer un secteur. Lorsque vous perdez une vie, vous recommencez au dernier point de sauvegarde. Lorsque vous n'avez plus de vies, vous devez recommencer entièrement le secteur.

Les armes et bonus : en cherchant bien, vous trouvez des armes, des piles d'énergie et des bombes pour progresser dans les secteurs. Ils sont indispensables pour mener à bien vos missions !

Le contact avec l'eau ralenti vos déplacements, le contact avec l'acide vous ralenti et vous fait perdre de l'énergie, le contact avec les pics vous font perdre toute votre énergie.

Votre androïde de navigation, le TF800, vous contacte régulièrement pour vous indiquer la marche à suivre ou les pièges éventuels.
============== FRENCH ===============
