New Windows and Mac OS builds available! I'd be very interested in other BM user's thoughts on these. This is actually my first BlitzMax game - I have been using BM for years for application and tool development and only recently started to take a stab at making a game.




Mac users also get a bonus in the form of a early build of the Davey Edit application. This allows easy editing of screens. Beware - it's even more buggy than the game is at this point. ;-)

The new menu system in the game has commands for Saving and Loading but these currently do not work - next big thing to do.

Both of these are installer packages which will create a Davey folder with the application inside of it:

Windows: http://metasushi.com/software/releases/Davey5.sfx.exe
On Windows it will create a Davey menu group and menu item for you to launch the application easily.

Mac: http://metasushi.com/software/releases/Davey5.mac.zip
On Mac, unpack the ZIP file then run the PKG file within it. This will install the game to Applications/Davey.
