Push'Em is an Original (it realy is) game, which combines Sokoban and Lines with platforms.

The aim of the game is to remove all the Coloured blocks from each Puzzle, you do this by combining like coloured blocks togther in groups of 3 or more.

The rules are simple you can only push blocks and not pull or carry them, you cannot push more than 1 block at a time or push a block passed another block.

There are also teleports that can be used to move blocks around the level.

This is a beta, and still needs some minor tweaks. I Originaly coded this in BPlus but it didnt look very colourful so here it is in a much more colourful face.

This game will be Shareware but you can get it for FREE!. Simply use the Editor to create at least THREE challenging puzzles, if they are any good and make it into the game then when the game is ready for the public you get a FREE! registration code to unlock the full version. Email your Puzzles found in the Editor directory to jabawokie@..., dont forget your name and a return email.


I hope you enjoy this game and any feedbeek would be welcome.
