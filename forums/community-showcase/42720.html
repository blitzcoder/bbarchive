Hello,

yesterday i found some time to do a little tool (streifen .1) for win, with which you can create animationstripes out of several frames.

Glueit didn't do the job for me as it wasn't able to load and save PNGs with the alpha-channel - as exported by 3dmax for instance.

Beware this is NOT an Editing/Animation-Tool. It's just storing pictures in one file. The filesize is somekind of big because it's done with MM-Director.

specs:

- imported formats: png(*),jpg,bmp,tga,swf
- exported formats: png(*),jpg,bmp(*)
- maximum filesize: 4095^2 pixels

(*) alpha is supported


zip contains:

- info.txt
- streifen.exe

streifen_win.zip


Greetings,

taumel
