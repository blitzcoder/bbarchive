jfk EO-11110(Posted 2013) [#5]
exports avi, using your installed codecs.

note: issues with win8: if you replace ffmpeg.exe with the latest version, as suggested on the homepage, there may be a privilegs conflict. Solution: copy smk folder to desktop and run from there.

hint: using "mpeg4" for import (both Hd and mini version) is lightning fast on average 64 bit dual core cpus under win8, using hardware encoding. like, 80 fps conversion speed...

Note, smk is a puristic tool, although allowing detailed editing, not very speedy for mass production. Nice for some 3 Minutes clips, but for editing a 90 Minutes Movie better use something pro, like premiere, vegas, final cut, avid...
