I've decided to release my project, Chemical Imbalance, as a free game to the Blitz community. I've spent close to a year developing this game in my spare time. This is the 1st project I was able to complete and it was a great learning experience. However, I don't feel it is up to the level of a commercial release.

Chemical imbalance is a puzzle game made in Blitz3d using the Sprite Candy library.



It is currently on a free host so I don't know how long it is going to stay available. If anyone is willing to host it, please feel free to grab the file and post a link in the thread.

Thanks to everyone who helped me out with advice, and/or testing the game.

Instructions on playing the game can be found here: http://fetcher.sitesled.com

The game can be downloaded here (for the time being):
http://upload2.net/page/download/8WARQqIdgkeekQR/Setup.exe.html

Have a great New Year.
