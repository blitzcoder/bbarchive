This is a MP3/OGG/WAV media player coded in Blitz Plus. It's in Beta stage right now but it can do FX like ECHO, CHORUS and REVERB. IT also has an EQ (real-time) and real-time bar graph.

A note about the audio FX. They take hold on the next song played. So if you set for example "ECHO", it will effect the next song (and any after) you play. The next version will have this fixed so the FX take hold right away.

I have Chours set to phase between left and right channels. Sort of an AM radio'ish kind of thing. The next version will allow you to set-up your own FX :)
