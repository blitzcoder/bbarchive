Hey, I posted about the signup period for FreeWorld Online last time and got quite a few people from here, so I thought I would this time as well :)

Anyway, basically FreeWorld is an online role-playing game that has been made using blitz. Everything from the server to the game to the patcher (prog that makes sure everyone has the same version of the game) was made using BB3D.

Some fairly recent screenshots can be found at www.freeworldonline.com/sshots.php
Here is an example:


Basic info can be found at www.freeworldonline.com/about.php

And of course if you want to signup to help with the testing go to www.freeworldonline.com/signup.php

Some of the additions since last time:
+Recreated main playable model replacing the old temp one :)
+1 new playable race + female versions of each race added
+Added tiredness
+Added team play
+Added mini games - poker (can be played with other people) & hide 'n' seek
+Improved GUI
+World has almost doubled in size
+Now have over 180 items and equipment
+Over 40 NPCs (non-playable characters)/monsters
+Guard NPCs - protect towns from monsters
+Numerous improvements to media
+Player area added to website for account management/seeing if friends are online.

Full patch-news since the start of the phase can be seen at www.freeworldonline.com/temp/phase4.txt

We will be accepting signups until the 31st of May, and we will start the phase on the 2nd of June
