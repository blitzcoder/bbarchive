After a month of hard work, the version 1.1 of BSP Factory is finally finished. It will be sent to all the existing customers soon by the publisher.

The new features are:
- Windows XP style interface!
- Added WSAD keys movement (also the cursor keys).
- Mouse button 3 now moves the camera on the X and Y axis.
- Mouse wheel now move the camera on the Z axis.
- Shift key to speed up camera movement.
- You can add and delete entities from the program now.
- Set the distance of entity names appearing.
- A new entity defaults to camera position.
- Change the name of an entity.
- Ability to change the properties of each entity (position, light colors, etc).
- Entity names float on the scene, so you can see the name of each entity in real time, and light sprites have the color of the light.
- Sprites for entities instead of red boxes and yellow spheres
- Fixed one bug accidentally introduced in 1.1 BETA-A regarding B3D exporting.
- Much faster loading of maps !
- BSP loader and B3D exporter completely rewritten.
- Imports B3D files !!!
- Support for BMP, JPG, PNG, PCX, TGA and TIF textures.
- Lightmapper updated to last Marcelo's YAL version.
- Masked surfaces now receive shadows.
- An .ini file added to change program settings.
- Generates a log file with all the operations performed.
- Enhanced command line arguments and BatchCompiler program.
- New texture flags:
'|' - Fullbright.
'%' - Specular highlights.
'#' - Cube mapping.
