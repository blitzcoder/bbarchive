I’ve been obsessed with creating a dynamic, evolving environment lately. One part of it includes an evolving sky, day/night cycles etc. I dug up an old project to reference for the new work I want to do using BMax and Leadwerks. I thought I would share it, see if anyone could improve it or use it. It’s sort of slow. This just does clouds that sort of evolve over time. The evolution is exaggerated to demo the effect. This basically is a fusion of code from the archives and forums. I did a poor job of acknowledging the authors of everything, for that I am sorry in advance. I can’t remember who did the skybox code that is in there, maybe someone recognizes it. Also, please excuse the poor coding and organization.

Controls:

Mouse and arrow keys, rotate etc
INSERT adds clouds
DELETE removes clouds
T to use skydome
R to use skybox

Add some clouds and watch them for a minute. You should see some fade away and some fade in etc.

Download:
info.friendlysoftware.com/~wallace/clouds.zip
