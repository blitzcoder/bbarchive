A small update for my AlphaGUI library:

- Now includes the source code for the editor, as some have asked me to make available.

- Corrected some small bugs in the editor, now you don't have to hit return to update gadget´s ids and texts.

- Added a Uselibs file, place this in your Blitz3D\userlibs\ folder, to make the commands/functions highlight in the editor.

- Now you will also be able to use the double click f1 help for AlphaGui, by copying some help files to the blitz3d commands folder.

Feel free to modify it anyway you like, as long as you give me credit :-)
