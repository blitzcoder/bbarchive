Yesterday I was finally able to release my first BlitzMax game. A wrestling game for Mac! Nope, you didn't read that wrong. There is finally a wrestling game for OS X. This action packed anime wrestling game comes full of different match types. From the standard one on one match to the deadly Deathmatch where the ring gets loaded full of weapons. Not only is it my first BlitzMax project it is my first game to be available on the Mac App Store!

So I want to thank everyone involved with BlitzMax and the community. Also a big thank you goes out to Mat Dickie & Erik Hogan for all of their help.

The game features
18 Wrestlers to choose from
10 Different match types
48 Round Tournament
Final boss battle on the 48th round
Actual Refs to referee the match
Unlockable: Hilarious Bighead Mode
Unlockable: Editor Mode - Create up to 95 wrestlers!

An update on the Powerslam to Monkey process:
Erik is working hard on a project called Monken, a Semi Automatic BlitzMax to Monkey converter.

This is very exciting stuff, because it doesn't just mean a Monkey port of Powerslam but if it all goes well it means anyone can make porting their BlitzMax code to Monkey a whole lot easier.
