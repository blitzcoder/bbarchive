semar(Posted 2004) [#17]
A note for the ones having this timing problem.

For testing purpose, I linked the spacebar key to a function which reduces the available round time to 3 seconds. Sadly, I've forgotten to remove this link.

That means that if you redefine your keys, and choose the spacebar as the fire key, it will not only place a bomb, but also make a draw game after 3 seconds - that is, both the players will explode.

So, check if you have redefined your fire key with the spacebar, and if so, please choose another key for fire.

I'll post a new version soon which will have this glitch fixed. In the meantime, check if redefining the fire key to any other key but the spacebar fix the problem.

Many thanks to James Frost for spotting this out.

:)

Sergio.
