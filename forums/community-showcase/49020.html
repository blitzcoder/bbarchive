perseus(Posted 2005) [#29]
once again, thanks for all comments..

@ Pertutbatio:

I've made 3d glasses feature for a red-blue one. it will probably not work with different color and polarized glasses. if you have red-blue 3d-glasses and still see colors bad, following method may help about it:

1. press F
2. press 2 on numeric keyboard
3. press and hold down 4,5,6 in numeric keyboard for a while (until everything gets dark)
4. press 7,8,9 on again numeric keyboard to adjust color

(this may be used to adjust contrast when all mirrors are enabled too..)

@ {cYanide}
yes, I like that game too.

@ Cygnus:

Many things are used in combination: cube mapping / spherical mapping / feedback effetcs / etc..

btw. you may use (-) key on numeric keyboard to slow down camera.


@ jfk EO-11110

I've done reflection by calculating cube map and assigning the result to top texture layer of objects. However, an additional small trick is needed: only shiny parts should be rendered while generating cubemap.
