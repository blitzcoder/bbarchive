And some background info...

Currently the game runs with a full weather system, game server design is based on a distributed system so we can host across multiple servers with a central character/validation server, coupled with remote weather and NPC control servers for each zone or multiple zones, i.e. it's fully extendable. This is all in place already, it's only now that we are beginning work on the client. These screenshots are early tastes for you so you can see what kind of flavour we're going for. It's still early days and we are using a slow connection for the test server to ensure we code everything as tightly as possible. If nothing else comes from it we'll have a great game engine that we can use in future projects. That said it's all good fun anyway isn't it?
