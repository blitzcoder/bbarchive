Hi Sergio, cheers for the feedback, ok, here goes :)

1) While using the crop tool, and pressing the 'delete' key on the keyboard, the app bombed me out,

Strange one, I'll have a play with that tool and see if I can hunt down the possible clash.

2) would be really nice if you can add another behaviour while using the brush. Right now, the brush is always drawn with its background,

Ok, generally speaking, what ever the colour is held by the Right mouse button, that will be used for transparency. To change this after cutting a brush or loading one, Select "Brush > Brush Manipulation", when the window pops up you will see various brush options, one will be change trasparency, click that button and which ever colour is held by the right mouse button will become the new transparency. Also, click "Dock", many people seem to like all windows Tabbed.

3) You use CTRL-Z to undo. This is valid on a UK keyboard.

Sure thing, I'll look at a better method for this.

4) When I select a brush mode, I have no way to know which one is selected,

Will do, I will add it in brackets after the file name in the title bar, or, in the status bar.

5) The filled rectangle tool does not work like the normal rectangle tool...

Hmm, seems to work fine here, only the filled square and square tool behave as explained. Ensure you have Filled Rect selected. Currently I'm unable to repeate this.

6) ProPixel seems to work only on two images at the same time. Is this only a demo limitation ?

This is currently a version restriction, as ProPixel2D is the low end product in a range that is being developed over the next year or so, ProPixel Animator and ProPixel Studio will be far more flexible. PP2D is aimed to be simple to use, though, don't rule out full multi-image loading in future updates as it's bigger brothers will have this.

7) Would be nice to select a brush from a list of brush images

As mentioned before, this is only a version restriction, and, as with full multi-image support don't rule out PP2D eventually having multi-brush support.

I hope this answers your questions, and I'm pleased you like it so far. :)
