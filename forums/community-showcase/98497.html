Kryzon(Posted 2012) [#6]
Use 7-Zip.

Create a ZIP of your game installation folder so that people can extract wherever they want. You can make it a Self-Extracting archive if you want the automation.

This is a minimalistic approach [no registry tampering etc.] that you should prefer; it lets people install and delete your game with absolutely no residue left (something that cannot be said of all install shells).
