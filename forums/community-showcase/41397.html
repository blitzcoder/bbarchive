marksibly(Posted 2004) [#6]
Matt,

You can probably streamline the release quite a bit.

Have a look at the firepaint sample. It contains something like:

Framework BRL.GLMax2D

Import BRL.Basic
Import BRL.System
Import BRL.FreeAudioAudio
Import BRL.PNGLoader 
Import BRL.WAVLoader


By specifying a 'framework', you get to pick and choose what actually gets included in the release - eg: the above only imports the PNG and WAV loaders since that's all firepaint uses.
