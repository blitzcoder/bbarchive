Hello all,

I'm still working on this (very simple) game, and here is another piece to play with.
Although I've spent a lot of time with it, there is still a very very long way to go.




and the download is at:

http://home.people.net.au/~Matthew.Lloyd/lloyd/BNP2/BNPDemo.zip

(17 MB)

Note:

LOW RES (for download reasons) - All graphics half sized (unfortunately this messes with the antialiasing)


Level - very rough so far (ie things like wooden beams out of place on bridge etc)

4 Playable characters (Barbarian, Knight, Mage, Archer)

3 Types of Enemy (Skeleton, Warlock, Black Knight)

Controls: (I really do recommend a joystick of some sort for this)

Character Select-

Cursor / Space To Select / Enter To Start (if more than 1 player)
Joystick Left & Right - select character, button 1 to select/start (press Enter if more than 1 player)

In Game-

Cursor / Arrow Keys & Space Bar (to hit things)

OR

Joystick / Joystick Button (need to change InputMap.txt file - instructions in readme.txt although it should not be too hard to work out.)

ESCAPE to Quit.

F1 to mute / unmute sounds

Thanks,

Matt
