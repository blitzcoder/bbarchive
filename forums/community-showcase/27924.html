01/12/2003 : New demo version 1.1 is out, enjoy !

***About this version 1.1***
*New timing system ( better than version 1.0 )
Normal : classic timing, but absolutely not the best
60Hz : set your DirectX ( search dxdiag.exe ) or Windows to 60Hz ( this is a default refresh mode ) to run the game as smoth as console. It is the best method.
*Press "Tabulation" to reset the game,
*Better gameplay,
*Level 13 of the full version is easier,
*The game starts faster,
*The game runs in full screen 640x480 optimized in 16bit,
*The reeteam logo.
