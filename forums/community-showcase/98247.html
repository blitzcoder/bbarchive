Concept Engine (WIP name)

I have been developing a fully moddable 2.5D engine for the past couple years. The idea behind Concept (working title) is to create a generic, modular based game engine with ability to create new game types via "gamekits". Currently a RPG gamekit is developed along side and serve as a test bed to test the limits of Concept.

Today, I am releasing a sample game developed using Concept with the RPG gamekit and hopefully check out the feedback from the community. The sample game outlines many elements of a typical "RPG" game with the except of a battle engine. (The battle engine has been taken out as there are still small bugs to thin out in the script so it will be added back later)

Link to download: Dropbox Link




Controls
- Esc : Call up menu (currently doesn't do much, but you can see how one can script menu)
- Numpad Arrows : Movement (including diagonal arrows 7, 9, 1, 3)
- Numpad Enter : Accept
- F1 : Exit

*I apologize for the people who has no NUMPAD, a NUMPAD must be used for this demo. (Controls are fully changeable, but not for this version.)
