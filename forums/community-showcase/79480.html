This is my first attempt to create (in BlitzMax and for BlitzMax) a Synaptic-clone.
The program downloads (from my site) a list of applications that can be downloaded and further installed.
BlitzMax Companion will install SubVersion (and it will create links on desktop if required for Install & Update batch files) and MinGW (it should automatically modify the Path and installs a new environment variable MingW as required in the manual installation)

The list of applications includes

- SubVersion
- FrameWork Assistant
- Help Search Utility
- UPX
- ResHacker
- MinGW
- Italian LangPack for MaxIDE Community Edition

* the application ask for BlitzMax path and WinRAR. It can be required to decompress some files. ZIP files are managed directly by the Companion.

* BlitzMaxCompanion is in BETA, so please handle with care!

* WIN32 version, tested only on WindowXP. I dont' know IF and HOW it could works on Vista or other Windows systems.Tested on Windows XP and Vista Premium.

Video: http://www.youtube.com/watch?v=k1k7PgBRcDA (based on v. 1.00)
