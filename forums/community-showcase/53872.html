Just updated a little fun project complete with a Max GUI interface.

*v1.3* - couple of tweaks and fixes



See the contained ReadMe.txt for usage but here's a quick start guide:

1) Extract the zip contents to any folder
2) Run the MaxWebcamViewer
3) Select a cam from the listed ones
4) Click the View webcam button

If all is well a separate view window should open shortly.
You can open any number of cams for viewing but bear in mind the GUI will become unresponsive whilst Max is fetching images from the web.
Note: You can scale these windows too.

Closing the app tidies everything up (closes all opened cam views) and saves whatever links you have entered in the list too. More details in the ReadMe.txt though.

Enjoy!! (and let me know if you have any good links)
