I made a nice platform game called "Erika E. Eppeley".

My goal was to make an oldskool platform game as in good old times, though I did try somethings with the engine that may make it not as oldskool as it looks. :)

The game is completely freeware, and I'm am planning some expansions, but I'm not planning a commercial version, so the game will always be free to download.

Some nice features Erika E. Eppeley supports
- Levels in "worlds". Each world (except the final one) has 3 levels following the theme the world was set up for.
- Compatible in both MacOS and Windows (MacOS is a PPC build, but I tried it on an Intel Mac too, and it works perfectly).
- Cheat codes if you hate fair play, you can try out winning the game with those :)
See if you can figure them out :P (I may post the cheat codes later)
- Patching. Erika E. Eppley has a patching system, that allows you to play EEE without modifying the original game, though I never created any tools to make use of the patching system, a document on how that works is provided on my download site. :)
- Customizable keyboard setup. If you don't like the keys as I set them up you can change them.


Standard keys
Direction keys - move Erika
Alt - Jump
Ctrl - Fire
Space - Activate switches
F1 - Pause game
F5 - Save game*

* To prevent save game abuse, the game only saves either the start of the level or the state of the game when you activated your last checkpoint.


Known bugs
- Sometimes when standing on a "moving platform" your feet may get stuck in it. I'm not quite sure how to fix that, but I am working on it.
- One grave at the beginning of "Twilight Zone - Level 1" may block the way while that was not intended. However you can jump over it. I know how to fix that and will do so in the next version of the game.

If you see parts in which the game can improve, let me know (however, I must impress that I know that some graphics suck. I am a poor drawer and I didn't have the luxury of having a graphics designer. If you know somebody who can do a better job than I did, lemme know. I'm always interested).
