Treeparty 1.42 ( Blitz 3D ) has received a small update to allow the use of FastExtension within the library.
Please note that this update doesn't include the FastExtension library by Mikhail Vostrikov ( you may find the extension here http://fastlibs.com/ )

If you don't own the FastExtension library then please update your Treeparty to the non FE version. Fe versions of Treeparty assume you have FastExtension.dll, decls and the wrapper installed and ready for use, these files are not included within the Treeparty bundle.

A small Treeparty FE demo source is included with the update to demostrate using Treeparty and the FastExtension together.

This update adds two new Treeparty functions that make using real time shadows ( fast extension ) with your trees a snap!



A new demo download has been posted - http://www.gothasoft.com/index.php?option=com_docman&task=cat_view&gid=29&Itemid=78
