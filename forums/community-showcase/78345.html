Here is a fun little simulation ( wouldn't call it a game since there is no objective. ) I put together quickly, you might like it, might not. It does not contain AI for the human npcs, as in, they do not try to elude the attacks.

When you run program it will ask you for blood intensity settings 1-5, I find 5 to be beyond normal unrealistic, and find 2 or 3 to be the best setting, 1 is just too little.

Option 2 being the formation, putting setting 1 will make them in regular formation ( Lined in rows ) and 2 being stacked into one person ( I did this for physics testing, which may I add is pretty bad in this program )

Features:

Weapons including-
Machine Gun
Missiles
Bombs
Carpet Bombs
Nukes
Carpet Nukes
In-game you can see which buttons to press to fire the weapons by pressing the '2' key in-game.
