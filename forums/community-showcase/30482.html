Download here!

qoob is a game of skill, cunning and square bashing.

We have made some major changes and are almost ready for a full public release. This version does not yet have all 64 maps but does come with the editor we created and used to build the maps so you can build your own for inclusion in the final release (due in a few days).

Please download, try and let us know your thoughts. Many thanks.

WHATS NEW?
General
The qoob worlds are now formed into a galaxy consisting of 8 qoob systems each with 8 qoob worlds, the player now has to find the decryption key for each qoob world in order the crack the 64 bit qoob galaxy code.
A limited version of the editor developed by us is now included FREE along with qoob galaxy pack 1, so why not create your own qoob worlds and send them to us for inclusion in the soon to be available, much awaited qoob galaxy pack 2.
The full un-limited version of the qoob editor (allowing maps of up to 25*25*25) is now available for purchase at a very reasonable (negotiable) price.
Install script now has a new custom logo!
Game executable and shortcuts now have a new custom icon!
Game splash screen now has updated image and new sound effects, you can also now interupt the splash sequence by pressing the following, Mouse buttons 1, 2 or 3 - Space bar, Enter or Escape. Are we nice or what?
We have now added our new company name "Boiled Sweets" to the help dialog and this document as well as changing the location of the web site. www.boiledsweets.com
A new graphic has been added to mark the player start position.
Major code tidy and effeciency improvements made.

Menu
New music.
All menu cube gfx improved (high res now).
Menu has been changed to include galaxy concept with onscreen world information text, this includes the code which the player must find by visiting all the qoob worlds.
We have updated the developer pictures in credits.
Save and Load routines are now fully functioning.

Editor
Added an 'insert cube here' button in the editor, the 'undo last deleted cube is now mapped to "U" (was "Insert").
Play game music when in editor test game mode.
When creating a new map the user must change the map name (from 'untitled').
Stop death in editor f4 test mode, as this was a royal pain!
Allow start position on top level of the map

Game
Introduced a nice decryption routine at end of level to replace the 'exit', the player now has to find the decryption key hidden somewhere in the levels.
Added colour coded doors and keys concept to the game, this has increased playability ten fold, also if the user needs a door key in the game we now tell them.
Modified the starfield to add some randomness (ie. colour), it also turns slowly, this makes the game feel better.
Look up/down distance increased to allow users to see more.
Add a player start position marker, hope you like it.
The two tutorial levels have been replaced with a nice new tutorial maze with on screen instructions at specific points of the map.
We now allow the player to fall any distance without penalty.
The landing sound now only plays on big distance falls.
Added lots of new maps for the players to attempt.

Bugs Fixed
Stop game crash when setting start/end positions outside of cube array.
All maps have been updated and fully tested.
