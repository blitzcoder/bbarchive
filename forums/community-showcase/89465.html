My first BM project is still evolving and now the beta 0.9x is available for download.

Star Opus is space action adventure type game. The aim of the game is to complete all the jump points and beat the enemy you'll encounter in the last planet called 'Kaou Pih'. There will be 54 jump points to navigate - with full of cool enemies, black holes and alien hex-cities.

All the graphics are made by using Inkscape. This app really rocks.



http://www.emsproject.com/staropus_094.exe
(zero setup)

Key info;

Use cursor keys to move and thrust.
A - shoot
S - throw grenade. Hold button down to gain speed.
