Contributions are now being solicited for the penultimate Blitz demodisk of the century.

We are looking for demos and code in the following categories:

* GAME DEMOS (source code optional) - a great way to get your demo spread!

* MISCELLANEOUS - reserved for those truly unique entries!

Please send all offerings by link or attachment to demodisk1@...

If you are submitting a demo of a commercial game, feel free to include some HTML advertising it and
linking to the full version.

Glory and praise to be offered in gratuitous quantity to all entries accepted.
