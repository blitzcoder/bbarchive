RepeatUntil(Posted 2008) [#1]
RakNet is a VERY powerful network library, with a lot of possibilities (see here).

The wrapper of RakNet version 3 was available since a long time now for BlitzMax, and, thanks to Paco, is now also available for Blitz3D and BlitzPlus! There is still no example for Blitz3D, but you just have to follow the example for BlitzMax (and if anybody wants to write an example, feel free to send it to me).

The wrapper is open source and any improvement is very welcome.

You can download it here: http://repeatuntil.online.fr
