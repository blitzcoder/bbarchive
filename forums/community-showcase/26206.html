fredborg(Posted 2003) [#4]
BTW. since Photoshop 7 don't support TGA with Alphachannel anymore....
Of course it does, they just changed the way it works (to a better method in my opinion). The seperate alpha channel is not exported, but the built in image alpha is. This means you need to make the image transparent (so that you can see the checkerboard), in order for Photoshop to export it. This makes it much easier to control the final output.

The windows are just JPG textures loaded with the alpha flag (black=transparent, white=opaque). Of course, if you use a file format that supports alpha (TGA or PNG), the alpha information in the texture will be used instead of the color map.
