A short time ago I decided I'd make a game on a weekend. It ended up taking a little longer than this, and in its current state is still not finished - all I am going to do from here though is put in a 'title' and 'menu' screen and alter the 'pub' model.

Hopefully when it gets finished (should not be too long) I will post a completed version. However if anyone wants to try it out then download from here:

http://home.swiftdsl.com.au/~gezeder/lloyd/BarRoomBrawl.zip


(1.8 MB)

It is not intended to be single player but rather multiplayer (2-8 players) on the same PC with each player using a USB gamepad. Left thumbstick/d-pad moves your player around, button 1 punches/kicks/headbutts/picks up bottle/throws bottle, button 2 drinks from bottle if you are carrying one which puts your health back to full but increases your drunkeness.



The object of the game is to be the last man standing in the pub.

Oh..there is a bug which does not occur on my laptop with an ATI card but does occur on my old PC with a Geforce 2MX -when the screen fades to black after the end of the game only half the screen does, don't really know why that is but I am using Blitz3d 1.90.

Source code is included (I think) so feel free to fiddle around with it.


There are no keyboard controls in the game - only joystick/gamepad controls. There are not going to be any keyboard controls either (although they would be simple to put in).

I am primarily interested in feedback from anyone who plays it with 2 or more people.
