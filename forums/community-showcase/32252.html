Vertex assignment and auto/manual vertex weighting now complete.


Select from 1 to 4 bones :



Select Vertex :



Hit 'Assign Selected' to assign selection to selected joints. The weighting values will be individually calculated for each vertex based on distance from the joint and its position in the hierarchy.You can modify the Primary Joint Bias each assignment. This gives you more control over the auto-calculated values without having to set the values manually.



Or just set the weighting values manually :




There is also an 'Assign To Nearest' option that does a pretty good AutoBind. Demo to follow shortly.........
