Now Available Arkhelom 3D Version 1.1 that addresses some of the problems identified by the first users to play.

Version 1.1 features:

- Real-Time Shadows (various settings in options menu selectable)
- 2000 Initial Bonus points (for initial configuration of the ships)
- Items to start a game to help improve the ship and lower the initial difficulty of the first level.
- Improved speed ships and response to mouse movement. Placed in the options menu some configuration about mouse sensitivity.
- Improved visibility of some types of shots.
- Small improvements and correction of some bugs.
