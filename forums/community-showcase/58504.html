TomToad(Posted 2006) [#1]
This is just a simple maze game. It's like those on the web where you must guide the mouse through a maze, but this one has a twist, literally. Just click on the smiley face in the middle, then try and navigate your way through to the smiley in the upper left corner without hitting the walls.
You can get the program here.

Edit:New version uploaded. Now the program starts with a main menu. If you hit a wall, you have the option of retarting that maze by pressing SPACE or exiting back to the main menu by pressing ESCAPE. Also changed the Twister pattern and fixed a bug that left maze #4 unsolvable.

Edit 4-13-06: Version 0.2 now done. Fixed a bug which detected collisions when there were none. Also changed the logic in the programming to make further inhancements to the game easier. Also used Framework Assistant and UPX to make a much smaller executable.
