Blitz IDE Manager v1.0
------------------------------------------------------------------------
Blitz IDE Manager is a freeware tool written by R.Baldewsingh
as a simple way to manage and create prefs files. This was mostly
aimed for use with Blitz3D but can e used to Blitz2D or BlitzPlus.
Tough this has not been thuroughly tested.
------------------------------------------------------------------------
This tool is freeware and may be treated as sutch. Just don't use
it for commercial purposes without my permission.
------------------------------------------------------------------------
Application Features :
- The Ability to alter Prefs Files.
- 10 Template Prefs Included.
- Keyboard Shortcuts for fast Access.
- Fast And Easy Prefs Creation.
- Painless Blitz Look Switching.
------------------------------------------------------------------------
If you find any bugs or have some suggestions please feel free to
contact me 'darklordz@...'.
------------------------------------------------------------------------

P.S This tool is very small (50Kb/20KB UPX) so that makes it efficient in almost every way. (The Install is 109Kb tough)
