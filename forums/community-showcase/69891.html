If you aren't familiar with Second Life then this will be an idle curiosity at best, but I've been working on a simple tool to help people making clothes for that virtual community:



In essence it's just a viewer for you to apply textures to the standard avatar in Second Life and see how your work is shaping up, but it has a few nifty features to make it especially helpful, in particular the Reload buttons so that you can update the changes if you are working on the texture at the same time as previewing it.

I actually wrote this a while ago in Blitz3D, and have been wishing I could port it to Macs. Well I found out about miniB3D on Saturday, immediately purchased BlitzMax, and now the port is almost complete. Thanks to MaxGUI, the interface is much cleaner and easier to navigate.
