Hello, here is an implementation of Craig Reynolds boids algorithm, written in Blitzmax. I intended to use this to manipulate small "armies" for a game but I'm not sure it is suitable for my needs. Maybe someone will find it useful.



Video with 1000 boids : https://youtu.be/WUXq7GYH62Y
Source code : http://www.blitzmax.com/codearcs/codearcs.php?code=3297

This helped me to learn types synthax in Blitzmax and especially types that differs from Blitz3D, I hope the code is not a mess :D

Anyway it's quite fast I think (or Blitzmax is fast, really fast when you come from Blitz3D !), depending of the number of boids of course and the number of neighbours each boids has to process. If you're interested, check the code archive link, you can tweak some parameters to make different patterns.
