Grisu(Posted 2009) [#4]
Played the demo. Things I would change:

Menus:
- the main and options menu don't match the sytle of the other ones.
-> use the same font / image style to make all sections feel as "one".
- offer mouse support
- offer full screen support

Game:
- it would be cooler if the submarine would shoot torpedos (even cute ones)

- your're "wasting" 1/3 of the screen for the sky. If there are no planes or birds to shoot. Why use so much space for it. You could also add clouds, that cast shadows... :)

- I played the game in English. Some translation errors, though native speakers might be better for pointing those one out.
- "sound volumen" -> "sound volume"
- "ambient volumen" -> "ambient volume"
- "music volumen" -> "music volume"
- or even shorter: Sounds, Background Sounds, Music

If you reach the end of a mission I wouldn't expect a sign "Puerto". When playing the English version. I would choose a graphics for thia that everyone can identify as "port".

Grisu
