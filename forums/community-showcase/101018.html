Wartactical for windows & Mac

WarTactical 520B is for windows & Mac(A version for the Mac for the first time ever) and is free! there is an older game at web-grognards but the site isn't excepting any new files as you will know.

I will give you some info here.

The game is an infantry squad based tactical game but you can use vehicle like tanks etc... it has a computer opponent and is turn based. you use chits for your units on a map, the map is aprox 1 mile square, but the entire map is visible on screen from an overhead view. you have commander HQ units that give bonuses to unit whithin range of your hqs. the units can become shaken and have penalties for being shaken by attackes, so attacking an already shaken unit is a good stategy. the dash at the bottom of the screen shows info on the selected unit such as TEM terrain effects modifier, when a unit has a TEM it gains advantages in defence. you can also see on the dash if it is an amored unit and if it has anti-armor abilities. the dash give real-time info like if your shot hit or missed. Line of sight is also included in the game, if a building is in your way and you have set -up lineOfSight when you or scenario designer created the scenario.

there is a editor to setup a scenario, its very very easy to create a scenario, you can place TEMs roads,rally flags,victory points,walls, lineof sight blockers and units.
