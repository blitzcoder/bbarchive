Good Day to you all once more,

Had a day off work for a public holiday on Wednesday, so using a few bits and bobs from around my PC I put this little game together.

Some screenshots below.

Download link: (11MB)

http://home.people.net.au/%7Etmalcolm/dungeongameagain/DungeonSnakePit.zip

Youtube Footage:

http://www.youtube.com/watch?v=v3hBXwp1yco

Basic Instructions / ReadMe:

http://home.people.net.au/%7Etmalcolm/dungeongameagain/ReadMeDungeonGameInstructions.txt

However this game despite the level geometry being quite simple, there is a lot of texture information in the sprites so a hefty 512MB VRAM is required (EDIT - I could be wrong as I estimate that it really should work on a 256MB card as well...)

This is the first game I've written in possibly a year or more..about time too. Spent about 16-20 hours on it, all finished and working except for a couple of bugs with the high score table...
Admittedly I remember now why I stopped making games - get too absorbed in them and forget to take care of everything else!!!

The object of the game is to score as high as possible within a 3 minute period. Power ups which improve different stats are located around the dungeon, as well as lots of monsters (quantity not diversity).
To Play - SPACE to shoot fiery arrows, TAB to blast enemies with magic, ARROWS to run around and attack enemies with your sword (by running into them).
