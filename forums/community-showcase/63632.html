stayne(Posted 2006) [#21]
If you're using the code from the WaterSample2.bb file, disable refraction (Refract=0). Once you do this your water will be opaque. In the main code there's a section that deals with the fog according to the camera's Y axis and you'll see 2 render modes. Comment out render mode 152,1 (or something like that) and leave the 48,1 mode active. The water will be transparent/translucent again, but not refractive. This will also eliminate the hud reflecting and other negative visual issues (like sprites being cut off).

Please let me know if you find a better way :).

The reason it needs so much RAM is because all the scenery is loaded at the begining and the vegetation system eats a lot of memory, sorry...
