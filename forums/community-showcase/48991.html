It's been a long pause in development of nano... it's now resuming after several months, so I felt I want to release something.

the game is still VERY unfinished and so is the menu and so are the game modes, sounds etc. but you can drive around. :)

Human player controls
p1 - arrows + control
p2 - mouse

Weapons are randomly selected when starting a game. Currently there is homing missiles, homing cannon and double cannon.

press down / middle-mouse to grab stuff with your tail when the tail is touching the object.

Also press 1 in the game to see some statistics (fps etc...)



download at http://www.rippedsoftware.com/nano_03_alpha.zip


I would like to know if it works for everybody.
