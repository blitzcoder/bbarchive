Heya, i knocked this together, because i needed a program that would adjust the vertex alpha of a mesh, whilst keeping simple one set of UV co-ords intact upon saving out to .b3d. The control are available by pressing the (S) key any time. Will save out simple, textured, or untextured meshes, retaining one set of UV co-ords. I will put more work into it later, but for now i have to go away for a week. I hope this helps someone, as i couldn't find a free app to do this.

RUN "code.bb". You model must be called model.b3d, but you can change the load line to whatever you want. Check the code to see what it saves the model as. :o)

www.rosscrooks.pwp.blueyonder.co.uk/alphasave.zip
