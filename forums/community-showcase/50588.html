Mapcrafter has gotten some work done on it lately, its been awhile so ill touch on what it does.

it lets you make a map in a 2d editing fastion by painting
the landscape with texture brushes..wich set texture and
heightmap info all at once.

then you can select a small icon image rpresenting an
object, and stamp it anywhere on the landscape.. these
icon stamps range from foiliage to buildings

an example of the 2d editor (shot from old versioin . new
version has newer gui)

http://www.empowergames.com/newmc/MC_2D_edit.JPG?
and you can export to a 3d map.. now with the 3d map you
go into 3d edit mode wich gives you a few more options..
you can further tweak the map by scaling , moving, and
rotaing objects . you can lightmap the scene in various
ways. and export to a .b3d model.. also you can dump
images of the map. wich can be used for pen and paper rpg
games using miniatures.


The dumped images range from small 800x600 to large
3200x2600 with hexagon tiles on them. for PNP games.

RifRaf(Posted 2005) [#23]
Mapcrafter has gotten a few updatates, and a user base 25 strong. woowhoo. lol

changes.
2D MODE. -----------------
you can select if you paint with height brush, texture brush, or both.

You can autogenerate heightmaps, and autotexture them. based on the currently selected texture pallette.

You can Load all model/stamp packs, or just the ones you want to work with.

3D MODE -----------------------------
Changes you make to models in 3d mode are saved in the .mc m file, and so when you go back to 2d mode, your 3d changes arent lost.

Building catagory models allow objects to stack on them . meaning in 2d mode, you can place castle, or large building stamp.. then for example stamp a flag or another castle level on top. and in 3d mode it will stack as intended. other objects will stack on buildings catagory objects too, but you wont have to worry about other objects stackin on each other.. like trees ect.
example .. http://www.empowergames.com/mapcrafterpromo/totem_spires_all.jpg?

ROAMER MODE ------------------
cloud cover, and animated water added. as well as water wiggling view when beneath the water.

--PLANS.
I plan to incorperate a network game launcher and client directly into Mapcrafter.. these will allow you to launch with MC maps as your game level, inside the game you will have movement measuring, dice to roll , chat room, and game peices you can drop and move based on rolls. as well as making non player game peices spout dialoge and such. The aim is to have an environment with very loose rules to allow players and game hosts to play in table top style rpg. using whatever rules they like.

It will have very easy to script special effects, dialogues and game peice movement/spawns. to enhance the game play.
