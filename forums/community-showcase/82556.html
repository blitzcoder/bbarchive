Hello everyone.

I have not posted anything here for quite sometime, because
I have been so busy working on my first 'real' game.
This game is far above all my previous ones, because I actually
went to quite a bit of trouble to draw the graphics and animations

And thus, the graphics might not be great, but are at least, fair.
Also before posting this here, I had a few select play testers help
me work out the bugs.

This is a remake of the NES game, Dragon Spirit, the new legend.
I originally was going to enter this in the Retro Remakes
competition, but I couldn't get it done in time.


Dragon Spirit is still far from complete, but the first two levels are
done, and now that I have gotten the rest of the game pretty
refined, the next 8 levels will not take long to complete.

Anyways,
I hope you enjoy the game, and please enter the How to Play
section found in the options menu if you have any questions.

Use the arrow keys, and Enter to navigate the menus.

Enjoy!
