This is my first game made from BlitzMAX. Although it has been 3 years(!) since the release of the test demo, I'm glad to have the game almost completed!

The game is indeed a remake of a 1991 PC game of the same name. Perhaps anyone here have played it? I've used the original concept of keeping the score above zero and add more levels and obstacles.

Currently, I am looking for SOME players to try the game out during the closed beta. In order to try it out, you can visit the project page and submit a request to participate. (You will need an account at Acoders in order to request.)
