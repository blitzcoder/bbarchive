Mapcrafter is still kicking. Current version is 1.7 . I added a topic for it in the Blitz Toolbox area, under 3D terrain tools.

Feature rundown.
Use 2D and 3D editing to create your terrain
Drop scenery objects in 2D mode, and fine tune in 3D mode.
Perform lightmapping in both 2D and 3D modes
Save work as models in b3d or .x
Save printable PnP images with or without hex grids
spanning 25 pages (5x5)
Randomly generate 2048x2048 heightmaps
Manually tweak heightmaps/colormaps with brush tools
Autotexture heightmaps based on textures choosen. using
height/slope algorithms
Over 80 scenery objects included with Mapcrafter, and
increasing with each update.
Create maps from the toonish, to the more realistic.



Http://www.empowergames.com
