What's new?

nSprite Pro 2.2.0 (28/11/2004)
-Good speed gain on nS_Cls calls
-Fixed a scaling bug in nS_ResizeImage
-Speed up nS_LoadImage and nS_LoadAnimImage a bit
-Fixed an issue with black lines appearing
-You can now use vertex lights, creating ultra smooth transitions (but you
have to use nS_ImageFX 2 for vertex color and set the image as vertex lit -
see nS_ImageLightAffect)
-You can now create lights that also affect image alpha (see nS_CreateLight,
and use nS_ImageFX 32 for vertex alpha)
-You can now create lights that act inverted (reduce color and/or alpha)
(see nS_CreateLight)
-You can now move particles associated with an emitter (for scrolling) (see
nS_MoveEmitter)
-Complete Bitmap Font System overhaul, it is now single-surface (for the
whole font, not by character), making it way faster!
-Fixed a small bug with nS_FreeAll in combination with 3D Objects

nSprite 1.5.0 (28/11/2004)
-Good speed gain on nS_Cls calls
-Fixed a scaling bug in nS_ResizeImage
-Speed up nS_LoadImage and nS_LoadAnimImage a bit
-Fixed an issue with black lines appearing
