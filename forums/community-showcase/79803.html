Stevie G(Posted 2008) [#4]

Just curious - do you store the whole level in memory at once?
Or do bits get created/destroyed as they are needed during gameplay?


Yes. The whole level is held in memory. The levels are quite low poly - less than 10k polys in most cases.

The mini-game ideas are very simple :

Survival - Each team has an offroad with no weapons, chased by a team of tanks with a set number of lives - last team standing wins.

Copland - First team to kill a set number of cop cars wins.

Deathmatch - First team to kill a set number of enemy vehicles wins.

The Light - First team to be first to the light a set number of times.

Stevie
