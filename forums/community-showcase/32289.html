Hi, I've just finished work on a free 3D model viewer designed to replace the standard Blitz3D media viewer.

It can be used to open b3d, 3ds, x, md2, and Blitz3D compatible Quake 3 bsp model files by double clicking on their file icons.

It can be used in place of the standard Blitz3D media viewer, in which case it can only open the file types that the IDE passes on to it. You can also set the supported file types to be opened (shell executed) with it directly, in which case it can open all the supported file types without the Blitz IDE being started in the process.

The viewer features a variable speed, fly around movement system with mouselooking. The Windows mouse pointer can be toggled on or off by right clicking on the screen to allow you to work with other applications while the viewer is running.

The program is designed to make minimum use of the CPU when idle (it shows up as 0% on the Task Manager) and multiple versions of the program can be running at once without interfering with each other.

Check the 'READ ME - mediaview.txt' file that comes with the download for more information.
