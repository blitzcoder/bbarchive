Hi there,

Strategy/Stratego is a turn based game from old times. 2 Opponents
with 40 pawns each try to catch the opponent's flag. I think many
of you know how this works.

I finished this project about 2 months ago and don't want to keep
it back from you.



It is a multiplayer only game. you can save your pawn-setup and load
it again. If you have a LifeLeech account ( http://lifeleech.0x25.de )
you can save your scores (win,draw,loss) and get achievements
like on XBox, Playstation or Steam.

There are still a few bugs and not every function has fallbacks

However, it's free, try it out and if you have any questions,
just ask ;-)

EDIT// It has no proxy support, yet.

http://0x25.de/proj_stratego/stratego-1.00.rar
