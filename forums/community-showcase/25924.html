The generator part of xTexturemaker 0.4 is now finished.
xTexturemaker is a program made for creating textures, pixel by pixel, using an algorithm based on the nearby pixels that have already been created. Which pixels that will be used is determined using one or more normal 24 bit bmp files. Colours can be chained to each other and the factor of randomness, and many other factors can be altered.

So far, xTexturemaker 0.4 can only be launched using a bat file (One, easily alterable, is included), or using another program (No programs support xTexturemaker 0.4, yet).

Comments, ideas and questions are welcome.
