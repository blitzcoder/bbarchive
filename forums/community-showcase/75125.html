Project Studio Express v1.0.42 now available

Community Forums/Showcase/Project Studio Express v1.0.42 now available

Matthew Smith(Posted 2008) [#1]
An updated Project Studio 2007 Express for BlitzMax and Blitz3d/Basic is now available. The following changes have been made in this release:

* Updated a large number of intellisense/tooltip identification issues around type inheritance (especially self and super) - BlitzMax plugin
* Updated the Set Startup Object feature to force a rebuild of the application when the startup file is changed
* Moved the Font and Color selector for each language to the Settings page - BlitzMax and Blitz3d/Basic plugins
* Fixed an issue where web urls in descriptions caused malformation of tooltips - BlitzMax plugin
* Fixed an issue with Rem/End Rem (thanks Mike Tilley!) - BlitzMax plugin
* Fixed an issue identifying keywords after Import, Framework or Module - BlitzMax plugin
* Fixed an issue identifying namespace keywords after a colon when setting a variables object type - BlitzMax plugin
* Fixed an issue identifying keywords after arrays ie. myarray[1].Bar (thanks GFK!) - BlitzMax plugin
* Fixed an issue with encoded text being output to the output/debug windows (thanks JCKodel!)
* Added an option to turn off the advanced syntax highlighting (Options > Languages > BlitzMax > Settings) making the highlighting match VS - BlitzMax plugin
* Added a proper Build Configuration (Debug/Release) option for the Express/direct file compilation
* Added a 'Project Studio Forums' button to the Help menu
* Added a 'Report a bug' button to the Help menu
* Upgraded the development environment to VS2008 (very nice!)

Special thanks to GFK, Michael Lindholm, Mike Tilley and JCKodel for assisting me to fix some of the defects.

Download Now!
