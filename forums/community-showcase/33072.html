============
Image Packer
============

Lots of changes/tweaks/bug fixes to Image Packer.
Help docs are now in place. These are accessible via a separate HTML view window.
There is a new 'Fit image to frames' option in the animation options. This allows the image to be scaled so that it fits in nicely with the frame ranges.

================
SpriteMaster Pro
================

Tons of changes to SpriteMaster.
Now called SpriteMaster Pro.
Total of 34 commands/functions to play with.

Some highlights:
* Collision checks with SpritesOverlap()
* Grab sprites/images/textures from a pack
* Sprite 'Text' commands
* A little faster and cleaner than before
* More flexible

SpriteMaster uses might want to read up on the changes

===========
ImageMaster
===========

The 2D version of SpriteMaster Pro.
For extracting standard 2d (animated) images from packs created with Image Packer.

See link in signature for more details.
