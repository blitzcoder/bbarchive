Well, the mission editor for Space Hulk BE is coming along nicely. I've included a screenshot for those interested. Click for larger



It's pretty smart - places walls correctly based on existing layout, etc. You can see the big empty panel over to the right - this is where a tabbed dialog with mission parameters, texture, model info, etc. will live.

The 2D map creation part is working fully. I have to add in UI hooks from the tabbed dialog now. The 3D preview/texture paint mode isn't started yet but I have it planned out and it's really a fairly minor part - the map structure is the key and that's all in there.

Don't know for sure when the editor will be done but, when it is, I'll release it with a couple of sample maps - and all of you Space Hulk fans can build your favorites ;) I don't want to release it until the map format is stable though and I'm still working on how to do mission parameters, etc.

Later,
M
