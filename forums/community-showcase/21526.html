Skitchy(Posted 2003) [#6]
No, because .x only supports one UV layer ie. no lightmap co-ordinates are stored.
At least, that is, as far as Blitz is concerned. I'm unsure but I think that the version 9 .x files might be able to store such data, but as Blitz is DX7 it would be unable to use it.
I assume you're asking because you want to use LightBulb for DarkBasic ('cos for Blitz .B3D is a superior format anyway). I've been thinking about doing a DB version of LightBulb, but I'll probably wait until they've got their new .dbo format out.

LightBulb DOES export .B3D files, with all the texture tricks 'baked' in.
