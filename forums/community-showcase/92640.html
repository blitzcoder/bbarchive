Sort the score table:

1) Very easy Bubble Method:

Function SortTable()
For n=0 to NumTableScore
For m=0 to NumTableScore
If Score(m) < Score(n)
AuxScore = Score(m)
Score(m) = Score(n)
Score(n) = AuxScore
Endif
Next
Next
End Function

; Insert a new score
Function InsertNewScore(NewScore)
AuxScore=NewScore
For n=NumTableScore to 0 Step -1
If AuxScore > Score(n)
Score(n)=AuxScore
Return
Endif
Next
End Function
