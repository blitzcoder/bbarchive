N(Posted 2004) [#7]
I only put restrictions on stuff if I want to make sure I know who's using them. Usually the restriction is along the lines of
"You are permitted to use so and so in so and so applications, software, etc. without getting my permission. If you wish to use the aforementioned media outside of so and so applications, software, etc. contact me via e-mail and describe in-detail what you're using it for and who else this permission should apply to."

Normally this gets the point across and people do the right thing and ask for permission. Including a license or read me with a file is usually a good idea if you care about who uses it.
