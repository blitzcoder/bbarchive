Kanati(Posted 2004) [#1]
I have released BlitzPrinter Userlib. It has a full installer and instructions on how to use it are in the docs. The docs are saved in HTML and RTF formats and should be print-friendly.

I did not include a demo program or code, and I'm sure I'll get around to that, but the docs spell it out pretty clearly how to use it (I hope). Just copy the dll and decls file to your userlibs directory and you can use the commands as you would any other blitz command. And when you create an install for something you distribute, include the vb runtime included with the install, the vbadvance.tlb type library, and the blitzprinter.dll.

It is free to the community and is classified as "donationware". If you want to donate... feel free. I would LIKE credit in the docs or the credit screen for something you create, but if it's not... no big deal. So use it however you want.

Get it here: http://www.outergods.com/blitz/blitzprinteruserlib.zip


Kanati
