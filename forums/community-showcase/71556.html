Starburst 1.20!

Boiled Sweets(Posted 2007) [#1]


Another release and loads of cool updates...

* Added shooting stars (S to spawn in interactive mode)

* Added nebulae

* Added values to the various configuration scroll bars

* Added the regular Boiled Sweets branding

* Major rewrite to the starburst effect
1. Looks LOADS nicer
2. Can have FAR more particles with less GPU load!
3. Each particle has it’s own characteristics

* Setting the Star Density to high caused crash – fixed

* Disabling the rockets, asteroids or UFOs and then trying to spawn one caused crash – fixed

* Made comets slightly smaller to make them look more like comets

* Disable the Resolution pick list if the Use Desktop Resolution option is ticked in the configuration dialog

* Various minor cosmetic improvements to the configuration dialog

* The Show FPS configuration option wasn’t working - fixed

* Replaced the F10 / F11 FPS display with 3D text to avoid the slow 2D drawing!

* Update stats every second not every millisecond!

* Fixed all the Z-order draw problems, near object always obscure more distant objects now

* Background stars now smaller

* In interactive mode “A” now spawns an asteroid

* Restore all defaults in the configuration wasn’t working – fixed.
