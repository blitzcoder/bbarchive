TROPIC[ALE] - Advanced Landscape Editor for Tropical FPS Game

It's a simple tool for create/editing Tropical terrain with a new terrain system engine based on Eole Terrain System.

Future features :
- Texture splatting (load your own alphamap).
- Tree system
- Foliage system
- Total occlusion system for trees, foliages and objects.
- Cast shadow objects
- and more... :)
