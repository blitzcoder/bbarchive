Having been drawing for years I never found the right tool to check and tweak animations. For this I have created this tool. It is a handy animation viewer for celled animations.

I tried to make it user friendly, I hope it is.

Download Animation Viewer

Cool ideas and suggestions for future versions are welcome. The source code is also in the zip. Oh and there are over a 100 frames of (w.i.p) animations included.

Notes :
- selecting more then one row of animations will usually not work.
- Top/left not implemented and no skip x,y pixels per cell.
- The program saves its info for each unique path+file used. Currently only the cell info. Delete the text file in the config dir to remove it.

Tip :
Use the left mouse in the animation window to move the animation manually, this way you can test animations in motion.
