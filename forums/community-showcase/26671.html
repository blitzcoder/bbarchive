PaceMaker 0.1 - An animation tool for Blitz3d

Much credit to Chris Fuller for BlitzUI !

Still very much at early stage of development but already has the following features :
- Animation Viewer
- Joint/Vertex Viewer
- Sequence Extractor
- Joint Creation
- Vertex Assignment
- Reference Frame Creation

Features under development :

- Mesh Animation Keyframer for .3ds, .x and .b3d
- Skeletal Animation Keyframer for .b3d
- Attachment of Accesories, weapons, clothing etc.
- Walk Designer/Generator

Demo of Basic Tools version out soon......
