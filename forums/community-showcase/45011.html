This is a program I made as an 'eCard' for the Vernal Equinox last year. It was my first attempt at a screensaver (thanks Grisu) and also at using a Borland C-Builder front end to create resources for a Blitz demo.

For anyone who is interested in understanding the whole Vernal Equinox deal, this is the screensaver. It uses a spheremap from spacegraphics.com that I haven't cleared permission for, so please do not distribute. This should install a screensaver (Equinox.scr)to your Windows folder (including a folder called EquinoxResources) and a widget to set your own dates (ControlPanel.exe)



The whole thing should be accessible through the Display control panel in Windows. {I seem to remember a problem with the Borland RunTime dll, but it was a long time ago...}
