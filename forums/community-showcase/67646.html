Please take a moment to try the first public offering of our new game, Interstellar Highway. It's a modern day homage to the BBC (Model B) classic Space Hiway. As far as we are aware there have been no remakes of this simple yet hugely addictive game.

The goal of remaking it was to keep the gameplay the same but bring the graphics up to date. We think we have managed to do that pretty well.

This was written in about 6 weeks from scratch, development times hugely reduced by using Sprite Candy, Particle Candy and IDEal.

We think it's ready for a wider release so please do try it and offer constructive criticism. As mentioned the gameplay is not going to change as it's a remake but please do report bugs & annoyances. We are always open to suggestions too.

We would be particularly interested in:

1. Ideas on how to limit the demo version (thinking of just allowing first 3 levels).

2. Does it run smoothly on your system?

3. How much should we charge for the full version ( and don't say <= 0 )

Lastly this version is time limited, i.e. it will stop working on 5th March 2007.

Many thanks...
