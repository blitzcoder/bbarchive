A demo version of ifsoGUI is now available in the downloads section of the www.ifsogui.com website or my website at www.ifso.net.

This is a demo of the latest version, it includes buttons, labels, windows, scrollbars, and textboxes. This demo is one mod, the full mod has a mod for the base gui and individual mods for each gadget, so you can only include the ones you are using in your final app.

Limitations of the demo version:
It is not possible to create your own gadgets or subclass any gadgets.
Any compiled app will have a window pop-up with an OK button that you must click every 60 seconds.
