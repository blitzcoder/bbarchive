Oh..I know that many people thought I should include keyboard support. Keyboard support is
available by passing the command line argument 'keyboard' (although the game really feels much better with a gamepad, it loses something with keyboard controls (cursor keys, A/Z shoot/drink potion, 1/2/3/4 select character)

Known bug/issue: If the graphics setup program is run it ignores command line parameters. This
usually means that on the first time the game is run it will ignore command line parameters.
