Cancerian(Posted 2004) [#5]
Thank you for the comments :)

It took me about 3 months to get Ghost Manor, Muddles and the website done and my next game, Marine Meanies, about 50% done. From the very beginning I wanted to be able to pump out games quickly so I'm sharing a lot of the non-game related code (GUI, initialization, etc.) between each title. It's about as close to a game "assembly line" as I could get.

Things would go even faster if I could get myself to use more 3rd party tools and libs but something in me wants to know exactly what each tool is doing code-wise and therefore I have to write them myself.

Like I said, Blitz was much easier to pick up than I thought it would be. I think having a project (and a deadline) in mind from the start and staying well within my design/programming abilities helped a lot.

Thanks again!
