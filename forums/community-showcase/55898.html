OK so I lied. There is a new version...

General

Trivial bug in the colour scale wallpaper routine fixed.
Added a ‘pause’ option – the P key.

Tunnel

Added 7 sided tunnels.

Textures

The tunnel’s individual texture layers can now be colour scaled (in real time), either random, red, green, blue, purple, yellow, cyan or grey. There is also an option to set the chance of a random colour scale.

Texture layers, 2 (lines) and 3 (stripes) can now be rotated. Also added an option to set the chance of rotation.

Expanded the dot & plasma layer’s diversity.
The “allow unfilled rectangles” option was not being saved in the configuration settings.
