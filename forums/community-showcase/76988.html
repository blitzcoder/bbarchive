Had a quick play. These both look like they have potential but I found the interfaces to be unclear. For example, in the Darts game it took me a while to figure out how to start a game and then when I did, those moving aim lines were very small. Then I couldn't figure out how to exit, so had to do an end task. When I started the start trek one I kept getting blasted by Romulans and the only button I seemed to get to work was the short range scan. Is it turned-based? If so perhaps you should dim out your buttons when it's the Romulans' turn. Again it was hard to figure out how to exit.

You clearly know how to program as you've made some quite complex games but accessibility is key. If you can make them easy for people who've never seen them before to pick up and play, you'll get a lot more feedback and success. Good luck with future games!
