Hi all,

It's that time again, ProPixel 2D Demo has been updated and is now in-line with the full version, additions to v1.22 are listed below. For a full break-down of additions please view the "Version History.txt" supplied with the download.

Comes complete with installer/uninstaller.

ProPixel 2D v1.33 Showcase/Download

v1.33 - Tuesday 28th October 2003

Fixed: Timer on Update Downloader
Fixed: Small issues found from v1.32

v1.32 - Tuesday 28th October 2003

New!: Saturate Brush/Drawing Mode
New!: Mix Brush/Drawing Mode, 25% of Brush/Colour laid per drawing/application
New!: Extract a palette from png, bmp or ppg images
New!: Save/Load Windows palettes (*.pal)

Fixed: Inconsistancy with Locked resize display data
Fixed: Some minor fixes incured from v1.3 to v1.31
Fixed: Ability to use resize filter on paletted images
Fixed: Load/Save Palette not remember last used directory

v1.31 - Saturday 25th October 2003

New!: Smooth (Bi-Linear filter) Brush/Drawing Mode
New!: Blur Brush/Drawing Mode
New!: Smear Brush/Drawing Mode
New!: Tint Brush/Drawing Mode
New!: Lightmap Brush/Drawing Mode
New!: Icon saving enabled (*.ICO)

Updated: FreeImage Library to v3.01

Fixed: Blank images when converting to 24bit then using Save
Fixed: PPG 8Bit loading bug with strange pitch results
Fixed: Glitch when changing brush size etc and not retaining visual representation of brush mode.

v1.3 - Thursday 16th October 2003

New!: Chm Help system fully integrated, press F1 over tools
New!: Web-update system activated, please register.

Fixed: Minor fixes with display update.
