This is probably in my series of useless apps, but some time ago I made a little app, that converts a font file (eg. ttf) to png-file.

I made this app for Monkey X apps to use ttf-fonts as png with Monkey X's DrawImageRect command.

I noticed later in Monkey X forums, that there is a better way of using ttf fonts in Monkey X apps. I really should find time to read these forums... :-)

In the package there's little Monkey X code included that uses the dat-file generated by Font 2 PNG (the dat file is for the DrawImageRect command).


But as said, this is probably quite useless app. Anyway it was made in BlitzMax.
