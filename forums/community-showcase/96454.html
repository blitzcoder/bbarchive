I've just written a new, improved Blitz Coder Archives downloader, available here. Please use the link below to get a copy of the code plus the current archives, rather than downloading all from scratch, unless you intend to work on the code.

http://www.hi-toro.com/blitz/codearc/codearc.zip

The program downloads all current entries that you don't already have into separate BlitzBasic and BlitzMax folders, sorting them into category sub-folders (Graphics, Networking, etc) and renames all files using the entry number and title.

Any entries that haven't been declared as Public Domain by their authors will be marked as such at the top of the downloaded code.

You can delete any entry or sub-folder to have it re-downloaded.

This is a multi-threaded BlitzMax program, but there's an executable in the linked archive so non-Max users can update too.
