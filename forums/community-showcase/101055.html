Ive Invested some spare time on creating this Notepad similar.
but with some extra features.

Like:
'StayOnTop' makes the notepad stay on top
'Refresh Text' Which is good for situations where the current opened text
is being modified externally.
'Goto text Folder' Takes you to the text folder (if you closed it)

More simple Addons will be added (Ideas are welcome)

The goal is to keep it light as notepad is but add extra features
that I personaly needed.
Im gonna keep it light so it will load as fast as windows notepad.

many things are still not implemented like print.. undo...
but i invest an hour a day for this little project.

It may grow to be something more interesting that i plan but it will always remain simple and fast.

for now its not a replacement to notepad since it lakes stuff as i mentioned.
but feel free to check it out from my dropbox. https://www.dropbox.com/sh/1u0e3xjmgw78yko/CSUNG9vAnH
