There's now version 1.03 of Ball Shooter available (freeware game).
Link
(notice the new WordPress based website :-) )

Full sourcecode and all the needed datafiles to compile included. Made in BlitzMax (my first BlitzMax game and it shows).

The source will be installed in the public documents folder, if you choose to install it.

Not good code, but some basic things for beginners demonstrated.

From version 1.02 functions lessThan4() and checkCircle() have been bug free.

New in version 1.03:
- playing times of levels are restarted after the ending
- checkCircle function simplified/optimized
- some comments added to the source

In the way from version 1.00 to 1.03 the source has become better commented.

Version 1.00 was written in December 2014 in a hurry.
