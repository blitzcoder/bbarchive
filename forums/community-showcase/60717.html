* About Crazy Chilli
Crazy Chilli is THE ultimate screen saver for chilli lovers every where! Watch chillies bounce around your desktop in hypnotic patterns. Crazy!

Features 100 different chillies including the world's hottest chilli, the Dorset Naga!


* What new - Version 1.01

Upped number of chillies from 35 to 100 !

Added 3 new different types of motion, Bounce X, Bounce Y and Fall.

Added configuration interface to allow the configuration stuff to be changed without needed to edit the cfg file

Added option to allow the desktop image to be used as a background so the chillies get drawn on the desktop not a black background

Added option to allow the chilli names to be shown

Added option to draw shadows (only active when using desktop as background).

Added option to periodically change the chilli image.

http://www.boiledsweets.com/CrazyChilli
