Warpy(Posted 2008) [#7]
I've updated it again. It's now almost a proper game.

The aim is to get as high as you can. The jewels are floaty, so you should try to keep them with you in your ascent. Bubbles get heavier the higher they go, so you need to keep switching to new bubbles to proceed. The problem is, bubbles that don't have any colours in common can't merge and will bounce off each other.

I've added some nice sounds, courtesy of the freesound project.

Thanks for your feedback so far, I'd like to know if anyone thinks this version is an improvement on how it was.
