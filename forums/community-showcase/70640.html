Thanks for testing.

Stevie G, there are some deadly 'vaporous' objects in some levels that you must shoot to kill or they will kill you.

You can shoot the white blocks for a small score or you can leave them for other situations involving flying discs whereby score may be maximised.

You can shoot the treasure blocks to release a single treasure object (sometimes this is easier and safer than moving into them).

You can shoot the '? blocks' to release a random flying disc rather than moving into them to release treasure. There are situations where this is more favourable for potential score.

The game is about maximising score and not getting killed in the process. Your score is only applicable if you reach the end of the level. There is some strategy involved though perhaps the game is not fun enough trying to discover it.

Heh, the graphics are simple and an unusual style but not that bad I thought. Ah well, I guess I’ll abandon this project and go back to 3d.
