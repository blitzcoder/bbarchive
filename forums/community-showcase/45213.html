Craig H. Nisbet(Posted 2005) [#4]
I am using n-sprite for the map clouds and explosions.

As for the gameplay, my goal is barrow some concepts, but not to make an advances wars clone. So the range fire will stay in. Many of the machanics are not in place so the game feels a tad imbalanced right now, so I'm asking that you bair with me on it. It'll be good. I've played lots of wargames and although Advance Wars was fun and I loved it, it wasn't nearly as detailed as my taste prefer. This is intended to be a "Hard-Core" strategy game at heart. Something between Advance Wars and Civ. The x-com return fire style will stay in, but be varied quite a bit from the way it's used now. The accuracy system, and an upcomming recon alert system will make the return fire system more plausable.

I have a strong possibilty that I can port this to an e-mail game. I'm not sure if people would like that but I wouldn't mind finding some interested people to test the concept.
