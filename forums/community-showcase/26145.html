TeraBit(Posted 2003) [#26]
Hi QuickSilva,

The demo gives you a code. When you register, you enter this code in the field provided, then an unlock code is emailed to you.

When you enter this, the demo becomes the full version.

This registration persists through subsequent updates, and normally there should be no reason to need a replacement code.

If you uninstall Tattoo or reformat your hard drive etc., a new code will need to be generated. If this is the case, simply drop us an email using the link on the website and a replacement will be mailed out to you.
