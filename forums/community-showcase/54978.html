Ricky Smith(Posted 2006) [#7]
You may find some files have more than one mesh chunk but it is uncommon in models built for animation. You need an ANIM chunk for each MESH chunk.
What you will find very often is files with more than one TRIS chunk. This is a multi-surfaced model - it only has one mesh but uses more than one texture map. Each TRIS chunk will use a different texture. Blitz3d treats each TRIS chunk as a seperate surface.
