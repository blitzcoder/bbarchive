Some of you may remember I was writing a cricket game some time ago, anyway, I decided to get back into it the last month or two.

Anyway, I released a very early beta version which is still largely unfinished. This is not the ideal version I'd envisaged, the idea is to get this basic version to a working product and then move on to writing a more enhanced version - firstly for the PC/Mac/Linux platforms and then hopefully for mobile platforms.
