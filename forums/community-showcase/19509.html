Description
========
BlitzAmped (project name) is video screen "FX" system driven by music from various digital media such as mp3, oog, CD and more. BlitzAmped(c) uses BlitzBass(c) (www.lab-au.com.com) to receive real-time audio data and turn that into real-time 2D FX (requires DX 7.0 and 3D).


Direction for Installation
================
1. Unzip BlitzAmped(c) into a directory.

2. Currently BlitzAmped(c) is in development and uses a WinAmp(c) "Play-List". All you do is save a copy of a Winamp(c) "Play-List" in the same directory as BlitzAmped(c) and name it "bitzamped.m3u".

3. Run BlitzAmpped.

The "." (period) key will advance to the next song.
The "Escape" key will exit BlitzAmped(c)
BlitzAmped(c) will play through the play list till it reachs the end.
