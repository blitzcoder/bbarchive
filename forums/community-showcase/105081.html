Tank Sparks is an intense, action-packed, tank combat game.

Up to 32 tanks duke it out with missiles and rapid-fire machine guns. Because Tank Sparks has six unique artificial intelligence personalities, computer-controlled tanks are worthy opponents.

Ok. Game isn't completely finished.... but let me know what you think.
Some levels may appear impossible or extremely hard. I can complete all of them except for a few, which maybe impossible. You can create your own levels also.

One neat feature is you can create your own tanks in your favorite paint program and then put it in the tank directory with a corresponding text file with same name for tank parameters...... you will then be able to use it in levels you create.

Features

ultra smooth control
bullets
heat seeking missiles
mines
smart AI
design your own levels/missions
design your own tanks
design your own walls
edit tank parameters

Written in Blitzplus
System Requirements
Minimum Screen 1024 x 800
Windows XP to Windows 10
Linux under WineHQ

Dan60(Posted 2016) [#7]
The graphics might suck, but you can draw your own tanks and walls and put them in the tank and wall directory and build levels with them. Just draw tank any size with cannon pointing up in your favorite paint program with black background save file with any name you like as .png, that's it no programming involved. You can add as many tanks as you want. You can also edit tank parameters for each tank such as armor, shot frequency, missiles, speed with notepad. Every tankxyz.png file has an associated text file of the same name tankxyz.txt parameter file
You can pretty much build your own tank game.
