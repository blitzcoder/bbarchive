Well, been working on a cloud library. So far it's proven to be pretty fast..





Beta Demo here:
http://www.kcstudios.net/EZCloud/EZCloud_Beta1.zip

Samples included... Planning on a release early next week.

This was inspired by the cloud system for MS Flight Simulator 2005. I tried to emulate everything I could (minus the shaders of course). I intend to port it to Blitzmax (when the 3d module comes out) and make that update free to registered users.

The software will include (at release) the EZCloud builder (Pictured above) and the library to run the clouds saved out. All clouds are entitites that can be moved, scaled, rotated, etc. The release will also include a huge selection of samples as well.

Clouds are fully configurable and customizable.

From the library one could create simple cloud lanscapes, tornadoes (moving in real time), explosions, volumetric fog, and a few other things I haven't thought of due to exaustion :-)

Feedback welcome ;-)
