I am working on a new Lightmapper. Well in fact it's an old lightmapper :) It's based on YAL (the version modified by sswift to prevent too much bleeding)

I just added a gui and some Mesh exporting code. There are still some bugs, especially one UV problem that I try to fix. Is this is once fixed, I'll release it asap. But of course, it is a very simple lightmapper and it still lacks of a lot of features that I will probably implement in later versions.



Here you can see a Room that was made with Maplet. Then I added a chair and a box which was rotated. Then lights and a lighmap where applied. So the pipeline could be: maplet>crapdropper>decorator.

I really hope this UV Problem is solved soon. BTW. CrapDropper is only a working title (well, you never know :) )
