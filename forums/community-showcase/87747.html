There's been lots of updates to the Xors engine.

If you're talking about Xors3d, download the latest revision from here;
http://hot.xors3d.com/

replace the dlls in your /blitz3d/bin dir, decls file in the userlib and replace xors.bb in your codebase with the copy in the rar file.

At the start of your programme, add a command;
xKey("your licence code here")
with 'your licence code here' being the code you should have recieved on purchase.
