Hi all,

The Tattoo texturing tool is now at release 1.2:


(Texture Size 256 x 256)

New for 1.2:-
* Added the much requested Smear Brush
* Set the wireframe view on the Texture Window to be shown only when the Wireframe view is on the model.
* Added the ability to reload altered texures (i.e if you've doctored a texture in PhotoShop press ctrl-insert and it will be blasted into your model. All texture files are CRCed to check for changes. Only those you have modified externally will be reloaded.
* Set Value button has been moved to the Edit menu to make way for a bigger brush view. (also F4 Hotkey)
* You now have the option of turning off the 'This texture is missing' warning on loading a model. Set this from the config options in the edit menu.
* The Texture View Window no longer keeps disappearing ;).
* Fixed some UV accuracy issues with seams and the 'Tattoo Image' brush.
* Tidied up some inteface elements.
* Improved the speed at which surfaces are cleared. It should be much faster to load different models now, but it will still hang around in the background on shut down for a few seconds while it cleans up properly

Check it out
