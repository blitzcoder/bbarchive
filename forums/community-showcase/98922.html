matibee(Posted 2012) [#2]
It works very well. WinXP Pro, Geforce 8800gts.

The modes are misleading: "800 x 600 x 32" or "800 x 600 x 16" are generally considered full screen modes.

Windowed mode must use the bit depth of the desktop so there are three valid options for "800 x 600":

800 x 600 x 32 = 32 bit full screen
800 x 600 x 16 = 16 bit full screen
800 x 600 windowed = desktop bpp
