This program is under construction. I am writing a simple gamebuilder. It will teach children how to build 3d games.
This part of the program will allow you to compound several 3d objects into a single, animated object.
It will allow you to create (basic) game characters.


Please download the program here:
http://members.home.nl/bramdenhond/compound.rar
I would like to get some feedback on it.

A few basic shapes are supplied. In the final version, you can create different basic shapes using a 3d clay-like editor and an extrude tool.

The program is partly functional. You cannot load or save your designs yet.
