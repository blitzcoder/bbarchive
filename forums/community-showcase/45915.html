Hi guys,

As many of you know, I decided to start a community project some time ago - The Blitz Artificial Intelligence Tournament 2005. We've just started our second thread, and after much discussion, we've decided that the game should be based on "Robot Wars", each contestant writing AI to control their own robots. We have also discussed the many features of the game, such as what commands the contestants can use to script their AI, what weapons the bots can equip themselves with, what objectives there could be for each round of the tournament, and so on.

A large number of the ideas are already in place, and we've written a fair amount of code too. However, we can only make this project a true community success if we get more helpers. In particular, we could do with:

- People who know how to use BVM: We've been offered a free BVM license to help with the scripting engine, and if we can use it, it would make the task much easier.

- Media: We currently have no media, so if any budding artists or musicians could help us out on that front, it would save us time later on.

And of course, we could do with lots of coders!

I've temporarily handed over the project to WolRon as I'll be away for a while (and I mean it this time!), so he's the man to ask regards how you can help out. He's started a new thread here, and on it you'll find links to the previous discussions, info about the coding conventions, and a detailed ToDo list.

Anyone can help out, and your support would be most appreciated.

Many thanks,
Ryan
