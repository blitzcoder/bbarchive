Version 1.2 release with this features:

- Fixed several bugs that caused the game did “crash”.
-A new level "AREA X-511"with 6 new types of ships, intermediate and final bosses.
-2 control types: type 1:only mouse and type 2: control with keyboard and mouse to aim (Some people like to move the ship with the keyboard and aim with the mouse and asked for this extension)
- 5 levels of sensitivity of the mouse (affect the speed of the ship)
- Start game by clicking button or waiting (for slow computer that have difficulty loading the level).
- Added information about game interface in loading menu.
- Smooth camera transitions, pauses scroll and improvement of levels.
- Notification of new downloadable versions available and downloadable levels to start the game.
- Improved balance of difficulty / achievement levels.
- Improved display of the World Ranking.
