Hi!

Dax Robot is a Platform game I'm working on. I still have a lot of work to do on the game but thought I'd release a Pre Beta so people can test the Map Editor and give me some feedback.

Screenies:









Download here:

http://www.kamikazekrow.com/projects/DaxRobot_Alpha1_.zip
============================================

Press F6 to go into the factory. It's the only mode that works but does allow you to load/save maps, make your own and test it out.

Use the keyboard Arrow Keys to move left and right and hit spacbar to jump.

When making a map make sure to have atleast 1 Energytile on screen for you to collect. Collecting all the energy tiles completes the map. The shield, Transporters, Conveyer Belts, Magnets and bombs are all workable except for the bombs not inflicting damage when they explode. Will add this feature in soon.

Once the game is complete I may sell it for a small fee or release the game as free but have the offer to buy the source code, again for a small fee.
