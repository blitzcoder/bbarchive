It turns out a new feature in iOS7 is to allow individual view controllers to show/hide the statusbar in iOS. To disable this and return functionality back to 'normal' simple add the following to MonkeyGame-info.plist

View controller-based status bar appearance

and set it to 'No'

There you go, statusbar hiding/showing according to the 'Status Bar Style' attribute in Deployment Info.

rIKmAN(Posted 2013) [#7]
From the link that Difference posted you should be setting it to 'NO' rather than 'false'...

You should add this value to plist: "View controller-based status bar appearance" and set it to "NO".

This would let you set the status bar to hidden mode and it's global not like other answers.

UPDATE: If you want that the status bar would be hidden on splash screen don't forget to mark "Hide during application launch" on target status bar options. Also, you can add "Status bar is initially hidden" to "YES" on the plist if you don't want to do it with code inside the app.
