nullterm(Posted 2015) [#6]
pit, as far as I know there's no way to run from Monkey into the simulator. Xcode does have a cmd line interface, so maybe it could be hooked into the iOS target? But that might be a more advanced topic. https://developer.apple.com/library/ios/technotes/tn2339/_index.html

Edit - also this link talks about running the sim from cmd: http://stackoverflow.com/questions/26031601/xcode-6-launch-simulator-from-command-line

A possibly easier solution would be to add the Ted/Trans build step as a custom build phase in the Xcode project so you could edit your .monkey files in Xcode and hit Run, and also be able to debug. The downside is Xcode doesn't understand Monkey source syntax.
