MikeHart(Posted 2013) [#7]
A current way you could do is to put both images into a strip and then use eng.CreateAnimImage. Then handle the animation manually, by pausing the animation with SetAnimPaused and set the frame you want to display with SetAnimFrame.

I am in the process of adding the ability to add seperate images to an existing image object, so there it will be even more easier to setup an animated object.
