Shagwana(Posted 2013) [#4]
You need a display before DeviceWidth() will work.

Extend APP and you will see it work.

Import mojo

Function Main()
	Local cGame := New Game_c
End Function


Class Game_c Extends App
    

    Method OnCreate:Int()
        SetUpdateRate 30
        Return 0
    End Method
    
    
    Method OnUpdate:Int()
        Return 0
    End Method


	Method OnRender:Int()
		Cls
		Print DeviceWidth()
		Return 0
	End Method	

End Class
